Order None clause

Allows to specify no navigation order for the For Each command or Data provider group.

This clause prevents GeneXus from choosing the order of the attributes of the base table’s primary key by default, and makes it use an undefined navigation order.

Syntax

Order none

Scope

Commands: For Each command, Data provider Group statement
