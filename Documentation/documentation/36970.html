JPush - Channel property

Specifies channel value for the application package, in order to get sub-channel statistics.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple

Description

The property value is used to specify channel values for the application package, in order to get sub-channel statistics.
More information can be found here.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 15 Upgrade 8.
