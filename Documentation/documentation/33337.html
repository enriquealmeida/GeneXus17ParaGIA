Replace option

Allows searching for and replacing text in source code.

Values

Find what String to be replaced.
Replace with String to be replaced with.
Match case Indicates if the search must be case-sensitive or not.
Match whole word Indicates that the search must match the entire word.
Search up Makes an infinite search upwards.
Use Check this option to enable the related combo box:
  • Use Regular expressions: enables a Regular Expression search.
  • Wildcards: The search includes wildcards. For example, pattern E* will find the strings Event and EndEvent; pattern E?? will find at least the End string.
Search hidden text Forces the search to check within hidden text, such as collapsed code.
Search in selection The search will only consider the selected code.
Mark with bookmarks For each matching string, a bookmark line is added.
Find Next Performs next search.
Replace Replaces current instance.
Mark All Marks all items matching the pattern.
Replace All Replaces all instances from current position to the end.
Close Closes the dialog.
