ActiveLinkColor property

To allow indication of the color for the active link.

Description

This means that the user has already made a selection, which is under the arrow without pressing any button of the mouse. Any color may be used, but we recommend that the criteria be maintained throughout the whole application in order to avoid confusion for users.

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
