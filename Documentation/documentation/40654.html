Data Type Filter property

Indicates the data types list for which the user control is appropriate. A comma-separated list (,) is expected with the following values: Numeric, Character, VarChar, LongVarChar, DateTime, Date, Image, Video, Audio. and any other valid SDT type.

Description

This property is offered for User Control objects that have their Is Control Type property set to True.

The value of this property must be a comma-separated list (,) with the following values: Numeric, Character, VarChar, LongVarChar, DateTime, Date, Image, Video, Audio. For example, a valid value for the property can be: VarChar, Character, LongVarChar.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16.

Scope

Objects: User Control
