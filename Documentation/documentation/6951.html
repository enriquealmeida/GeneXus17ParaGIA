JDBCDriverName Property

Returns or associates a JDBC driver name to a DBConnection variable.

Syntax

&DataType.JDBCDriverName

Type Returned:
Character

Example

&myConn.JDBCDriverName='com.inet.tds.TdsDriver'

Scope

Extended Data Types: DBConnection
Objects: Procedures, Transactions
Languages: Java
Interfaces: Web
