Dark Theme property

Theme variant to be used when dark is the user preferred color scheme on supported platforms.

Scope

Objects: Theme for Smart Devices
Platforms: Smart Devices(IOS), Smart Devices(Android)

Description

This property is available when you create a new Theme object with a new Color Palette that defines which are the colors for the Dark Theme.

Considerations:

  • The Dark Theme should be the son of the main Theme.
  • Additionally, the Enable Preferred Color Scheme property must be set to True.
  • It is required that the device has the dark mode set so that a Dark Theme can be shown.

Note: This property is available for Android as of GeneXus 16 upgrade 7 and Android 10 is needed.

 

Run-time/Design-time

This property applies only at design-time.

Samples

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

Availability

This property is available since GeneXus 16 upgrade 6.
