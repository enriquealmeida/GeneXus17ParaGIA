GeneXus .NET Generator (GeneXus 17 upgrade7)

C# (CSharp) is a language developed by Microsoft for running applications on .NET platforms (.NET Framework).

.NET generators allow generating applications in C#, of Web and Win type—in this case, the latter can have two or three layers. The programs generated are C# source code files (.cs), which are then compiled into assemblies (.dll or .exe) in common intermediate language (IL). At runtime, they are interpreted by the .NET virtual machine.

The applications communicate with the database through ADO.NET, and the DBMS available are all those supported by GeneXus: DB2 UDB for iSeries, DB2 Universal Database, Informix, MySQL, Oracle, PostgreSQL, and SQL Server.

.NET Generator Requirements

In addition to GeneXus’ basic requirements, such as disc space, volume of memory, etc., for every development team the following software must also be available:

  • Microsoft .NET Framework 3.5 or higher
  • IIS 6.0 or higher
  • ADO .NET provider associated with the selected DBMS.
