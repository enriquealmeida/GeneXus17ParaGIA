Previous Event

To bring up the previous set of records in the table, to be shown in the grid, according to the current index.

Description

This event occurs when the user clicks the associated button 

Comments

The GeneXus developer cannot associate any code to this event.

Apparently, the Refresh event does not run when Previous() is executed.

Scope

Objects: Transactions
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
Interfaces: Web
