How Can I Create My Own Categories?

You can create:

  • A Static Category:  Through the Category View.
  • A Dynamic Category:  Through the Search window (by specifying search criteria), and also by the Category View (by adding a query and/or properties in the Category Properties window).

See Categories for further information.
