Which objects can be defined in a module?

Almost all GeneXus objects can be defined in a Module object.

Exceptions

Below is a table that shows the objects or concepts that do not belong to modules.

  • Language object
  • Theme object
  • Web Theme object
  • Color Palette object
  • Attribute
  • Deployment Unit object

Note

  • The module of a Table is defined by the Transaction object that it defines
  • Use Design System Object instead of Theme, Web Theme, or Color Palette.
