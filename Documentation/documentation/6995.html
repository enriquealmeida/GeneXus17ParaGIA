BCC Property

Indicates the list of hidden recipients (Blind Carbon Copy) of a message.

Syntax

&DataType.BCC

Type Returned:
MailRecipientCollection

Description

This property is read only, it cannot be assigned.

Note

If you use POP3Session Data Type, this property never returns the addresses of recipients.

Scope

Extended Data Types: MailMessage
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
