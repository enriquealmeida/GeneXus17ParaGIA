Disambiguation Message property

Specifies the disambiguation message.

Scope

Objects: Conversational Flows Instance

Description

Allows defining the message that is going to be displayed when a disambiguation occurs and how to present the options. The default value is "Did you mean:"

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 upgrade 3.
