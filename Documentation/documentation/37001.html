Application Id property (WeChat Pay)

Application ID obtained when registering your application in WeChat Open Platform, to be used for WeChat Pay integration.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple

Description

When WeChat Pay integration is required, a registration process need to be followed for the application.

In this process, an identifier is created for the application, which value has to be set in this property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build theÂ Main Object.

Availability

This property is available since GeneXus 15 Upgrade 8.
