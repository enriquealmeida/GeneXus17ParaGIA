Output device location property

To allow indicating either the printing (in text mode) will be sent to the Server printer or to the Client printer. It happens only when the “Report output” property value is “Only To Printer”.

Values

Client The printing is sent to the client printer (it can be the default or the configured printer).
Server The printing is sent to the server printer. This is the default value.

Description

It only applies to reports going directly to the printer. The solution is available for Linux and Windows platforms, for either the server or the client machine. In case of Linux machines, the report must  be “txt”, meaning that “Text Report Output Property” must be set to “Text” value.

Scope

Platforms: Web(.Net, Java)
