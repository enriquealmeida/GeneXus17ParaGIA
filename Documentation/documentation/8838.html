Focus control property

Determines if Web objects –when showing a page– try to put the focus in the first input control on a plain level Form (non-read-only controls) or leave the position of the focus up to the Browser.

Values

Browser dependent
Use Environment property value
First input att/var on the page
First input att/var on the page only if not embedded

Scope

Objects: Web Component, Web Panel
Level: Generator

Description

You can find this property at the Back end Generator level and also at the Web Panel objects and Web Component objects level.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
