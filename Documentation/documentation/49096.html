gx-unselected-forecolor property

Note:
This property of the Design Systems object corresponds to the Unselected Forecolor property of the Theme.

 

Text color for the unselected segment of the ToggleButtonGroup user control.

Scope

Objects: Panel, Work With
Generators: Android, Apple

Run-time/Design-time

This property applies both at run-time and at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 17 upgrade 6.
