BeaconProximity Data Type

This domain is used in the Proximity field of the BeaconState Data Type to determinate the proximity of the beacon from the device. The value in this property gives a general sense of the relative distance to the beacon.

Values

Value Description
Unknown  
Inmediate  Within a few centimeters.
Near  Within a couple of meters.
Far Greater than 10 meters away.
