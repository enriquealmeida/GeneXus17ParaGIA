Dashboard Main color property

Main color for the filters and objects included in the Dashboard.

Scope

Objects: Dashboard

Description

Defines the Main color of the control layout and associated dialogs for widgets in a Dashboard object.

Run-time/Design-time

This property applies only at design-time.

Samples

Carmine

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.

Availability

This property is available since GeneXus 16.
