Calendar property

It allows determining the calendar with which the time will be counted through a dialog that shows all the BPD Calendars defined on the Knowledge Base.

Description

If we don´t specify a calendar the process will be continuous, 24 hours a day and 7 days a week. By default, the calendar will be the process calendar if it corresponds.

Scope

Objects: Business Process Diagram Object in Evolution, (BPD Tasks, BPD Subprocesses)
