HowTo: Logging Users to GXflow Client by Programming

It allows logging into GXflow Client application by programming, thus avoiding logging in through the Client login dialog.

&Session.Set('WorkflowUser','WFADMINISTRATOR')

&Session.Set('WorkflowPassword','WFADMINISTRATOR')

//Java
Link('com.gxflow.wfautosignin')

//Net
Link('wfautosignin.aspx')

commit

Where:

&Session data type is WebSession
