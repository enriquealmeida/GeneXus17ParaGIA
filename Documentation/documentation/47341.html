Timer expression type property

Selects whether the timer expression is a rule or a procedure.

Values

Procedure
Rule

Scope

Objects: Business Process Diagram

Description

Possible values:

  • Rule: Allows setting start information at design time(1).
  • Procedure: Allows setting a procedure to set the Timer Date at runtime.

(1) - The Timer Control scheduler must be running to trigger the associated event. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

Availability

This property is available since GeneXus 16 upgrade 4.
