HttpClient data type (up to Genexus 16 Upgrade 11)

Builds a request, send it to a URL, and read the results.

This object reflects an HTTP connection.

Properties

BaseURL ProxyServerHost and ProxyServerPort
Basic and Digest ReasonLine
ErrCode Secure
ErrDescription StatusCode
Host Timeout
Port  


The URL format is the following: http://Host:Port/BaseUrl/Resource

Methods

AddAuthentication AddVariable
AddFile Execute
AddHeader GetHeader
AddProxyAuthentication ToFile
AddString ToString

Scope

Objects Procedure object, Transaction object, Web Panel object, Work with for Web, Work With object, Panel object
Platforms  Web(.NET, Java, NET Core), Smart Devices(Android, IOS).
   

Availability

This data type is available since GeneXus X

  • Since GeneXus 15 the methods can be used from offline procedures in Smart Devices apps. 
  • Since GeneXus 15 upgrade 12 the methods can be used in the user-defined event of Smart Devices apps. 
  • Since GeneXus 15 Upgrade 6 multi-part files are supported using the AddFile method or the AddVariable Method.
