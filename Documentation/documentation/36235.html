Deselect method

Deselects an item in a Grid by giving its index. If the item is not selected or if it is out of the Grid's scope, it will not have any effect.

Syntax

<Grid>.Deselect(<index>)

Where:
<Grid>
    Is the grid control name

<index>
    Is a Numeric expression (attribute, variable or constant value).

Scope

Controls: Grid
Generators: Apple, Android, .NET, Java, Angular

Notes for Smart Devices:

  • This method will not execute the default action associated with the Grid.
  • This method will properly switch the default selected item layout.

Availability

This method is available since  GeneXus 15 Upgrade 6 for iOS, GeneXus 15 Upgrade 7 for Android and Genexus 15 Upgrade 10 for Web.
