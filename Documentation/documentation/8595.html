For To Step Command

To allow iterating a certain number of times from a certain value.

Syntax

For &var = start to end [step leap]
       code
Endfor

Where:
start, end
    Are numeric expressions showing the start and the end of the loop.

step
    Specifies the leap of the loop.

leap
    Is a constant value.

code
    Sequence of valid language commands.

It is a loop from the start value of &var, as long as &var is <= than end, it increases a leap. The default value of leap is 1. The leap value can be negative.

Notes

  • The scope of the loop variable is, as usual in GeneXus, the whole object
  • The value of the loop variable outside the loop is not defined/documented and may change through platforms and versions.

Example

Event Refresh
    &Tabs = LoadPageTabs(&PageType, &PageFullName, &PageId, &PageLastVersionId1)
 
    // Select the subset of tabs that will be shown.
    &FirstTab = 1
    &LastTab  = &Tabs.Count

    &IsFirstTab = True
    For &Index = &FirstTab To &LastTab
        &Tab = &Tabs.Item(&Index)
        Do 'LoadItem'
        &IsFirstTab = False
    EndFor
    ...
EndEvent
