Horizontal Alignment property

Horizontal Alignment is a control property for Responsive Web Applications, whose purpose is to align the control to the left, right or center of the cell where it's contained. In fact, it's a property of the control cell. 

Values

Default, Left, Center, Right.

Example

Consider a web page header where the logo is sent to the top left corner, and the "welcome user name" text is sent to the top right corner for medium and large screens.

1. Drag a Responsive Table to the form, which will contain:

  • the image of the logo, and
  • the Responsive Table containing the welcome message (named "HeaderTable" here).

Configure its width in percentages for the cell containing the logo. The "HeaderTable" occupies the remaining percentage.

2. In the "HeaderTable", insert a common Table control whose control name is "MessageTable" in this example. The table control has no fixed width and takes all the horizontal space necessary for its content to be displayed without wrapping.

3. Edit the "MessageTable" properties and configure the Horizontal Alignment property to Right.

Scope

Objects:    Transaction object, Web Panel object
Controls:    All
Interfaces:    Web

Availability

As from GeneXus X Evolution 3 Upgrade 4.
