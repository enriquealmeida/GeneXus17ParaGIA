Page variable

Stores the Procedure's current page number. GeneXus automatically manages the value of this variable and the page count starts at 1. If another starting value is required, the value of &Page can be changed. GeneXus manages the variable &Page by adding 1 to the initial value each time there is a page advance.

Data Type:
Numeric (6)

Scope

Objects: Procedure
