Query Viewer: Font weight property

Sets how thick or thin characters in text should be displayed. Values 100 to 900 defines character thickness from thin to thick. 400 is the same as normal, and 700 is the same as bold.

Values

Not set
100 Thin
200 Extra light
300 Light
400 Normal
500 Medium
600 Semi bold
700 Bold
800 Extra bold
900 Black
bold Defines thick characters.
bolder Defines thicker characters.
lighter Defines lighter characters.
normal Defines normal characters.

Description

Defines the characters thickness.

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Query
