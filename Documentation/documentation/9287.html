Web Root property

Allows to select the root execution location in web platform. Is "read only" when Deploy to cloud property is set to Yes.

Samples

http://localhost:8080/3679JavaWeb-ModifiedProperties/servlet/<object>

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Platforms: Web(.Net, Java)
