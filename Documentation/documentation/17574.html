QueryViewer with High Charts (JS)

Using the property ShowQueryUsing, located at version and QueryViewer control level, we can define how to render Charts.

By default converted Knowledge Bases will use the AdobeFlashPlayer value, new Knowledge Bases will use the new Javascript implementation. Some snapshots on the new charts are:

 

Pie
 Line
Bar
 Column
Timeline

Doughnut 3D

Javascript rendering advantages

  • Performance: faster loading than AdobeFlashPlayer implementation.
  • Portability: this implementation doesn't depend on additional components like AdobeFlashPlayer.

Availability

  • Timeline Chart is available since GeneXus X Evolution 2 Upgrade 2 and new features are available in GeneXus 15. (For more information refer to: QueryViewer TimeLine)
  • Doughnut 3D Chart is available since GeneXus 15.
