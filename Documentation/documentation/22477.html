Qualified Name property

Read-only property showing the actual object name. It is comprised of the Module path and the value of Name property.

Syntax

[{ModuleName.} ... ]{ObjectName}

Where:
[{ModuleName.} ... ]
    Is the Module path.

Note: the Root module is always ommited in the Module path.

{ObjectName}
    Is the name of the Object. Same as Name property.

Description

This — read-only — property is intended to provide a quick way to identify an object, as several objects can have the same value for the Name property if they have a different Module path.
