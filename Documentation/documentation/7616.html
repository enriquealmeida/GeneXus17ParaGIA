Language Translations Property

This option indicates the behavior that the import process will have regarding the Application Localization. That is, what will happen with the strings translations when importing Language objects.

Values

Update : Updates the translations of the language with that existing in the XPZ, adding the new translations possible in the XPZ. Translations not included in the XPZ are not deleted.
Keep : Only add the translation of the XPZ that do not exist in the language object. 
Replace All : The language object of the XPZ fully  replaces the existing one in the Knowlegde base.

Default: Update

Scope

Action: Import option
