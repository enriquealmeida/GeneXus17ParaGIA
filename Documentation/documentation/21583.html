HttpOnly property

It is a property of the Cookie data type. It allows to avoid to save or set a cookie from JavaScript code.

Values

True To avoid to save or set a cookie from JavaScript code. This is the default value.
False Ignore to avoid to save or set a cookie from JavaScript code.

Java Considerations

In JAVA, the httpOnly property is supported for servlet 3.0 specification or higher.
