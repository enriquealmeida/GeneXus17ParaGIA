DB2 UDB Version property

Defines the DB2 Universal Database version installed on the Database Server.

Values

8.0 or higher The DB2 UDB version is 8.0 or higher. This is the default value
7.x The DB2 UDB version is 7.x

Description

Every new DBMS version has new features that are used by GeneXus, which generates different SQL sentences or considers properties that can be applied to the new version.

How to apply changes

Rebuild all objects.

Scope

DBMS DB2 Universal Database
