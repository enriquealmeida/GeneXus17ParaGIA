Firebase Apple File property

Indicates the configuration file to use for Firebase in iOS. This file can be obtained in the Firebase Console.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple

Description

Firebase configuration is done in the Firebase Console and downloaded as a file.

To use that configuration in a GeneXus-generated application:

  • Add the downloaded file to the KB as a File object, and
  • Add a reference to that File object in this property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build theÂ Main Object.

Availability

This property is available since GeneXus 17 upgrade 4.
