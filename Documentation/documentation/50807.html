Newest Version

Web Frontend Version Code property

A value that represents the version of the application code, relative to other versions.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Angular

Description

Every time you make significant updates to your application, it is recommended to publish a new version and update this property.

When possible, follow the semantic versioning spec ({Major}.{Minor}.{Build}) described here.

  • Major: when you make an incompatible change.
  • Minor: when you add functionality in a backwards-compatible manner.
  • Build: when you make backwards-compatible bug fixes.

Run-time/Design-time

This property applies only at design-time.

Samples

  • 1.0
  • 1.0.1
  • 13.5.2

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 17 Upgrade 10.
