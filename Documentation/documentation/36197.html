Enable Multiple Selection property

Allows the developer to enable multiple selections over a Grid control.

Values

Value   Description
False Default value. Multiple selections are not allowed.
True Multiple selections are allowed.
It displays a set of properties for setting its behavior.
• Show Selector property
• Selection Flag property
• Selection Flag Field Specifier property

Run-time/Design-time

This property applies only at design-time.

Scope

Controls Grid control
Platforms   Smart Devices (iOS, Android)

Availability

This property is available as of  GeneXus 15 Upgrade 6.
