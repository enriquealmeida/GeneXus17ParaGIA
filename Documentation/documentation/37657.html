Grid theme-class for Smart Devices

Allows the developer to set some properties to a Grid control.

Properties

  • Name property
  • Description property
  • Transformation property
  • Animated property
  • Animation Duration property (for Animation Theme class) (if Animated property is enabled)
  • Border Style Property
  • Border Color property
  • Border Width property
  • Border Radius property
  • Margin group
    • Margin Top property
    • Margin Left property
    • Margin Bottom property
    • Margin Right property
  • Background Color property
  • Highlighted Background Color property
  • Background Image property
  • Highlighted Background Image property
  • Background Image Fill Mode
  • Show Horizontal Line Separator Between Rows
  • Odd Row Class
  • Even Row Class
  • Group Separator
  • Elevation property
  • Padding group
    • Padding Top property
    • Padding Left property
    • Padding Bottom property
    • Padding Right property
  • Loading Animation Class (1)

Notes

  • (1) Available as of GeneXus 15 Upgrade 8

Scope

Objects: Theme object  
Platforms: SmartDevices(iOS, Android)
