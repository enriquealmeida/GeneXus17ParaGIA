Zoom Factor property

This property is available for Table and Canvas controls included in SD Panels and WWSD.

It is a numeric value that increases the width and height of a table or canvas to simulate a zoom effect when scrolling on the control (or set of controls) indicated by Scroll Attachment property. 

The default value for this property is 0, meaning that no zoom effect has been applied to the control. In other cases, a smooth zoom effect is displayed when the end user scrolls on any of the controls where is attached. It will be a zoom-in effect if its value is positive, else will be a zoom-out effect.

Note

It is commonly used with the Scroll Factor property to simulate a Parallax Effect.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices

Controls: Table, Canvas

Languages: .NET, Java

SD Generators: iOS

Availability

This property is available since GeneXus 15
