UI Test object

Defines a sequence of User Interface interactions and validations.

Represents a GeneXus Procedure object with the ability to control the device using methods of an External Object. Many user interface actions are available, including tapping on controls or typing in different controls to input values, checking text on panels and values in controls (variables/ attributes), etc.

You can create a UI Test object through the New Object dialog. There, you have to select the Test category, and you are offered to create a UI Test object.
