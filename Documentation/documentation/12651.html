Second method

Returns a numeric value representing the seconds of a datetime argument.

Syntax

Datetime.Second()
 
Type Returned:
Numeric(2)

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

Samples

If &DateTime.Second() > 0
   &TxtMinute = "Past o'clock..."
EndIf
