Twitter external object

It enables you to send tweets from your application.

Warning: To use this method, setting Consumer Key and Consumer Secret is required.
In the settings option of your Twitter account, for iOS you need to set a Callback URL following this documentation from Twitter; for Android, you need to set this Callback URL: https://twitter4j, and check the option "Allow this application to be used to Sign in with Twitter".

 

Properties

It does not have any.

Methods

Tweet method

It allows sending a status update (optionally with an image on the device), showing a dialog which is to be customized by the end user.

Return value  None
Parameters Text:Character(140), [ , image:Image ]
 

Follow method

It allows following a user using his/her username (without '@' (at)); maximum 15 characters.

Return value  None
Parameters userName:Character(15)
 

Events

It does not have any.

Scope

Platforms  Smart Devices (iOS, Android)

 

Availability

This external object is available as of GeneXus X Evolution 3.
