Delimiter Property

This is a field delimiter that is used when opening text files. 

Syntax

&ExcelDocument.Delimiter

Type Returned:
Character

Description

The default value is a comma (“,”). 

Scope

Extended Data Types: ExcelDocument
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
