Front-end applications deploy

Front-end applications, such as Angular applications, can be deployed to any Cloud Provider Object Storage or to docker.

The Application Deployment tool supports the deployment to AWS S3 and Azure Blob Storage and to docker containers.

You can use the Application Deployment tool from inside the GeneXus IDE, or MSBuild commands.

Availability

The deploy to docker support is added since GeneXus 17 Upgrade 11
