Access Contacts Notes property

Adds the com.apple.developer.contacts.notes entitlement to the generated project.

Values

False
True

Scope

Platforms: Smart Devices (IOS)

Description

This property adds the "com.apple.developer.contacts.notes" permission to the generated project, in order to access the notes stored in contacts on devices with iOS 13 or later. On devices with iOS 12 or lower, this property will not be affected.

Before you can submit an app with this entitlement to the App Store, you must first get permission to use the entitlement. Request permission at https://developer.apple.com/contact/request/contact-note-field 

Run-time/Design-time

This property applies only at design-time.
