HowTo: Opening a Module Object

There are many ways to open a Module object. Below is a list of the most common and easiest ways to do so.

A Module Object can be opened by:

  • Selecting it from the KB Explorer and pressing F12.
     
  • Right-clicking on the module from the KB Explorer and selecting Open.
     
  • Using the Work With Objects — Ctrl+J — or Select Objects — Ctrl+O — dialogs, by double-clicking on them from the corresponding list.

Once the Module is opened, the Module Interface Tab is displayed. Then the Module Diagram Tab can be accessed.
