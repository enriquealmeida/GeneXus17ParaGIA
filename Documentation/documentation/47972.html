Mapbox API Private Key property

Sets the Mapbox API Private Key.

Scope

Objects: Panel
Generators: Android

Description

This property is available when the Android Maps API property is set to "Mapbox API." It is necessary to use the Mapbox Maps provider. To get a valid API key value, refer to Mapbox Access Token.

Availability

This property is available since GeneXus 17 upgrade 4.
