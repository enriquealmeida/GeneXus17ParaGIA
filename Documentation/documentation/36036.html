Fixed Grid header with vertical scroll

It is possible for a grid in Web environments to set a fixed header. Its purpose is to enable scrolling of the whole grid keeping the column headers visible at any time.

To use it you need to:

  • Set the grid AutoResize property to false.
  • Set the grid Height property to some value different from 0; for example you could set a fixed size such as 500px or relative values such as 80vh or 80% (or any other value).
  • Set the grid Rows property to 0 so paging is disabled).
