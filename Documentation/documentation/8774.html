Font property

Sets the font (the type of letters), the size and style to be used for the text inside the control.

Description

The letter size is specified in pixels and is fixed. The possibility of enlarging or shortening the font that the Browsers have is not applied to the controls.

Applies to:Â Check Boxes, Combo Boxes, Dynamic Combo Boxes, Dynamic List Boxes, Edits, Error Viewer, Grid Columns, List Boxes, Radio Buttons

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
Controls: Attribute/Variable
