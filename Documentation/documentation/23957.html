Clipboard.setText method

This method is used to copy text into the clipboard. Copying text to the clipboard replaces any previously stored contents.

Syntax

Clipboard.setText(<TXT>)

The <TXT> parameter indicates the text to be copied into the clipboard.

Example

Event 'setURLtoClipboard'
   Clipboard.setText(&URL)
Endevent

Availability

As of GeneXus X Evolution 3.
