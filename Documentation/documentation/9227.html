Time format property

Controls the format in which the time part of datetime attributes are accepted and displayed (either on the screen or in printed documents).

Values

12 hour format
24 hour format

Description

Default value: 12 Hour format

It also controls the length of datetime attributes/variables. This property is applied to the Knowledge Base but at the same time, each model can have a different value for this property.

Note

If the use of 12 Hour Format is specified or inferred, according to the selected language, the fields will be 3 characters longer (screen or printer) corresponding to the strings ‘AM’ or ‘PM’.

Scope

Objects: Language
