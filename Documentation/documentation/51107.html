Metadata files generation messages

The following list shows the mtd messages that can appear when generating metadata files.

Code Message
 
mtd0001 The control type '%1' is deprecated in control '%2'.
 

It indicates that a deprecated control type is being used. Consider using another control type, because it is possible that in the future it will no longer be supported.
 
mtd0002 Invalid value '%1' for property '%2' in class '%3'
 

It indicates that a value is being used in a class that flexible clients do not support. The value is ignored.
 
mtd0003 Invalid value  '%1' for property '%2'
  It indicates that a value is being used in a token that flexible clients do not support. The value is ignored.
