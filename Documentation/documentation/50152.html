Item Pressed property

A variable based on ActionGroupItem SDT with the item pressed.

Scope

Objects: Web Panel, Web Master Panel

Description

The ActionGroup item that triggered the event is instantiated in the variable configured in this property.

Run-time/Design-time

This property applies only at run-time.

Availability

This property is available since GeneXus 17 Upgrade 9.
