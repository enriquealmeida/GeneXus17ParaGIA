SD Horizontal Position property

Horizontal alignment for the content of the layout element.

Values

Center Center the text.
Default For both platforms, the value is Left.
Justify Justify text.
Left Align text to the left.
Right Align text to the right.

Description

Text alignment determines the appearance of the text in a paragraph. If the paragraph is left-aligned, the text will be aligned to the left margin. If the paragraph is justified, the text will be aligned to both margins. 

Note:

Justify is available as of GeneXus 16 upgrade 8. In Android, it applies only to text blocks and read-only attributes/variables. 

Run-time/Design-time

This property applies only at design-time.
