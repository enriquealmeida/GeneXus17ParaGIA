Image theme-class

Allows the developer to set some properties to an image.

Properties

  • Border Style
  • Border Color
  • Border Width
  • Border Radius
  • Margin
  • BackgroundColor
  • Highlighted Background Color
  • Heigh
  • Width
  • Content Mode
  • Label Class
  • Label Width
  • Label Horizontal Alignment
  • Label Vertical Alignment
  • Placeholder Image
  • Image Loading Indicator

Notes

  • Images can be Image object, Image-based properties and attributes/variables based on Image data type.

Scope

Objects: Theme object  
Generator: Android, Apple
