Data data type

Describes an item of a data-set in the context of GeneXus Artificial Intelligence for custom models.

Members

  • Purpose: Purpose, GeneXusAI.Custom
  • Input -- Your input data
    • Resource: VarChar(512)
  • Output -- The inference for your input data
    • Label: Label, GeneXusAI (mandatory for training)
    • Region
      • Top: Numeric(4.0-)
      • Left: Numeric(4.0-)
      • Width: Numeric(4.0-)
      • Height: Numeric(4.0-)

Notes

  • The Region's subfields are negative where they are not available.

Scope

AI Task Train procedure, Predict procedure
Platforms  Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This data type is available as of GeneXus 16 upgrade 6.
