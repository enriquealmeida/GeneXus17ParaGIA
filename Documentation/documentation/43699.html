Allow declaring references to non existing classes

Configure if users can declare references to non existing classes in Class property. That is to say, classes that have not previously been defined. (*)
(In any case, the referenced classes will exist in the KB after saving the object)
Values:
Yes: You will be able to set a class name that has not been defined. The IDE will automatically create the theme class.
Ask: The IDE asks you before creating the class. This is the default option.
No: The IDE won't let you set a class name that does not exist.
 

(*) This dialog appears when setting a class that has not been defined and the "Allow declaring references to non existing classes" is set to 'Ask' (Default)
