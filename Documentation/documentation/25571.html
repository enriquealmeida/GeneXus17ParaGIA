Flip.Start event

It defines an action to be performed when the main object starts running if Flip Navigation Style is selected. It is a Client Event that takes place before the Server Events (Start, Refresh and Load) and the ClientStart Event.

Since it is a Client Event, the possibilities and limitations are the same as for any other Client Event.

Syntax

Event Flip.Start
Event_code
EndEvent

Where:
Event_code
Code associated to the event.

Examples

Event Flip.Start
  //Call the tab where you want to start the application
  SDPanelStart.Call()
EndEvent

Considerations

It is important to note that, by default, the main Dashboards are shown as tabs. Therefore, if we have a Dashboard as a main object (with the default control type) and the navigation style property is set to Flip, the event executed will be the Tabs.Start event.

Scope

Objects SDPanel, Menu object
Platforms IOS
