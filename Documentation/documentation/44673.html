Continuous Deployment

You achieve Continuous Deployment when your Continuous Integration and Continuous Delivery process is so smooth/automatic that you can deploy automatically and continually.

GeneXus offers the option to deploy to Docker Container to be able to make more continuous delivery of your application. This allows creating a binary containing the entire application inside to deploy. The good thing about this is that it is programmable and automatable, so it is easy to include it in your DevOps process.
