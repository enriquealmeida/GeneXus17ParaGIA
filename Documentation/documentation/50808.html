Web Frontend Version Name property

A string value that represents the release version of the application code as it should be shown to users.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Angular

Description

Uses a string value to represent the Release version associated with the Version Code property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build theÂ Main Object.

Availability

This property is available since GeneXus 17 Upgrade 10.
