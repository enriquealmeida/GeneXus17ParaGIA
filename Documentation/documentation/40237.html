Scroll Bouncing property

Specifies whether the scroll view bounces past the edge of content and back again

Values

Always Always bounces in any direction
Always Horizontally Always bounces horizontally, and only bounces vertically when the content vertical size is bigger than the container's vertical size
Always Vertically Always bounces vertically, and only bounces horizontally when the content horizontal size is bigger than the container's horizontal size
Platform Default In each case, it solves whats best for its context (default)
Never Disables bouncing in any direction
When Content Allows Scrolling Bounces only (horizontal or vertical) when the content is bigger than the container's size. This is what Platform Default resolves too in most cases

Description

Refer to Scroll properties group for a broader description and to use it in conjunction with other properties.

How to apply changes

To apply changes made by this property, Build a main object.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Theme for Smart Devices
Platforms: Smart Devices(IOS)
