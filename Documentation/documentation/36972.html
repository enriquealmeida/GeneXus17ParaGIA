JPush - NDK ABI Filters property

ABIs that will be included in the Android application.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Android

Description

Indicates the filters for the CPU and instructions sets of the Android devices for which the JPush service will apply.
Detailed information about ABI Management in Android applications can be found here.

Run-time/Design-time

This property applies only at design-time.

Samples

Default value:
armeabi;armeabi-v7a;armeabi-v8a

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 15 Upgrade 8.
