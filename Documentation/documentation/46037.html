Cell method

Returns an ExcelCellRange data type which contains the indicated cell.   

Syntax

&ExcelSpreadSheet.Cell(Row, Column)

Where:
Row
     Is the row where the cell is.

Column
     Is the column where the cell is. 

Type Returned:
Numeric

Scope

ExcelSpreadSheet data type

Description

It returns an ExcelCellRange data type with the cell located in the Row (row specified) and Column (column specified).
