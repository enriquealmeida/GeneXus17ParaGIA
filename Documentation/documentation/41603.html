Indicator Symbol property

Symbol used for the page indicator of the Horizontal grid. UTF-8 symbols are supported (For example, using \2022 for a dot).

Scope

Platforms: Web(.Net, .Net Core, Java)
Controls: Free Style Grid

Description

The Indicator Symbol is a property of the grid Page Controller Theme class.

This class is used for configuring the appearance of a Horizontal Grid (a grid whose Custom Render property is set to Horizontal grid).
