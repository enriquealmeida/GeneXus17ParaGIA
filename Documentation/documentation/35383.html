Column Header Class Property

The objective of this property is to allow setting the Cascading Style Sheet Class for a specific grid column's header.

Description

It's a property of the GridColumn Theme Class and its descendants, and its purpose is to allow setting the style to a grid column's header.
At the same time, it must be assigned to any Theme Class that isÂ below the node of the "GridColumn" predefined class.

Example

Suppose that you assign the Column Class property to any column of the grid at runtime, as shown in the following code:

Event grid1.Load
    if AuthorName = 'Pablo Neruda'
        LiteraryWorkName.ColumnClass = ThemeClass:GridPreferredColumn
    else
        LiteraryWorkName.ColumnClass = ThemeClass:GridLWColumn
    endif
Endevent

Besides highlighting that pair (row, column), you may also need to highlight the column's header.

That's where the Column Header Class property comes into play.

Set the Column Header Class property to a class where you'd define the style settings as you wish.

Availability

Since GeneXus 15 upgrade 5.
