AddHours method

Adds hours to a datetime.

Syntax

Att | &var.AddHours(Hours)

Where:
Hours
   The number of hours added to the Att | &var.

Type returned:
DateTime

Scope

Objects: Procedure, Transaction, Web Panel

Generators: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol

Samples

&DateTime.AddHours(1)  // Adds 1 hours to DateTime variable
