Add method

Adds an item to a collection.

Syntax

Add( Item [ , Position ] )

Where:
Item
    Variable based on SDT.

Position
    Relative position.

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Description

It adds an Item to the collection in a relative position: Position.
If Position is omitted, or if “0” is specified, the Item is added at the end of the collection. Position starts at 1.

Samples

Before adding each element to a collection, it is necessary to create a memory space by using the new operator.

for &i = 1 to 1000
  &item = new()
  &item.SomeMember = "SomeValue"
  &sdtCollection.Add(&item)
endfor
Note: The same method can also be used to add a new item to a list of Java Scripts (in Transactions and Web Panels), like shown: Control.Add(Item). For more details, check this article: Access to a Web Object's Header
