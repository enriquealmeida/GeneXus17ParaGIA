Options property

Specifies the Design System options for an image.

Scope

Objects: Image
Generators: Android, Angular, Apple, Java, .NET, .NET Core

Description

It is possible to vary an image at runtime not only according to the Style, Language, Density, Layer, but also to the Options of the Tokens set (only when the Style property specifies a Design System Object that has parameters for that Tokens).

The Options property allows you to specify a value for each option (parameter) of the Tokens Set, in order to use the current image when these options values are the current ones in runtime.
