Map Type property in Grids with Control Type = Maps

Sets the type of map to display.

Values

Hybrid Displays a mixture of normal and satellite views.
Satellite Displays satellite images.
Standard Displays the default road map view.

Scope

Generators: Android, Apple
Controls: Grid (Control Type: Maps)

Description

This property is visible only for Grids whose Control Type property = Maps.

It allows selecting one type of map to be displayed.

It is an interface that defines the display and usage of map tiles.

The different Values are given by the Maps Platform provided.

Run-time/Design-time

This property applies both at run-time and at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.
