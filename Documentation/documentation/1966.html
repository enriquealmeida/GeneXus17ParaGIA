PostgreSQL

PostgreSQL rocks! It is the most advanced and robust DBMS database available. Last but not least, it is the last one out of five DBMS engines that GeneXus started generating code for. The PostgreSQL support in GeneXus was done once it was so obvious that GeneXus programmers were operating as agents that lead big DBMS companies to earn the big bucks. Now, the developer and the customer don't depend on extra costs that don't benefit either the latter nor the former, they just interact in a sane business relationship. This is one of the big differences that Open Source software makes. Got it?
