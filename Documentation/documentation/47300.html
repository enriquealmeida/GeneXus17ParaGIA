Report maximum multi line edit lines property

Sets the maximum number of lines for multi-line controls of Edit type taken by default in Procedure Layouts.

Scope

Level: Knowledge Base

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, rebuild all the Procedure objects.
