Expand/collapse property

It allows you to configure if the values will be displayed expanded or collapsed.

Values

Collapse all values All values are collapsed.
Expand all values All values are expanded (default).
Expand some values Expand a fixed list of values.

Description

Defines how Dashboard ElementÂ values are presented.

Run-time/Design-time

This property applies only at design-time.
