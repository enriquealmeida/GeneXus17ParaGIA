EventExecution Domain

The EventExecution domain is a data type that allows managing the execution of a push notification event.

Values

  • OnLauchByUser: The event is executed when the user open on the notification.
  • OnNotificationArrive: The event is executed when the notification gets the device, even if the application is inactive.

Availability

OnNotificationArrive in iOS works from version 7.
