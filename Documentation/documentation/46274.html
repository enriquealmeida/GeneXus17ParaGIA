Ads Provider property

Ads Provider used for AdsView controls.

Values

AdMob AdMob Provider
GoogleMobileAds Ad Manager
None No provider

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Android, Apple
Controls: Ads View

Description

This property is available at the object level and at the control level.

If you set it at the object level, the value applies to all the Ads View controls present in the object. 

You can also set for a specific Ads View control a different provider than the object provider.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
