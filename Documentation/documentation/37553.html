Minimum Height property

This attribute value is used to determine the number of rows in a Smart Grid with Multiple by Size in Items Layout Mode and Horizontal scroll direction. The grid will fit as many rows as possible with at least this height, 0 means to use all height.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Platforms: Smart Devices(Android, IOS)
Controls: Grid
