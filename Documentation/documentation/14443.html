About GeneXus Upgrades Installation

Disclaimer: The below Q&A refers to X Evolution 1 but also applies to newer versions.

Let's suppose you have GeneXus X Evolution 1 Upgrade N installed and you want to install Upgrade M, M > N. How to do?

1) If M is the latest one, download it from http://www.gxtechnical.com/xev1/download (i.e. http://www.gxtechnical.com/xev2/download , http://www.gxtechnical.com/xev3/download, http://www.genexus.com/v15-download, http://www.genexus.com/v16-download  )

If not, ask support to enable the one you want. www.genexus.com/issuetracking, and download it.

2) Optional: Backup your installation making just a copy of your installation directory. You will be able then to execute GeneXus.exe of your Upgrade N at any time.

3) Install Upgrade M.

4) Rebuild All.

Let's suppose you have GeneXus X Evolution 1 Upgrade N installed and you want to install Upgrade M, M < N. How to do?

1) Ask support to enable the one you want. www.genexus.com/issuetracking, and download it

2) Optional: Backup your installation making just a copy of your installation directory. You will be able then to execute GeneXus.exe of your Upgrade N at any time.

3) Install Upgrade M. The setup will uninstall your Upgrade N and then install upgrade M.

4) Rebuild All.

Note: The setup will not, in any case, uninstall or update the Extensions, Patterns or User Controls. The ones compatible with upgrade N may not be compatible with upgrade M. You will need to install the correct ones from the start page, Add-Inn Manager or from http://marketplace.genexus.com. Note that if the Extensions or Patterns are incompatible with Upgrade M, they won't load. But if the User Controls are incompatible, you will note that at specification time or even in runtime.

FAQ:

Q: If I have no X Evolution 1 installed, do I need to install the X Evolution 1 and then install the latest upgrade?

A: No, the X Evolution 1 Upgrades are complete setups, just install the upgrade and that's it.

Q: If  M > N+1, do i need to install N +1, [....M-1], M ?

A: No, just install upgrade M.
