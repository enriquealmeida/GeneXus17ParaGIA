Bold Property

It specifies whether the cells are shown in bold or not.

Syntax

&ExcelDocument.Cells(…).Bold
&Excelfont.Bold


Type Returned:
Numeric

Values

0: This value indicates that not all the cells are in bold.
1: This value will be returned only if all the cells in the ExcelCells object are in bold.

Scope

Extended Data Types: ExcelCells ExcelFont Data Type
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
