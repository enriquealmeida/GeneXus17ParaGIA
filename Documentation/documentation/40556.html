Items Layout Mode property

This attribute determines how the items will be displayed in the Smart Grid control.

Values

Multiple by Quantity This value indicates that a fix number of items will be displayed in each row/column of the grid. It will be determined by the Items Per Row or Items Per Column properties, depending of the grid Scroll Direction.
Multiple by Size The number of items displayed in each row/column will be determined depending of each item size. To achieve that, the values of the properties Maximum/Minimum Width and Maximum/Minimum Height are used.
Single One item per row/column will be displayed
Staggered by Quantity The grid is displayed with a staggered layout, where each item can have different size

Scope

Platforms: Smart Devices(Android, IOS)
Controls: Grid
