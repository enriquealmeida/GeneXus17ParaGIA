Compare with server object

When a working copy Knowledge Base is connected to a GXserver repository, the Compare with server object Team Development contextual menu is enabled to show the difference between the object latest revision against the revision available in the GXserver repository. Notice the Team Development output will display the following:

========== Get server object started ==========
Exporting Web Panel 'ObjectName'...
Get server object Success
