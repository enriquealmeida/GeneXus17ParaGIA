Web maximum single line edit length property

Sets the maximum length in characters for controls of Edit type. If it is exceeded, it is converted into a multi-line edit.

Scope

Level: Knowledge Base

Description

This property is used to define the default control for an attribute.

If the attribute length exceeds the value of the Web Maximum Edit Length Property but does not exceed this property value, it is defined as a single line edit with horizontal scrolling. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
