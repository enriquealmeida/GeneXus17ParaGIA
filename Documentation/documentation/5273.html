What is a User Control?

In addition to using the standard GeneXus controls, GeneXus supports the use of custom controls.

These controls are called User Controls (also known as Extended Controls) and allow youÂ to create applications with rich, user-friendly interfaces, for example, by adding menus, maps, charts, calendars, diagrams, and many other resources.
