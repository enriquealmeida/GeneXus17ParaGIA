Continent property in QueryViewer

Specifies the continent map that will be displayed.

Values

Africa
Antarctica
Asia
Europe
North America
Oceania
South America

Scope

Generators: Java, .NET, .NET Core
Controls: QueryViewer

Description

Continent map that will be displayed in the QueryViewer when choosing to display a continent through the Region property.

Run-time/Design-time

This property applies both at run-time and at design-time.

Samples

QueryViewer1.Continent = QueryViewerContinent.NorthAmerica

Availability
