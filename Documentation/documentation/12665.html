Delete column tooltip text property

To enter the description to be displayed to the user, when the mouse cursor passes over the first column in the transaction grids, which offers to delete a row.

Description

Each row of a grid in a Transaction has a column with a control that offers to delete the row.

This property allows the developer to enter the description to be displayed to the user when the mouse cursor passes over that column.

It has no default value. It's empty.
