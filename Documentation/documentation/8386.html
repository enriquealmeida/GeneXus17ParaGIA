Encrypt64 function

Encrypts a text using a key.

Syntax

Encrypt64 (&text, &key)

Where:
&text
     Is the parameter to be encrypted

&key
    Is the key to be used for the encryption, generated by the GetEncryptionKey function.

Type Returned:
Character
     Is the result of the parameter encryption, using the key generated by the function. 

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Description

It encrypts the value received as parameter, using the key indicated. The encryption is based on the Twofish algorithm using a default key of 32 characters.
