Date format in CTOD function property

Sets the format in which the CtoD function will interpret the parameters and also the resulting string of DtoC Function.

Values

mm/dd/yy month/day/year
mm/dd/yyyy month/day/year
Language Dependent The date type format will depend on the language selected for the current model. This is the default value.
dd/mm/yy day/month/year
dd/mm/yyyy day/month/year
yyyy/mm/dd year/month/day
yy/mm/dd year/month/day

Scope

Generators: .NET, .NET Framework, Java
Level: Back end, Front end, Generator
