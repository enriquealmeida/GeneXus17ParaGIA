HowTo: Convert a Folder into a Module

In order to convert a Folder object into a Module object, you can use the Convert to module option.

Note: It's not possible to turn a Folder into a Module if the folder contains Win type objects.

Steps

  1. Right-click the Folder object you want to convert into a Module.
  2. Select the Convert to module from the contextual menu.

When the conversion process is a new Module with the same Name property as the selected Folder will be created containing all the Folder' Objects and the Folder will be deleted.
