.Net Core External Type property

Data type name as it is known within the external language (.Net Core).

Description

This property can be set for properties, methods, events, and parameters that are defined in External objects to indicate a data type name, as it is called in the external platform.

That is, in addition to setting an internal data type, by means of this property, you complete the data type name as it is known within the external language (.Net Core).

An example value can be: System.Date.

 

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: External Object
Platforms: Web(.Net Core)
