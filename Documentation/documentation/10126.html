Class Conflicts Property

The property applies when importing a Theme and there is another Theme with the same name in the knowledge base.

In case a class has different values in both revisions, the Class Conflicts property will be taken into account to decide how to do the merge of both themes.

Values

UseFromExport The class settings of the Theme being imported overwrites the settings of the existing Theme. This is the default value.
UseExisting The class settings of the existing Theme overwrites the settings of the Theme being imported.

Scope

Action: Import option
