Server Connection Type property

Defines the Authentication Type to be used when connecting to the GeneXus Server.

Values

GXTechnical gxtechnical.com users will be used for Authentication.
Local In its database, GeneXus server has all the information on the users that have been created in this manner (local), when selecting this value that user information will be used for Authentication.

 

This property is defined when the Send Knowledge Base to GeneXus Server or the Knowledge Base from GXserver (X Evolution 2) actions are performed. It defines the Authentication Type to be used when connecting to the GeneXus Server.
