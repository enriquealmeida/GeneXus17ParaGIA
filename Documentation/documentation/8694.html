BackColorEven property

To allow modification of the back color of even lines in a grid or free style grid, with the value ‘Report’ in the BackColorStyle property.

Syntax

control. BackColorEven = expression

Type returned: Numeric

Where:
control
    Is the name of a control inserted in the form.

expression
    Is a numeric representing an RGB color.

Run-time/Design-time

This property applies only at run-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
Controls: FreeStyle Grid, Grid
