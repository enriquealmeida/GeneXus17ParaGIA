iOS Generator Extraction Directory property

Indicates the extraction path of a File object when iOS (Swift) Generator is enabled.

Description

This property is visible whenÂ Extract for iOS Generator property is enabled (True value). By default, it has an empty value taking as root directory the one indicated on Target Path property followed by "/mobile/iOS" subdirectory.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Objects: File
Platforms: Smart Devices(IOS)
