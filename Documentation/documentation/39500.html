Java Generator Extraction Directory property

For indicating the extraction path of a File object when using Java Generator.

Description

This property is visible when Extract for Java Generator property is enabled (True value). By default, it has an empty value taking as root directory the one indicated on Target Path property followed by "/web" subdirectory.

Notes

  • This property replaces Extract to path property when Extract property is different from "No" value and Generator property has "Any" or "Java" value.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Objects: File
Platforms: Web(Java)
