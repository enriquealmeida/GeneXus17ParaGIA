Initial Zoom property

Indicates how to apply the initial zoom on the map.

Values

Nearest point visible The map is adjusted to show the current device location and the nearest point.
No initial zoom No action is taken to adjust the map zoom.
Radius The initial zoom is indicated with a fixed radius in meters.
Show all points The map is adjusted to show all the points loaded in it.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(Android, IOS)
Controls: Grid
