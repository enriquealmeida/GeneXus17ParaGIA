Transaction Form

Every time a Transaction is defined, a default form is automatically created by GeneXus to specify the way the user will access data in graphic applications. This form can be modified. Once the screen is edited and saved, the saved version will be the one that is always shown. Note that all the attributes included in the form must first be declared in the Transaction structure.

Transactions can have several Forms.
