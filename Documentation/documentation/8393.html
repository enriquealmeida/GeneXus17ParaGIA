FTP functions

Alert: FTP does not encrypt the channel. It is recommended to use a protocol that encrypts the channel such as SFTP or FTPS.

These functions allow you to perform some actions on the servers via FTP (File Transfer Protocol) technology.

You can connect to a server, delete a file, determine if an error occurred in any executed command, upload and download, etc.

GxFtpCon
GxFtpDelete
GxFtpDis
GxFtpError
GxFtpGet
GxFtpPut
