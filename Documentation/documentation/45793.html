SAP Cloud Foundry Organization name property

Sets the organization name for SAP Cloud Platform Cloud Foundry Environment.

Scope

Level: Deploy Target Options

Description

You must specify the Organization name for your account in SAP Cloud Platform Cloud Foundry environment. You can get this information in the SAP Cloud Platform Cockpit (control panel),Â in the Overview section of your Cloud Foundry environment.

Availability

This property is available since GeneXus 16 upgrade 10.
