Stripe Subscription Item Object

Subscription items allow you to create customer subscriptions with more than one plan.

To create a SubscriptionItem use:

  • Create Options: StripeSubscriptionItemCreateOptions
  • Response: StripeSubscriptionItemCreateResponse
  • Errors: StripeError

To update a SubscriptionItem use:

  • Update Options: StripeSubscriptionItemUpdateOptions
  • Response: StripeSubscriptionItemUpdateResponse
  • Errors: StripeError

To retrieve a SubscriptionItem use:

  • Response: StripeSubscriptionItemRetrieveResponse
  • Errors: StripeError

To delete a SubscriptionItem use:

  • Response: StripeSubscriptionItemDeleteResponse
  • Errors: StripeError
