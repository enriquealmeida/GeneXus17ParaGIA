Default Platform Hint property

Deprecated: Since GeneXus 17 Upgrade 9. Read SAC #51065.

Hints the generator in layout selection when there is no specific Web option.

Values

Android Selects the layouts defined for Android, and uses the default theme specified for the Android platform.
Best fit Selects, from all the platforms, the layouts to be used.
iOS Selects the layouts defined for iOS, and uses the default theme specified for the iOS platform.
