ShowPageController property for Smart Devices

Specifies whether or not the page controller will be displayed

Syntax

control. ShowPageController

Values

False
True

Description

This property allows turning invisible the page controller in horizontal grid. 

It can be both read and write. For example:

Grid1.ShowPageController = True

&isPageControllerVisible = Grid1.ShowPageController //&isPageControllerVisible will be true.

Run-time/Design-time

This property applies both at run-time and at design-time.
