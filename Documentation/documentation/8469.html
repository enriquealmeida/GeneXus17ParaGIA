NewLine function

Returns the string needed to 'skip a line' on the platform that it runs.

Syntax

NewLine()

Type returned:
Character

Scope

Objects: Procedure, Transaction, Web Panel, Panel, Work With for Smart Devices, Work With for Web 
Generators: .NET, Java, Apple, Android

Samples

msg(Trim(&UserName) + '  you are registered to the meeting.' + NewLine() + '  See you there!.')

TextBlock.Caption = 'Welcome' + Newline() + &UserName
