Indicator Unselected Color property

Sets the color of the unselected pages indicator in an SD Paged Grid.

Description

This property can be set for grids whose Control Type property = SD Paged Grid.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, Build a main object.

Scope

Controls: Grid
