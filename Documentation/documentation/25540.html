Module/Folder property

Sets the Module or Folder to which the GeneXus object will belong.

Scope

Objects: Data Provider, File, Procedure, Transaction, Web Panel, Data View, Stencil

Description

This property allows you to indicate the Module or Folder to which the GeneXus object will belong.

Folders establish hierarchies where each folder can contain GeneXus objects as well as other folders — but no Module Objects.

Modules also establish hierarchies where each Module can contain GeneXus objects as well as other folders. Please read Modules for further details.

Run-time/Design-time

This property applies only at design-time.
