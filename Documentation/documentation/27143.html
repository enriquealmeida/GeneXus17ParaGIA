Border Radius property

Makes rounded the corners of the controls.

Description

It can be found for certain controls in Theme object and Web Theme object.

Notes

  • In Android, the images must be square shaped in order to fit a perfect circle; otherwise, the final shape will be an ellipse.
  • In iOS, this property must be set when a control is expanded beyond the limits of its container (e.g. a bigger image than the table containing it).

Scope

Objects: Theme for Smart Devices, Theme
