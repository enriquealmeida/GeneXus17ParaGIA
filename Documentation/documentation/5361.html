Data Selector property

Allows you to indicate the name of a Data Selector object defined in the Knowledge Base.

Description

Applies to Grids, Data Selectors and the elements of the WWSD. This property allows you to indicate the name of a Data Selector defined in the Knowledge Base, in addition to the "none" value.

At specification time, the Data Selector definition is combined with the grid that references it (or with the Data Selector that references it) to determine the resulting navigation, which will be a combination of both (Data Selector definition + Grid definition or Data Selector definition + Data Selector definition).

When used in a Data Selector, the property value cannot generate a recursive reference either directly or indirectly.
