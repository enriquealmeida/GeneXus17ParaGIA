Dashboard Card Include sparkline property

Specifies whether to include a sparkline chart for the values or not.

Description

This property is only available when the Type property has the Card value assigned and allows displaying (under the Card indicator) a Sparkline chart showing the indicator's evolution.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Dashboard
