ServerNow function

Returns a DateTime value corresponding to the current date and time of the DBMS.

Syntax

ServerNow()

Type Returned:
DateTime

Scope

Objects: Procedure, Transaction, Web Panel, Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Note: The following warning is displayed when using the function in a Panel with its Connectivity Support property set to Offline:

warning: spc0210: 'ServerNow()' not supported, using 'Now()' function instead.
