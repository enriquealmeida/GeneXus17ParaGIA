Next Event

To bring up the next record in the table, according to the current index.

Description

The GeneXus developer cannot associate any code with this event.

This event occurs when the user clicks on the associated button 

Scope

Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
