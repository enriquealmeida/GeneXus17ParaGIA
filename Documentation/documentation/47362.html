Task Metadata Id property

This is the internal metadata ID for the task definition.

Scope

Objects: Business Process Diagram
Controls: Task

Description

This is the Workflow metadata IDÂ for the Business Process Diagram's task selected.

It is the value stored in the Workflow database.

Run-time/Design-time

This property applies only at run-time.

How to apply changes

No action is required to apply the corresponding changes when the property value is configured.

Availability
