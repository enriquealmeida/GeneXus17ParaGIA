Purpose domain

Enumerated domain indicating the purpose of a data item in the context of GeneXus Artificial Intelligence for custom models.

Values

Train The data item is used for training.
Test The data item is used for testing.
Validation  The data item is used for validation.

Scope

Data type Data data type
Platforms  Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This domain is available as of GeneXus 16 upgrade 6.
