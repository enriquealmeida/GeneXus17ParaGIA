Identifier Property

Allows assigning a name to a table, row or cell that can then be used in JavaScripts to modify its properties in runtime. 

Description

This property applies only at design time.

Scope

Objects: Transactions, Web Panels
Controls: Embedded Pages, Forms, Tables, Tables’s Cells, Tables’s Rows
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
Interfaces: Web
