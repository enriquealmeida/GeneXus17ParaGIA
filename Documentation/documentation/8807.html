ItemDescriptions property

Assign a description to an attribute/variable defined under 'Dynamic Combo Box' or 'Dynamic List Box' values in the 'ControlType' Property or when the InputType property has the 'Descriptions' value.

Description

Attribute description for the dynamic combo box and dynamic list box only, or edit field depending on the InputType property value. This description is what will be displayed on the screen instead of the source data value (this is, the attribute/field that has been changed).

Applies to Dynamics Combo Boxes and Dynamics List Boxes.

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Procedure, Transaction, Web Panel
Platforms: Web(.Net, Java)
