GAM - Applications Registration option

This is a menu option: Tools -> GAM -> Applications Registration.

Registers all defined GAM Applications in the KB in the GAM Repository.

When performing a “Rebuid”, the GAM Applications Registration process is forced.

The option in the Tools –> GAM – >Application Registration menu is to force the same process (this step may be necessary when changes are made to the connection properties of the GAM data store, which doesn’t require a Rebuild for the rest of the objects in the KB because it can be costly).

At the IDE level, forcing application registration implies updating the Client Id and Client Secret information if it is neccessary to do it. These data will be used later to identify the application in GAM.

Note

After executing this menu option you need to Build the main dashboards.

See GAM Applications Registration process for more information.
