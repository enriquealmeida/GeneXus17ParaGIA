Manually configuring Tomcat

From Genexus 17 Upgrade 5 the Tomcat path property is implemented that replaces the Servlet directory property and the Static content directory seen from client property.

When changing the Tomcat path property to a new tomcat server after a rebuild all the application is generated in the new path and no manual configuration is needed.

For information on how to configure older versions please see Manually configuring Tomcat
