Background color property for QueryViewer maps

Specifies the map's background color.

Scope

Objects: Design System, Theme
Generators: Java, .NET, .NET Core

Description

Use this property to select a color for the map's background. The background may also be specified as transparent (or partially transparent used in combination with the Opacity property).

How to apply changes

To apply the corresponding changes when the property value is configured, Run the main object.
