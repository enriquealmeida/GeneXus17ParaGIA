On item click property

Action to be executed when clicking an item.

Values

Apply filters Use the clicked item as a filter in the remaining dashboard elements.
Do nothing No action taken (this is the default).
Highlight values Use the clicked item to highlight that item wherever it appears in other dashboard elements.
Raise ItemClick event Raise the Itemclick event for those attributes that have the Raise ItemClick Event property on true

Description

By default, no action is taken when clicking on an item of a query contained in a Dashboard. Use this property to assign a behaviour to that action. You can choose to use that item as a filter in the remaining Dashboard elements, or highlight that item wherever it appears throughout the Dashboard.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16 upgrade 4.

Scope

Objects: Dashboard
