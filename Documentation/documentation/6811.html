Remove method

Removes a session value.

Syntax

&WebSessionVble.Remove(Key)

Where:
&WebSessionVble
    Is a variable based on the WebSession data type.

Key
    Is the Session key. It must be a String.

Scope

Extended data types: WebSession data type
Generators: .NET, Java, Ruby
