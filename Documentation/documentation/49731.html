Setup Command property

Sets that for each main object an install script must be run once.

Scope

Generators: Angular
Level: Generator

Description

The default value for this property is "npm install -f."Â When a main object is executed, it checks if this command has not been run before and runs it. It will install all the dependencies required to run properly.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 Upgrade 5.
