Sysdate function

Returns the system date.

Syntax

Sysdate()

Type Returned: 
Date

Scope

Objects:  Procedure, Transaction, Web Panel

Note: In the iSeries Environment, this function returns the System Date of the client’s machine, and not the start date of the Job. See the Today function to use the Job's start date.
