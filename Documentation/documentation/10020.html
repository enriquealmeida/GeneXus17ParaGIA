Status Bar Information

Purpose

The Status Bar is used by GeneXus to display status information and messages at the bottom of the window.

It provides statistics about the position of the cursor and the status of some important keys.

Information

Knowledge Base directory: Indicates where the Knowledge Base directory is located.

Lin: Indicates the number of line positions advanced from the top of the page to the current position (or insertion point). This information will be shown if you are in one of the object editors, such as the Report Source Editor or Transaction Rules Editor.

Col: Indicates the number of character positions advanced from the beginning of the line to the current position (or insertion point). This information will be shown if you are in one of the object editors, such as the Report Layout Editor.

Ch: Indicates the character number where the cursor is positioned within the current line.

Build information: [Objects already specified / Objects with warning / Objects with error / Total objects to specify]  [Objects already generated / Total objects to generate]

A click over that numbers change info to : Time elapsed / Time remaining. Double Click shows navigation views of already specified objects.

INS/OVR: Indicates whether editing is being done in Insert or Overwrite mode.
