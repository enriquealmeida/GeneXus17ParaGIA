Apple Maps API property

Displays maps in Apple Maps.

Values

Apple Maps Default value. Uses Apple Maps.
Google Maps Uses Google Maps. Apple Maps API Key property must be set with a Google key.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)

Description

This property allows you to use another maps provider different from Apple Maps (e.g. Google Maps). When you change its default value, the Apple Maps API Key property will be displayed to set the access key.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 15 Upgrade 11.
