Grid Refresh method

This command refreshes only the corresponding grid and also triggers the refresh event.

Syntax

<Grid>.Refresh()

Example

In the example explained in HowTo:Develop a messaging web page each time a new post is added by a user, the Grid which displays the posts is refreshed but not the whole page where it is contained.

Event 'Postcomment'
     InsertNewComment.Call(PostId, &newcomment)
     &newcomment=""
     Grid1.Refresh()
Endevent

Scope

Objects: Web Panel object, Panel for Smart Devices, Work With for Smart Devices
Controls: Grid control, Grid control in Smart Devices
Languages: .NET, Java - The method is supported when Web User Experience property is set to "Smooth"
SD Generators: iOS, Android
