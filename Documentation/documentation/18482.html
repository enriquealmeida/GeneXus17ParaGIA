HowTo: Using Check Box for Smart Devices

Check Box is a control applied for Win, Web and SD applications. It is used to show two possible values for a variable or attribute, in a checked-unchecked way. In this tutorial we will explain how to use a Check Box in Smart Devices with a simple example.

Properties

Auto Grow If this property is true then the field will adjust the length of the attribute.
ControlTitle Used to define a title to the control.
CheckedValue This property defines the value that will be used when the Check Box is checked.
UncheckedValue This property defines the value that will be used when the Check Box is unchecked.
