Grid Load method

Loads a new line to the Grid.

Syntax

Grid.load()

Example

Event Grid1.Load
    For each InvCode
        &InvCode = InvCode
        &InvDat = InvDate
        &Type = "INV"
         …
        grid1.Load( ) 
    EndFor
EndEvent

This method is very useful when you have more than one Grid in the form, in this case, you specify to which Grid the line is loaded:

Grid1.Load( ) or Grid2.Load( )

Important:

  • For Win applications, this method applies to the Grid.Load Event.
  • In case of Web applications, the Load method is allowed in user events. SeeLoad Command and Load Method in User Events for more details.
  • For SD applications, this method is not offered nor allowed. You only can use the Load command .

Scope

Objects: Web Panel object, Work Panel object
Controls: Grid control
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
Interfaces: Win, Web
