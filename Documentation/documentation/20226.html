Empty Grid Text Class property

Indicates the Theme Class to be used when showing the Empty Grid Text in the Grid of a Front end layout.

Scope

Generators: Android, Angular, Apple
Controls: Grid

In the following example the "TextBlock" is selected to show the background text ("No data Found") of the Grid.
