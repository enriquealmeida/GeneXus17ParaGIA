HowTo: Get a Google Maps API Key for iOS

To use Google Maps in any compiled iOS application you must have an API Key from Google. If not, maps will not show when using the Maps Control Type, PickLocation Method (from Geolocation external object) or Geolocation domain. 

 

Step 1 - Create a project

Go to Google API console and if you do not have an existing project (e.g. for Android application) create a new one by clicking on "New project" option.

 

Step 2 - Enable Maps API for iOS

Click on "Enable APIs and Services" option.

Then, look for "Google Maps SDK for iOS" service.

Finally, enable it.

 

Step 3 - Retrieve the API Key

Go to "Credentials" option on "APIs & Services" section and look for the iOS key. Then click on copy button ().

Step 4 - Optional - Add your restrictions

By clicking on the API Key name, you are able to restrict access to your application (by default it is unrestricted, displaying a "warning" icon).

Once your requirements are satisfied, click on "Save" button.

Step 5 - Set Apple Maps API Key property

Go back to your GeneXus Knowledge Base and set Apple Maps API Key property by pasting the value that you had copied in step 3.

Availability

This mechanism applies as of GeneXus 15 Upgrade 11.
