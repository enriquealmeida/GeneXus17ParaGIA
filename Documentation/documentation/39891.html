Unique property

Allows setting the attributes that must not have duplicated values in the output of the corresponding query.

Description

Controls the query unique fields ofÂ grid controls of any type. It is equivalent to Unique Clause.

This property does not apply to grids in Transaction object.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, Build a main object.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Master Page, Panel for Smart Devices, Web Component, Web Panel, Work With for Smart Devices
Controls: FreeStyle Grid, Grid
