Supertype property

Indicates the attribute’s supertype if applicable.

Description

If the attribute is a subtype, the property shows the associated supertype.

The Data Type, , Autonumber, Value Range and Picture properties are disabled, since they are inferred from the attribute indicated in this property.

Only valid for attributes. This property is read-only.

Values

Attribute supertype.

Scope

Controls: Attribute
