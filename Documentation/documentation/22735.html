PageChanged event

To take place right after the user changed the current page of the grid by swiping it.
It might be helpful for developers who design UI elements such as "Page 9 of 13".

Syntax

Event GridControl.PageChanged
        Event_Code
EndEvent

Where:
GridControl:

Is the name of the grid inserted in a layout and GridControl Control Type property is one of the controls declared in the Scope below.

Event_code:

Code executed when the event triggers.

Example

Event GridImages.PageChanged
    &CurrentPage = Grid1.CurrentPage
EndEvent

Scope

Objects Panel object, Work With for Smart Devices object
Controls SD Paged Grid, SD Horizontal Grid, SD Magazine Viewer
Platforms Apple platform, Android platform

Notes

  • On this event, the developer can access attributes/variables values displayed in the first ítem of the grid and the whole content of the form.
