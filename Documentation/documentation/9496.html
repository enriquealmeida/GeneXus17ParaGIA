Data View Index Information

Purpose

To allow configuration of a Data View index.

Name

Specifies the index name that GeneXus will use internally in the application; this name can be different from the one of the associated table.

Primary Key/Unique/Duplicate

Specifies the uniqueness of the key.

Composition

Is the ordered list of Data View fields that are a part of the Index. The attributes used to define the index must be a part of the attributes defined in the Data Viewâ€™s composition.

The attribute's order can be changed (ascending or descending). All attributes belonging to the Primary Key and Foreign Keys of associated tables must have ascending order.
