Workspace Id property

Watson Assistant Workspace Id. If this value is empty, or the Workspace doesn't exist in your Watson Asisstant service, GeneXus will create a new Workspace.

Description

In the Conversational Flows object the NLP Provider property can be set to "Watson" value. In that case, the Workspace Id property indicates which is the workspace to be used in Watson. If you leave the property empty (or set it to a non existent value), a new workspace will be created.

It has to be a valid GUID.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows
