Integer method

Returns an integer value representing the integer part of the received parameter.

Syntax

numeric-expression.Integer()

Type Returned:
Numeric

Where:
numeric-expression
   Must be a numeric expression.

Description

This function returns the integer part of numeric-expression.

Examples

&Val = 5/3
&IntVal = &Val.Integer() // Result &IntVal: 1

&Val = -1.75
&IntVal = &Val.Integer() // Result &IntVal: -1

&Val = 1.5
&IntVal = &Val.Integer() // Result &IntVal: 1

Scope

Objects    Procedure object, Transaction object, Web Panel object
