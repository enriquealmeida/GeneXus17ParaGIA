Filter control type property

Filter's control type.

Description

Defines the filter's control type. The available values for the property varies depending on the filter's type and the filter's data type properties:

  • Edit, valid for any filter
  • Radio button, valid only for value type filters
  • Combo box, valid only for value type filters
  • Check box, valid only for boolean value type filters or any collection filter
  • Slider, valid only for numeric, date or date-time non-collection filters
  • Drop-down list, valid only for collection filters
  • Switch, valid only for boolean value type filters

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Dashboard
