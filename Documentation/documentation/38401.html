Unit Test object

Defines a Unit or Integration Test on programs with business logic like Business Components, Data Providers, and Procedures.

You can run the defined tests using the GeneXus IDE or automatically on different environments when using CI/CD capabilities.

You have two ways to create a Unit Test object:

1) In the same way that you create any GeneXus object.

2) In the KB Explorer you can right-click on the desired Procedure, Data Provider or Business Component and select in the contextual menu: Create Unit Test.

More information at Unit Testing.
