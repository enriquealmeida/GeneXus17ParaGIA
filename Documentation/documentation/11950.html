HowTo: Hiding the GXflow Header

To hide the GXflow header you can do the following:

Add this code at the end of the gxui-all.js file.

gxui.afterShow(function(){
Ext.getCmp('gxui--wfmain-Layout-North').hide().collapse();
}, window);

You can find this file in:

  • .NET applications: <KB>\CSharpModel\Web\gxui
  • Java applications: ...\Apache Software Foundation\Tomcat XX\webapps\<genexus_application_directory>\static\gxui (remember the web server reload).
