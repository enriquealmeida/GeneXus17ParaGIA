ExistsAttribute method

Indicates if there is an attribute in the current node.

Syntax

&DataType.ExistsAttribute(Name)

Where:
Name
    Is the attribute's name 

Type Returned:
Numeric

Scope

Extended Data Types: XmlReader
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Description

It returns 1 when the attribute called Name in the current node obtained through the Read or the ReadType method exists. Otherwise, it returns 0. It is valid only for nodes of the Elements type.
