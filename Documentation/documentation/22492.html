Modules - Known Limitations

As of GeneXus 15:

General

  • SDT 'Messages' must belong to the Root module.
     
  • Generators not supporting Module objects: Cobol and RPG — see Generators supporting modules.

GeneXus Access Manager (GAM)

  • GAM API external objects must belong to the Root module.
     
  • Permission Prefix property for two Objects with the same name is defined with the same value since, by default, the value is the name of the object. In other words Permission Prefix property is not affected by the Qualified Name property.
