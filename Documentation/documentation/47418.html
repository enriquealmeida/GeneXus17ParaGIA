GeneXus 17 upgrade 2

This article is an overview of GeneXus 17 Upgrade 2 features (compared to GeneXus 17 upgrade 1) and what needs to be taken into account to adopt it.

Upgrade 2 has been releasd on February 24th, 2021.

Download

https://www.genexus.com/developers/DownloadCenter?,,,5913 

Overview

This upgrade includes several improvements related to developer experience and adds new capabilities to the generated software solutions, like the generation for Huawei Mobile Services Platform, sophisticated APIs in Java, ODATA services integration in .NET Core, or the integration of Queries in mobile Chatbots.

IDE

  • Several stability, performance and usability improvements

Chatbots

  • Queries can now be shown in mobile chatbots

API Object

  • It is now supported in Java as in .NET and .NET Core
  • Improvements related to Docker and .NET Core, and also security (Authorization)

Android

  • Android Baidu Maps SDK updated to version 6.4.0 (SAC#48735).
  • Huawei Mobile Services Platform support
  • App bundles generation support 
  • Improvements in permissions management (storage, background location)

Java

  • API object support added

.NET Core

  • OData Support in GeneXus added

BPM

  • A specific Business Process Diagram object can now be referenced from code; more information at Workflow Data Types.

Reporting

  • In mobile, support for improved syntax: QueryViewer.Object = Query1(par1, par2, ...) 

Testing

  • Integration with Allure 
  • Several improvements for developer experience related to UI testing

Compatibility

  • New Android Requirements
  • New spc0235 warning in spec regarding load commands or methods in grids without base table: SAC 48736
