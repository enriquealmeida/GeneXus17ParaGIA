Empty Grid Text property

Shows a text when the Grid has no data.

Scope

Generators: .NET, Android, Apple, Java
Controls: Grid

Samples

Consider an example where you want to show a list of clients, so you apply the Work With for Smart Devices object to the Client Transaction and add an entry point in a Menu object pointing to the WorkWithDevicesClient.Client.List(). If there are clients in the data base and you go to the WorkWithDevicesClients screen, the list will be shown; however, if you don't have any records the list will appear empty and sometimes the user may think there is a problem.

In sum, this property allows you to show a text instead of an empty space.
