Header Text Class property

Class to set the style of the table or grid header.

Description

This property sets the Text Block class that is used to set the style of the header of the table or grid.

The default value is "TableHeaderText" (which by default is set to 'display:none' for compatibility reasons)

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since Genexus 15 Upgrade 10.

Scope

Objects: Theme
Platforms: Web(.Net, Java)
Controls: Grid, Table
