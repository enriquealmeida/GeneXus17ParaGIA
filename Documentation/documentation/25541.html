HowTo: Creating a Module Object

This article explains how to create a new Module object.

Creating a Module Object in the Root module.

Steps

  1. Right-click on the Root module in the KB Explorer.
  2. Select the New -> Module option.
  3. Type a name for the newly created Module Object.

Creating a Module Object in a module other than the Root module

Steps

  1. In the KB Explorer, right-click on the Module Object where you want to add a new child Module Object.
  2. Select the New -> Module option.
  3. Type a name for the newly created Module Object.
