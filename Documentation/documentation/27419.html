Minimum Time Between Sends property

Determines the time between calls to the synchronization Send services, which are in charge of sending the changes made in the device to the application server.

Scope

Objects: Offline Database
Generators: Android, Apple

Description

This property applies to Offline Native Mobile applications in which an Offline Database object has the Send Changes property set to "When connected."
