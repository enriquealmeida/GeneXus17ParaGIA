LocalName Property

This property returns the name that represents the namespace, without indicating the prefix. 

Syntax

&DataType.LocalName

Type Returned:
String

Description

Returns the name that represents the namespace, minus the prefix.
This property is only valid for element and endtag type nodes.

Scope

Extended Data Types: XmlReader
Languages: .NET, Java, Visual FoxPro (up to GeneXus X Evolution 3), Ruby (up to GeneXus X Evolution 3)
