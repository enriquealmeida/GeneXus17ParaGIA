Style and Default Style properties

Assigns the object (Design System or Web Theme) with which the style will be applied.

Scope

Objects: Transaction, Web Panel
Level: Version

Description

Every Web Panel object and every Transaction object offers the Style property so that you can assign a Design System Object or a Web Theme object to it.

If you do not assign an explicit object to this property, it inherits the object set in the Default Style property (offered at version level). 

To apply the corresponding changes when the Default Style property value is configured, you have to execute a Rebuild All. On the other hand, to apply the corresponding changes when the Style property value is configured, you have to execute a Build All.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 Upgrade 5.
