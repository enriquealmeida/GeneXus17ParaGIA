Geometry Layer Id Attribute property

Sets the attribute or variable that contains the geometry layer identifier for the geometry being drawn.

Description

The Grid Maps control will load a set of Geography values using the Location Attribute. 
These values can be grouped in a logical way from the Attribute/Variable indicated in this property.
This Attribute/Variable is expected to be defined as Numeric or Character data type.
