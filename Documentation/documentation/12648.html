Year method

Returns the year number of a given date. If the received parameter is null, zero is returned.

Syntax

date-expression | datetime-expression.Year()

Type returned:
Numeric N(4)

Scope

Objects:   Procedure, Transaction, Web Panel

Samples

To obtain the current year and assign it to a variable:

&ThisYear = Today.Year()

A string displaying text next to the year:

&TxtYear = 'Year:' + str(&Today.Year())
