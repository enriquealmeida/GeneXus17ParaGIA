HowTo: Resizing font size depending on the viewport size

Viewport units can be used on text to automatically change the font size depending on the viewport. 

This could be very useful when you need to comply with Accessibility for Web Applications issues.

The AttributeBigFonts has its Label Class property set to LabelBigFonts, which is a descendant class of Textblock. The LabelBigFonts Font-Size property is set to 2vmax, as shown in the picture below.
