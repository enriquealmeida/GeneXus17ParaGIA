Target JRE property

Defines the Java Runtime Environment version where the deployed application will run

Values

JRE 6
JRE 7/8
JRE 9 (or higher)

Description

The Target JRE property is used in the deploy engine to determine which jars to deploy when the libraries have different jars depending on the version of JRE that is used at runtime.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Platforms: Web(Java)
