Time domain

The Time domain is a DateTime(0.5) and behaves as the last (it's like a shortcut).

In the Smart Devices applications you will have a special picker that lets your edit just the Time part of a DateTime (all the Datepart is hidden).

Notes:

  • The Date part will be null.
  • The default value for this domain is 00:00
