Enable Notifications property

Indicates whether the application will support Push Notifications.

Values

False
True

Scope

Objects: Menu, Panel (Only Main Objects)
Generators: Android, Apple

Description

Once you set this property to True, another group of properties is available and processes are generated in the application to support notifications.
