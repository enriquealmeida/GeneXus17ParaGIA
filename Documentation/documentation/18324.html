SimpleAndroid Theme for Smart Devices

SimpleAndroid is a Simple Theme for Android Platform, imported by default in GeneXus X Evolution 2.

The following are the classes and properties used.

Attribute • Attribute.Title - font_size: "18dip"
• Attribute.Subtitle - font_size" : "14dip"
Dashboard • Grid Class: "Grid"
• TabClass" : "Tab"
Grid • Odd Row Class" : "GridRow.Odd"
• Even Row Class" : "GridRow.Even"
Tab • Selected TabPage Class: "TabPage.Selected"
• Unselected TabPage Class : "TabPage.Unselected"
Table
Additional subclasses		 • "Table.Detail"
• "Table.Section"
