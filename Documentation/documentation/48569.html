Theme property for Platforms (GeneXus 17 u4 or prior)

Assigns a specific Theme to a certain Platform.

Syntax

control. Theme

Scope

Generators: Android, Apple
Level: Platforms node > Platform

Description

In the KB Explorer, under the Customization node, you can find the Platforms node. 

Under the Platforms node, several subnodes are offered, each one corresponding to a specific platform.

For each platform, the Theme property is offered to assign to it the desired Theme object.

Run-time/Design-time

This property applies only at design-time.
