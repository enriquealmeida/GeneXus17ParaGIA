GeneXus IDE Toolbox

It contains the controls that you can drag and drop when designing a Form or a Layout.

If it is not displayed, you can open it by selecting in the main GeneXus Menu: View / Other Tool Windows / Toolbox.

Depending on the object in which you are working, the Toolbox will show more or fewer controls (because it offers what is allowed according to the context).
