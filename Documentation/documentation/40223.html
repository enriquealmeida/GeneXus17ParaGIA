Generate Foreign Flows property

If it is true, GeneXus will generate the foreign flows

Description

The Generate Foreign Flows property is available for the Transaction node under the Conversational Configuration (Pattern Settings). 

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows Pattern
