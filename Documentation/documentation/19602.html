Disable Expand Collapse property

Sometimes it may be necessary not only to completely expand a query but also not to allow collapsing any of the elements that integrate it, and vice versa. The purpose of this property is to show all the expanded elements from the start with no possibility to collapse them.

Values

False Any element can be expanded and collapsed. This is the default value.
True None of the elements can be expanded or collapsed.

 

Scope

Output type    PivotTable
