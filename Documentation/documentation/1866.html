GeneXus objects

A Knowledge Base is made up of different types of GeneXus Objects.

Most types of GeneXus objects can be created through the New Object dialog. In particular, some types of GeneXus objects can be created using contextual options.

Note

The concept of GeneXus Object has nothing to do with OOP objects.
