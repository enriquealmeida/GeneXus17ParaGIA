Main Platform property (GeneXus 17 Upgrade 1)

When more than one platform is selected (i.e. Android and iOS), it is necessary to set one of them to be considered upon pressing F5 for executing the application.

Values

Android This value is available when the Generate Android property is set to True.
iOS This value is available when the Generate iOS property is set to True.

Scope

Generators: Android, Apple
Level: Front end

Description

When more than one platform is selected (i.e. Android and iOS), it is necessary to set one of them to be considered upon pressing F5 for executing the application. Nevertheless, pressing F5 will generate everything necessary for both platforms.

When you have only one platform, it is assumed by the property by default.
