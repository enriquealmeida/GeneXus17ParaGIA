Google Cloud Project property

Specify the Google Cloud Project Id.

Description

After doing the necessary DialogFlow setup to use it as a provider for your chatbot, specify in this property, the Google Cloud project Id you've defined.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows
