File Size property

Returns the image's file size in bytes. Read-only property.

Syntax

control. File Size

Samples

&fileSize = &image.FileSize

Availability

This property is available since GeneXus 16 upgrade 2.

Scope

Platforms: Smart Devices(IOS)
