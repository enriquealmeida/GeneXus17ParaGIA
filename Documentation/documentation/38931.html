NLP Provider property

Selects the Natural Language Processing provider.

Values

Dialogflow Uses Dialog Flow provider.
None
Watson Uses IBM Watson provider.

Scope

Objects: Conversational Flows Instance
Generators: Java, .NET, .NET Core

Description

The value "None" indicates that a scripted chatbot will be generated.

When you save the instance, the synchronization with the new provider is triggered. So, you don't need to take any other action.

For more information about the configuration to be done, see Configuring GeneXus for using the Chatbot Generator.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

No action is required to apply the corresponding changes when the property value is configured.
