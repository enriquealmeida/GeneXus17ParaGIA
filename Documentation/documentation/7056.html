GetVariable Method

Returns, in a String, the value with which the variable is loaded in the post.

Syntax

&DataType.GetVariable (Character-expression)

Where:
Character-expression
Name of variable

Type Returned:
Character

Examples

Example 1: Getting a control value from a GeneXus generated web page 
&value = &httprequest.GetVariable(CliCod.InternalName)

The value of the screen attribute CliCod is stored in the &value variable.

&value = &httprequest.GetVariable(&Email.InternalName)

The value of the screen variable &Email is stored in the &value variable.

Example 2: Getting a control value from a screen control with name= "txtEmail"

&value = &httprequest.GetVariable("txtEmail")
The value of the screen control with name “txtEmail” is stored in the &value variable.

Scope

Extended Data Types: HttpRequest
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
