Type property (in Web Panels and Transactions)

Sets whether the object type is a Component, Web Page, or Web Master Panel.

Values

Component The object type is a Web Component.
Master Page The object type is a Web Master Panel.
Web Page The object type is a Web Page.

Scope

Objects: Web Panel, Transaction
Generators: Java, .NET, .NET Core

Description

Every Web Panel object and every Transaction object by default is set and generated as a Web page.

However, a Web Panel can be used also to define a Web Master Panel object or a Web Component object while a Transaction can be defined as a Web Component object.

This property allows you to set which is your objective (the type) for the object.

Run-time/Design-time

This property applies only at design-time.
