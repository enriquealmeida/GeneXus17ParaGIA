Kb Language property

Sets the language in which the messages and texts (integrity check failures, default caption button titles, etc.) are automatically generated.

Scope

Level: Knowledge Base

Description

There are a number of general messages and texts (integrity check failures, default caption button titles, etc.) that are automatically included by GeneXus in the generated programs.

This property allows you to choose the language in which you want the messages to be displayed (English, German, Japanese, Spanish, Portuguese, Italian, Simplified Chinese, and Traditional Chinese).

Note: Simplified Chinese and Traditional Chinese languages cannot be used with RPG, Cobol, and VFP generators.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
