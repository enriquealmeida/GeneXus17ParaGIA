Form Control

Form control gives the posibility to manage some elements of the generated HTML Document header for Web Panels and Transactions Web Form. It allows to add javascript references, meta elements or any HTML code to the header and set the title of the document.

In addition, design elements like backcolor, a background image or a CSS class can be applied to the HTML document form using this control.

Form control properties

  • Backcolor
  • Background
  • Caption
  • Class
  • HeaderRawHTML
  • JScriptSrc
  • Meta
  • MetaEquiv
