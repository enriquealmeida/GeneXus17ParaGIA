tvOS Deployment Target property

Sets the minimum tvOS deployment target to use when generating the Xcode project.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple
Level: Front end

Description

This property is available at Front end level and it is also available for Panel objects, Work With objects, and Menu objects configured as Main Object.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
