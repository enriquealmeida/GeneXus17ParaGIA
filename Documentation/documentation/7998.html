Rollback command

Forces a program Rollback.

Syntax

Rollback

Description

The Rollback command invalidates the set of database updates in a Logical Unit of Work (LUW).

It may be used in Procedure or Web Panel objects, but not in Transaction objects.

Note: The Rollback command is ignored by Environments that do not use Transactional Integrity.
 

Scope

Objects Procedure object, Web Panel object
Languages .NET, Ruby, Java, RPG, Visual FoxPro, Cobol
