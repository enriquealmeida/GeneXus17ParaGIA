None Start Event in BPD

It is used where the start of a process is undefined. It is always used to mark the start of the sub-process.

Symbol

Description

A Start Event without a trigger is known as a None. A trigger is not specified (triggers are not mandatory; they can be specified later). They are mandatory in subprocesses.

Scope

Objects: Business Process Diagram
