SimpleElements Property

Determines if the elements with a simple structure are processed as a unique Element type node. 

Syntax

&DataType.SimpleElements

Description

Determines if the elements with a simple structure such as: <element>value</element> or <element/>, are processed as a unique node of the Element type.

Note:
The default value is TRUE.

Scope

Extended Data Types: XmlReader
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
