DataSource property

Allows you to define the DBMS over which the executed model will operate.

Description

Values

SQL Server
DB2 Universal Database
Oracle
DB2 for iSeries
Informix
PostgresSQL
MySQL

Default value: SQL Server

How to apply changes

To apply changes made by this property, do a Re-Build All.
