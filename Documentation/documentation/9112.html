Oracle version property

Sets the Oracle version range for which the code is generated.

Values

12c or higher The Oracle version installed on the Server should be 12c or higher.
8.1.5 to 8.x.x The Oracle version installed on the Server is between 8.1.5 to 8.X.X.
9 to 11g The Oracle version installed on the Server is between 9.X to 11g.

Scope

Available at the Data Store.
Data Store: ORACLE

Description

This property is used, for example, to determine the syntax for OUTER JOINs and for table creation/reorganization (i.e. syntax of CREATE TABLE, CREATE INDEX, etc.).

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
