Consult Web Object property

Indicates the application that will be displayed when the task is queried in the history.

Scope

Objects: Business Process Diagram
Generators: Angular, Java, .NET, .NET Core
Controls: Task

Description

It applies only when using the GXflow Client for Web Applications and when the task has already been executed.

The application associated with the Query Application Property will be opened when the task is opened from the history.

Availability

This property is available since GeneXus 17 upgrade 7.
