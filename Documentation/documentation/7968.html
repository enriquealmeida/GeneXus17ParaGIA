Blank when empty property

This property allows attributes and variables of date/datetime data type with empty values to be shown with blank spaces (without backslashes or colons).

Values

False
True

Description

Default value: False

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)
