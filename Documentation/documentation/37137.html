Expand Bounds Limit property

Indicates if the control is able to be expanded beyond the application and/or status bars.

Values

Behind System Bars Default value. The container control will be expanded beyond both Application and Status Bars (in conjunction they are known as System Bars).
Behind Application Bars   The container control will be expanded only beyond Application Bar (space for Status Bar is still reserved).

Description 

This property is visible only when Expand Bounds property has not None value. 

For example, when the Application Bar is not visible we have the following result.

Behind Application Bars Behind System Bars

Run-time/Design-time

This property applies only at design-time.

Scope

Controls: Grid control, Tab control, Canvas control, Table control
Platforms: Smart Devices(IOS)

Availability

This property is available as of GeneXus 15 Upgrade 8
