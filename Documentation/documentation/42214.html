Pin Image Field Specifier property

The field containing the image that will be used to display a point on the map in a grid associated with an SDT.

Description

Allows indicating a field of the Structured Data Type associated with the Grid containing the image that will be used to display the points on an Maps Control Type.
This field must be of Image data type. Different values can be assigned to each point of the map, by assigning those values to the field in the Grid control  Load event.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(Android, IOS)
Controls: Grid
