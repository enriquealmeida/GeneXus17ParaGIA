Enable Firebase Crashlytics property

Enables Firebase Crashlytics crash reports.

Values

False
True

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Android, Apple

Description

When set to True, Firebase Crashlitics will be included in the generated application, so that if there is any application crash in production, it will be reported to you via Firebase.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build theÂ Main Object.

Availability

This property is available since GeneXus 17 upgrade 4.
