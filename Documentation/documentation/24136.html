ClientStorage.Remove method

Deletes the stored value for the given key.

Syntax

ClientStorage.Remove(<Key>)

Where:

<key> is the identifier of the value that we want to remove, it has to be Strings

Type returned

None

Availability

This feature is available as of GeneXus X Evolution 3

Scope

Objects: Objects for mobile applications development, Offline Procedures
Platforms:

Android platform, Apple platform
