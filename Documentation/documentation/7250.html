Significant object name length property

Sets the number of significant characters to ensure the uniqueness of the GeneXus Object names. 

Scope

Generators: Cobol, RPG

Description

This property is available in the Properties window of the Knowledge Base Versions.

Values

Number of characters in Object names:
 
Default Value = 128 (the minimum value is 6 characters and the maximum value is 128 characters).

Note:

When using Cobol, Visual FoxPro (dbfs) or RPG Generator, the maximum value is 7.

Remember that there can't be two Objects with the same name, which is why it is recommended that the assigned value is really significant.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
