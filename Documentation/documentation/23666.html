CallOptions

CallOptions are used to specify at runtime the transitions, behavior, and position in a call to a particular Panel object.

Basically, in the passage from one Panel to another there are several elements of user experience that you must consider:

  • CallOptions Transitions: Enter and Exit effect for the new Panel object. 
  • CallOptions Type: Behavior of the new Panel (Popup, Callout, etc.).
  • CallOptions Target: Where the Panel is going to be loaded.
