Do Case command

Performs a set of commands depending on the condition.

Syntax
 

Do case
{Case cond1
        block1}
{Case condN
        blockN}
Otherwise
       blockx
EndCase


Where:
cond1
  Condition evaluated number one.
block1
   Block that is executed if the condition cond1 evaluates to true.
condN
  Condition evaluated number N.
blockN
   Block that is executed if the condition condN evaluates to true.
Otherwise clause
    Establishes a final alternative in case all previous conditions are not matched.
blockx
    Block that is executed if all previous conditions are not matched.
 

Scope

Objects: Procedure, Web Panel, Panel, Transaction
Generators:  .NET, .NET Framework, Java, Angular,  Android, Apple


Description

All the conditions can involve attributes and/or variables.

The first Case whose condition is satisfied will be executed, and none of the others will be executed.

The sentences specified under the Otherwise clause are executed only if all Case conditions evaluate as false.

It works in the same way if/else works.

Note: In Native Mobile applications it works on both Client-side and Server-side events.
