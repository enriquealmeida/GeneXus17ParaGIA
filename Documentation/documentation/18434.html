HowTo: Using Radio Button for Smart Devices

Radio Button is a control applied for Win, Web and SD applications. It is used to show a group of possible values for a variable or attribute. In this tutorial we will explain how to use a Radio Button in Smart Devices. This will use the same transactions and domains that HowTo: Using Combo Box for Smart Devices; the only difference is the use of  the Radio Button control instead of the Combo Box control.

Properties

Auto Grow If this property is true then the field will adjust the length of the attribute.
Values Definition of all the possible values for the attribute/variable, each value is defined as a pair (String:Numeric). The String represents the information that will be displayed to the user when selecting or viewing this attr/var. Each item is defined as a pair separated with a colon (":"), and the different items are separated by a comma. For example, if we want to define the values for three different countries (Brazil, Argentina and Uruguay) we would have: Brazil:1, Argentina:2, Uruguay:3 (the value 0 is used for the empty item, so setting the EmptyItem = True and defining the property Values = Brazil:0, Argentina:1, Uruguay:2, is a wrong definition and will have unexpected results).
RadioDirection Decides whether the values will be shown in a horizontal or vertical way.
ControlTitle Used to define a title to the control (this property is not supported yet).
EmptyItem It enables to set an empty option for this field.
EmptyItemText It defines the information that will be displayed for the EmptyItem when EmptyItem = True.
