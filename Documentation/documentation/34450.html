FromBase64 function

Converts a base64 encoded string into a string.

Syntax

Str( s1 )

Where:
s1
    Is a string encoded in base64

Type returned:
Character

Scope

Objects: Procedure, Transaction, Web Panel, Objects for Native Mobile applications development

Samples

&MyString = FromBase64(&MyBase64EncodedString)

Availability

It is available since GeneXus 15 Upgrade 4
