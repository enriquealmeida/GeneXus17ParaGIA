Permits/Approvals Required for GX Library

In order to create libraries and programs in the iSeries; the user need the following authorizations for the GX library:

  • USE
  • CHANGE

Besides, for the selected Program and Data libraries; the user needs permissions to create, delete and modify it. The following rights are needed:

  • Create a library (command CRTLIB)
  • Create source physical files (command CRTSRCPF)
  • Create Cobol or RPG program (depending on the selected generator)
  • Create a CL program (command CRTCLPGM)
  • Create a command (command CRTCMD)
  • Create a physical file (command CRTPF)
  • Create a Save File (command CRTSAVF)
  • Restore Objects (command RSTOBJ)
  • Change object properties (command CHGOBJOWN)
  • Delete a program (command DLTPGM)
  • Execute command lines (command CHGUSRPRF)
