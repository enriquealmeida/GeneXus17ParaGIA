.NET Generator Extraction Directory property

Indicates the extraction path of a File object when using the .NET Generator.

Scope

Objects: File
Generators: .NET

Description

This property is visible when Extract for .NET Generator property is enabled (True value). By default, it has an empty value taking as root directory the one indicated on Target Path property followed by "/web/bin" subdirectory.

Run-time/Design-time

This property applies only at design-time.
