Services URL Configuration Panel property

Indicates the Panel to display when the Services location URL needs to be changed at runtime.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Android, Apple

Description

This property allows you to select a Panel to customize the assignment of the service URL at runtime.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 Upgrade 8.
