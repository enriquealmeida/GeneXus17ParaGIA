Count property (For variable based on SDT which is collection)

Returns the number of messages to be received, in the case of MAPISession/OutlookSession/POP3Session or the number of elements in the collection, in the case of MailRecipientCollection/StringCollection.

Scope

Level: Variable

Description

This property applies to variables based on an SDT which is collection.
