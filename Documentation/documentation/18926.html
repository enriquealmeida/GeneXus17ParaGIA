GAM Unblock User Timeout property

This is a GAM Repository property which can be configured executing GAM Web Backoffice. It controls how long (in minutes) a blocked user will have to wait until his/her account is automatically unblocked.

Values

Any integer value

Number of minutes the blocked user must wait until his/her account is unblocked.

0 means that the user is never unblocked; the repository administrator must unblock the user.

Example

The GAMRepositoryConfiguration Web Panel (located in GAM Example folder) is an example where this property is used.

The way to use it in GeneXus code (by using the GAM API) is the following:

&Repository.GAMUnblockUserTimeout   = &GAMUnblockUserTimeout
