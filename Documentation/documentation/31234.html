Paging Property for grids

The Paging property applies for Grid control and Free Style Grid control, and allows to determine if the grid will have Paging or Infinite scrolling when Rows property is set to a value different than 0.

Values

Infinite scrolling The grid will load using the Infinite scrolling pattern.
One page at a time The grid will enable Paging.


When Paging property is set to Infinite Scrolling, the Scroll bar property is enabled.
