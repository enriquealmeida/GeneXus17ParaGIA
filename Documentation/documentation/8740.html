Dynamic Combo Box and Dynamic List Box Properties in Web Forms

When the control type of a variable or an attribute is a Dynamic Combo Box or a Dynamic List Box, besides the properties described in the Attribute/Variable Control Properties, in web forms you can assign a value to these properties:

Control Appearance
ItemDescriptions property BackColor property
Sort Descriptions Property Fill property
ItemValues Property Font property
  ForeColor property
  ReadOnly

 

The autoresize property applies only to dynamic list boxes.
