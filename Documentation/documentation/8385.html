GetEncryptionKey function

Generates an encryption key.

Syntax

GetEncryptionKey()

Type Returned:
Character
    It is the key value to be used for the encryption. The function must be run again in order to generate a new key. 

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Description

This function generates a new encryption key with up to 32 characters. It is used to initialize the encryption.

Sample

&GetEncryptionKey=GetEncryptionKey()
