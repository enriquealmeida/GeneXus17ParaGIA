Sharing Licenses with other GeneXus versions

GeneXus licenses, though granted for a specific GeneXus version, can be used in prior GeneXus versions. 

When authorizing a GeneXus version, you are able to use previous versions without the need to authorize or transfer the licenses between them. 

So, when you get a GeneXus 17 license, you can use that with GeneXus 16, or even prior versions, as GeneXus 6.1.

However, in certain versions, you need to install some compatible upgrades, as follows:

  • GeneXus 9.0: Install Development Environment 9.0 Upgrade 4 or higher.
  • GeneXus 8.0: Install Development Environment 8.0 Upgrade 8 or higher.
  • GeneXus 7.5: Install Development Environment 7.5 Upgrade 5 or higher.
  • GeneXus 7.0 and 6.1: Download this file and extract the protrun.dll file to GeneXus installation directory.

 

Note: If you use centralized licenses (installed in a server), you will have to install a GeneXus Protection Server that can hold license keys for the latest version of GeneXus. Supported GeneXus versions are listed here: GeneXus Protection Server
