Refresh period property

Amount of seconds to refresh the dashboard automatically (0 means no refresh).

Description

Specify the amount of seconds to refresh the Dashboard object automatically; the default value is 0 which means no refresh.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16.

Scope

Objects: Dashboard
