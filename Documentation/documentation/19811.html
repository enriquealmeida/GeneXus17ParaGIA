HowTo: Back up and Restore an iOS Application using iTunes

This article explains how to backup an application that is installed in your iOS device and how to restore it.

Back up

  1. Open iTunes
  2. Go to File -> Devices -> Backup
  3. Open the contextual menu of the application you want to backup
  4. Select the "Show in Windows Explorer" or "Show in Finder" option
  5. Back up the file (*.ipa)

Restore

  1. Open iTunes
  2. Go to File -> Devices -> Restore from Backup...
  3. Open Windows explorer or Finder application
  4. Drag & Drop the file (*.ipa) from Windows Explorer or Finder to iTunes
