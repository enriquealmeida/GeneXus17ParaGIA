Offline Database Object properties

The Offline Database object has several properties for configuring how the data synchronization is handled.

Encryption (as of GeneXus 15 Upgrade 5).

  • Encrypt Offline Database property

Receiving (i.e. from server to device).

  • Data Receive Criteria property
  • Minimum Time Between Receives property
  • Data Receive Granularity property

Sending (i.e. from device to server).

  • Send Changes property 
  • Minimum Time Between Sends property
