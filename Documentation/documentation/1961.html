DB2 Universal Database

IBM DB2 Universal Database
GeneXus supports "DB2 Universal Database" for Linux, UNIX, Windows which is known as DB2 LUW; DB2 for z/OS is not supported as it is related to the mainframe.

For information about DB2 you can see the following URLs:
