Show Response As property

Select how the component will be visualized.

Values

Call Panel The component will not be integrated into the chat.
Component The component be integrated into the chat.

Description

The Show Response as property is activated when the Message Action property is set to Component view.

It is available for the Chatbot Message at the Response node of a Flow of a Conversational instance.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows
Platforms: Smart Devices(Android, IOS)
