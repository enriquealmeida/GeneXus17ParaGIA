Address domain

This domain is used to load addresses. In smart Devices and in Edit mode, the information is entered as text using the platform’s input method. In View mode, the information is shown in textual form.

Through a tap gesture allows locating this address using some native mechanism of the platform.

The Address domain type is VarChar(1024).

Scope

Plaforms    Smart Devices, Web
