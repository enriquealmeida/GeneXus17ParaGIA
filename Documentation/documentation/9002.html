Assign Function Keys to Standard Events property

Allows enabling or disabling all assignations to the set function keys.

Values

No Disables the set.
Use Environment property value The object will be generated, depending on the value specified in the corresponding property at Generator level. This value apply only at object level. This is the default value.
Yes Enables the set.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)
