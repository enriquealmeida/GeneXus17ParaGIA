Collaboration mode property

Assigns a task to another user or role.

Values

All roles Any user can be selected as collaborating user, regardless of the roles he may have.
Selected roles Only those users having any of the roles specified in the “Collaboration Roles” preference (only visible when selecting this collaboration mode) are electible.
Task roles Only those users having any of the roles associated to the task are electible as collaborating users.

Scope

Generators: Java, .NET, .NET Core
Controls: Task

Description

This property is active only if the Allow Collaboration Property is set to True.
