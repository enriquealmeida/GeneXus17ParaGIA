Close method

Saves the document and closes it.  

Syntax

&DataType.Close()

Type Returned:
Numeric

Scope

Extended Data Types: ExcelDocument, WordDocument, XmlWriter, XmlReader
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
 

Note: This method returns an error code, so it is possible to call it as a function (&Err = &DataType.Close()).
