Label caption property

The Label caption property is available for all the attributes and variables in the Abstract layout whose Label position property = Left.

It allows establishing the label contents of the variable or attribute. It's a design and runtime property.

Syntax

Control.Caption = Exp

Where: 

Control
    Is the name of the control inserted on the Abstract layout form.

Exp 
    Is the text that will be displayed.

Examples

ArticleName.Caption = &value

Availability

This property is available since GeneXus X Evolution 3 Upgrade 4.
