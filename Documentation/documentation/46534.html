Layer property

Specifies the layer in a multi-layer image. It is used to generate the adaptive icon for the Android Application Icon.

Values

None
Background
Foreground
Middle

Scope

Objects: Image
Generators: Android

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build a main object.
