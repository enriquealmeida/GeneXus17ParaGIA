Automatic enter property

Automatically executes an Enter after the last Attribute of Transaction levels has been entered.

Values

No An Enter has to be typed after the last attribute of the level is entered to continue with the processing (default value)
Use Environment property value
Yes An Enter is assumed after typing the last attribute of a Transaction level. This is valid when no grid is defined for the Level. If a grid is defined, Auto Skip will only be considered when in update mode

Scope

Objects: Transaction
Platforms: Web(.Net, Java)
