Android Application Icon property

This is the image with which the application installed on the device is identified.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Android

Description

Allows defining the icon that the application will have when it is installed on the device.

We recommend following the design lines established by Google - Material Design.

In Android 8.0 (API level 26), adaptive icons are introduced, which will be displayed with different shapes on different devices. For example, it is possible for an icon to be displayed as a circle on some devices and as a square with rounded corners on others. For this purpose, the Layer property is introduced.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
