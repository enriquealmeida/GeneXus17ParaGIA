Predefined Images for Actions and Tabs in iOS

There are some images that the iOS framework provides as standard icons for actions and tabs. Follow this link to find more information about this and the iOS UI guidelines.

Theese icons are used in many iOS native applications and an iOS user knows what this icons means. So if an application needs to represent one of this actions using this icons it is recommended to follow an standard and make the app familiar to any iOS user.

The icons are differenced if they are used in Actions of the Application Bar or On Tabs.
