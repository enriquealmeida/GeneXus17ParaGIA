Attribute property (for Attribute/Variable control)

Indicates the control's name. A default name is always assigned.

Scope

Objects: Transaction, Web Panel
Generators: Java, .NET, .NET Core
Controls: Attribute/Variable

Run-time/Design-time

This property applies only at design-time.
