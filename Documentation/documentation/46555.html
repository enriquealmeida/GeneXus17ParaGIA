Disable Image Resize property

Enables/disables automatic image resizing. It is enabled by default.

Values

False
True

Scope

Generators: Java, .NET, .NET Core
Controls: FileUpload

Description

This property allows you to enable or disable the automatic image resize made by the FileUpload control.
If False (default value), the image will be resized.
If True, the image will keep its original size.Â 

Run-time/Design-time

This property applies both at run-time and at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.
