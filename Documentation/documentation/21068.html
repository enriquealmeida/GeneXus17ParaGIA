Shrink To Fit property

This property allows the user specify if want to use the minimum Column Width (Table and Pivot Table) necessary to display the contents of all their cells.

Values

False The default Column Width is used. This is the default value.
True The minimum Column Width is used.

Scope

Output type   PivotTable, Table
