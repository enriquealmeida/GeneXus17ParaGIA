Required Condition property

Condition to evaluate. If true, the User Input will be required.

Description

This property is available in the Chatbot User Input and allows indicating the condition under which the user input will be required. It's supported only for Watson.

The property is activated when the Required property is set to "Condition" and it should be an expression based on a context variable.

Availability

This property is available since GeneXus 16 upgrade 5.

Scope

Objects: Conversational Flows Instance
