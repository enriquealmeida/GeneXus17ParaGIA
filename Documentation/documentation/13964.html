Autoresize Application Windows Property

It allows establishing whether web forms will be automatically resized according to the contents of its attributes and variables.

Values

Yes   Web forms will be automatically resized according to their content. This is the default value.
No  Web forms will take the values set in the “Window height” and “Window width” preferences.
