Logout method

Ends up the session.

Syntax

&DataType.Logout

Type Returned:
Numeric

Scope

Extended Data Types: MAPISession, POP3Session, SMTPSession
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Note: This method returns an error code, so it is possible to call it as a function (I.E.:  &Err = &DataType.Logout).
