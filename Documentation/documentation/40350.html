Other Buttons Class property

A Button sub-class to apply to other buttons in the full-screen player - such as Rewind and Forward.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, Build a main object.

Scope

Platforms: Smart Devices(Android, IOS)
Controls: AudioController
