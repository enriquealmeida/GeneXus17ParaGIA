Most Representative Unit Only property

Indicates whether only the most significant unit of time should be displayed (truncating).

Values

False
True

Description

When the property value is True, the largest time unit with relevant value is displayed on the screen. 

For example, with the property value Units = sm, seconds will be shown on screen until reaching 59s. After that, the counter will show only minutes without showing seconds. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, Build a main object.

Scope

Platforms: Smart Devices(IOS)
Controls: Attribute/Variable (Control Type: SD Relative Timer)
