Instance Language property

Select the chatbot language

Description

This property allows to set the language of the chatbot when it's defined in the NLP Provider.

In the case of DialogFlow, the languaje can be changed for an existing agent.

For Watson Provider, the language is used only at the creation time of the skill.

Availability

This property is available since GeneXus 16 upgrade 2.

Scope

Objects: Conversational Flows Instance
