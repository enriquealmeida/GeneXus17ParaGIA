Copy method

Creates a Copy of the current Excel worksheet

Syntax

&ExcelWorksheet.Copy(newName)

Type Returned:
Boolean

Where:
newName
    New value to name the copy of the Excel worksheet

Scope

Extended Data Types: ExcelWorksheet data type
Generators:  Java

Description

This method creates a copy of the current Excel worksheet with the name defined in newName
