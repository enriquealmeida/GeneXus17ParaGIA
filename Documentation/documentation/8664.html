Remove method - SDT Collection

To remove one element from the SDT collection.

Syntax

Remove(Position)

Where:
Position
    Relative position in the collection. 

Description

It removes the element in the ‘Position’ relative position of the collection, and moves up all positions to the next location. 

Note: This only applies to SDT collections.

Scope

Objects Procedure object, Transaction object, Web Panel object
Languages .NET, Java, Ruby
