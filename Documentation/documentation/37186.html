Header Row Application Bars Class property

Application Bar theme-class applied for smoothing transition with the theme class set on Class property.

Notes

  • Only displayed when Enable Header Row Pattern property is enabled.

Scope

Controls: Main Table control
Platforms: Smart Devices(Android,IOS)

Availability

This property is available as of GeneXus 15
