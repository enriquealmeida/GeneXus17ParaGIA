Associated table property (in Data Views)

Sets the name of the internal table associated with the Data View.

Scope

Objects: Data View
Generators: Java, .NET, .NET Core

Description

This property allows selecting a Transaction object whose Layout enables end-users to work interactively with the external table associated with the Data View.

You can select it through the Select Object dialog.

Considerations

It is not allowed more than one Data View object with the same associated table. If this occurs, when impacting the database, the following error is given:

rgz0035, 'Table %1 is associated to more than one Data View (%2).'
