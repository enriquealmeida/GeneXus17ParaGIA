Dashboard widget plot series property

Specifies if the chart series are plotted together in the same chart or alone in separate charts.

Values

In separate charts Each series is shown in an independent chart.
In the same chart All the series are shown in the same chart (Default value).

Description

It allows using one or several series on the same Chart.

Run-time/Design-time

This property applies only at design-time.
