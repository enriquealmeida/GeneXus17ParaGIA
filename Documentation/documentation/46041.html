DeleteColumn method

Deletes the column indicated

Syntax

&ExcelSpreadSheet.DeleteColumn(Column)

Where:
Column
     Is the column number that will be deleted

Type Returned: 
Boolean

Scope

Extended Data Types: ExcelSpreadSheet data type
Generators: Java

Description

It deletes the column indicated from the excel sheet. It returns true if the column is deleted successfully
