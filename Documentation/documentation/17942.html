Header property

Header text (caption) displayed above the table or grid.

Description

This property sets the Header of a grid or table.

The string set in this property generates the HTML caption tag of the grid or table.

It is recommended to set this property to improve the accessibility of the applications. More information atÂ Accessibility for Web Applications.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Availability

This property is available since Genexus 15 Upgrade 10.

Scope

Objects: Master Page, Transaction, Web Component, Web Panel
Platforms: Web(.Net, Java)
Controls: Grid, Table
