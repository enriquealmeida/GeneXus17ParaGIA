Application Class for Smart Devices

Allows the developer to set some properties to the whole application. GeneXus takes this theme class automatically when runs the application.

Properties

Background Color Color for the background of all the applications layouts.
Background Image Image  for the background of all the applications layouts.
Placeholder Image Image to show when there is no image asociated, for example a siluete for a person photo.
Image Loading Indicator Allows o not to show an image, meanwhile the real image is loading. Usefull for large images or when some delay is happening for some reason.
Date Picker Image (only for iOs) Image to show in date and datetime attributes instead of the default image prompt used by GeneXus.
Prompt Image Image to show in prompts attributes instead of the default image prompt used by GeneXus.
Launch Screen Animation Class Allows to set an animation to be used on the application Launch Screen
Application Colors Properties group  A group of properties to make general color effects in the application.
