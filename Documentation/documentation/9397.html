DB2UDB version property

Sets the DB2 UDB version range for which the code is generated.

Values

10.5 or higher The DB2 UDB version installed on the Server is 10.5 or higher
7.x The DB2 UDB version installed on the Server is 7.0, 7.1 or 7.2
8.0 or higher The DB2 UDB version installed on the Server is between 8.x and 10.5

Scope

Available at the Data Store.
Data Store: DB2 UDB

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
