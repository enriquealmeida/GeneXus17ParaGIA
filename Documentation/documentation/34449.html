ToBase64 function

Converts a string into a base64 encoded string.

Syntax

ToBase64( s1 )

Where:
s1
    Is a string

Type returned:
Character

Scope

Objects:  Procedure, Transaction, Web Panel, Objects for Native Mobile applications

Sample

&MyBase64EncodedString = ToBase64(&MyString)

Availability

It is available since GeneXus 15 Upgrade 4.
