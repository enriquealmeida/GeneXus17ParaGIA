Newest Version

GeneXus 17 upgrade 1

This article is an overview of GeneXus 17 Upgrade 1 features (compared to GeneXus 17) and what needs to be taken into account to adopt it.

It has been released on December 18th, 2020.

Download

https://www.genexus.com/developers/DownloadCenter?,,,5908

Overview

This upgrade improves overall Developer experience and adds power in a lot of details related to UX & multi experience. It also improves the backend, improving API Object's capabilities, BPM, Java, and .NET.

IDE

  • Improvements to Continuous Integration dialogs
  • Several Stability improvements

Web

  • File Upload control improved
  • User Control Objects

Native Mobile

  • DynamicCall External Object Allows setting the call options to a dynamically called object.
  • New Video Control
  • Apple
    • XCode 12.3 support added
    • New properties to set minimum iOS, iWatch, tvOS version supported in an app (some features of the app may require higher versions than the absolute minimum)
    • Allow third party extension libraries to extend application model from metadata
    • SSL Pinning Pin Set property added
  • Android
    • UI Test results improved
    • Images have now the potential of Advanced Image
    • Imagemap and Grid have now support for Long tab (SAC 48329)

API Mediation

  • Standard variable &RestCode in API Object can be set to return custom HTTP codes

Reporting

  • Query object refactoring for the Axis and Visible properties.
  • Improved the IntelliSense support.

BPM

  • Generate Metadata Domains property for Business Process Diagram object

.NET

  • New ADO.NET Provider supported: MySQL Connector. It is easier to use and has better support for Unicode. More information: ADO.NET drivers for MySQL

Java

  • Performance improvements in environments with high concurrency

Compatibility aspects

  • Query Objects are saved with a new structure to support new features. So GXserver, needs to be updated to 17 Upgrade 1 or higher, too. 
    More information at Axis and Visible property refactoring and Query Object - Troubleshooting.
  • Xcode 12.2 is required. More information: Apple Requirements
