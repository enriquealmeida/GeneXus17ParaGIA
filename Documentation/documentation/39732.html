Height property (for Flex Container)

This attribute determines the height in dips of the control.

Description

The height value is not fixed, depends on the values of Flex Grow property and Flex Shrink property. Even though, the height is not fixed if the cross-axis (row) has an alignment different from 'Stretch'.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
