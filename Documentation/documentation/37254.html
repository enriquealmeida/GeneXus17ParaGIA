REST Protocol property

Indicates whether the object exposed as a service accepts the REST protocol.

Values

False
True

Scope

Objects: Business Component, Data Provider, Procedure
Generators: .NET, .NET Framework, Java

Description

To expose a GeneXus object as a REST Web Service, set the following object properties:

  • Expose as Web Service property = True (after setting this property to True, the REST Protocol property will be shown)
  • REST Protocol = True

Run-time/Design-time

This property applies only at design-time.
