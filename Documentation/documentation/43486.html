Zoom Outside Control property

Allows the image to expand outside the bounds of the control.

Values

False
True

Scope

Objects: Image
Platforms: Smart Devices(IOS)

Description

Allows you to zoom in on images to expand them beyond the size of their containers. This offers application users an enhanced experience when zooming in on an image.

Run-time/Design-time

This property applies only at design-time.

Samples

 

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

Compatibility

Available for Android as of GeneXus 17 Upgrade 2
