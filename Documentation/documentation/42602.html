Web Application property

Indicates the kind of web application to generate

Values

Default Generate a regular web application. This is the default value.
Progressive Generate a Progressive Web Application.

Description

The property is available for main web panels.

Note

There can only be one main panel with the Web Application property set to Progressive per Knowledge Base

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Availability

This property is available since GeneXus 16 upgrade 5.

Scope

Objects: Web Panel
Platforms: Web(.Net, .Net Core, Java)
