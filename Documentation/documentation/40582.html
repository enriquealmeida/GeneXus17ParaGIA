Additional Styles property

Additional UI Design Systems or Themes associated with this platform.

Scope

Generators: Android, Apple
Level: Platforms node > Platform

Description

In the KB Explorer, under the Customization node, you can find the Platforms node. 

Under the Platforms node, several subnodes are offered, each one corresponding to a specific platform.

For each platform, the Additional Styles property is offered to allow adding additional Design System Objects and Theme objects to the specified platform (Android and Apple).

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

Availability

This property is available since GeneXus 17 upgrade 5.
