Open - Knowledge Base

The purpose is to enable opening an existing Knowledge Base. A dialog to select a Knowledge Base folder will be displayed.

Shortcuts

Ctrl+Shift+O and Open Knowledge Base option in the Recent Knowledge Bases dialog container.
