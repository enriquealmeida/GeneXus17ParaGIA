WriteText method

Generates character data with the indicated value string.

Syntax

&DataType.WriteText(Value)

Where:
Value
   String to write

Scope

Extended Data Types: XmlWriter
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual Basic (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Description

Special characters are replaced with character sequences ('<' is replaced with '&lt;', '>' with '&gt;', and so on).
