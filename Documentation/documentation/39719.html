Max Height property

Imposes maximum height constraints in dips.

Scope

Generators: .NET, Android, Apple, Java

Description

A control won't be expanded more than the value of these attributes (varies based on the Flex Direction attribute as to which attribute imposes the size constraint along the main axis) regardless of the Flex Grow attribute.

Run-time/Design-time

This property applies only at design-time.
