Category property

Sets the flow category in order to add it to a group.

Scope

Objects: Conversational Flows Instance

Description

This property is available for the Flows of Conversational Flows objects when using the Conversational Flows Editor.

It allows grouping the flows according to some criteria, defined by the GeneXus user, in relation to the model.

 

Availability

This property is available since GeneXus 17.
