Generate UI property (GeneXus 17)

Enables user interface generation (Panels) for this instance.

Scope

Objects: Conversational Flows Instance

Description

If set to FALSE, Panel objects related to the Conversational Flows object are not generated.

For example, the main object <InstanceName>SDUI (main Panel which calls the "PanelChat" resource object) is not generated. The objects specified in the Generated Component property of the Flows are not generated either.
