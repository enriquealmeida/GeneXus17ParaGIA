Secure Property

This property indicates (or assigns) whether the Http call will be made using HTTPS. 

Syntax

&DataType.Secure

Type Returned:
Numeric

Values

1: Indicates that HTTPS protocol will be used.  
0: Indicates that HTTP protocol will be used.

Description

It specifies whether the call to the remote object will be made by means of a secure protocol or not.
The default value is 0.

Note:
The secure property of the HTTPClient data type, in the C/SQL generator, can only be used in Microsoft clients.

Scope

Extended Data Types: HttpClient, HttpRequest, Location
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
