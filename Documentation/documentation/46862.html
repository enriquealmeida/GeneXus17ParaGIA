Maps - Clear Method

Clears the geographies drawn, one or all.
In order to delete an specific geography is neccessary identified by its id. This is obtained in edit mode, when use Editable Geographies property  or when using the DrawGeography method

Parameters

GeographyId (Optional) Character It allows clearing a specific geography (Line, point, or polygon) given by its id.


Syntax

MapGrid.Clear()

MapGrid.Clear(GeographyId Character)


Sample

MapGrid.Clear()

In order to clear an specific Geography, a geoline for example 

Event 'DrawGeoLine'
    Composite
        &LineId = MapGrid.DrawGeoography(&Geoline, "")
        msg(&LineId)
    EndComposite
Endevent

Event 'RemoveLine'
    MapGrid.Clear(&LineId)
Endevent

Availability

This method is available since GeneXus 17.
