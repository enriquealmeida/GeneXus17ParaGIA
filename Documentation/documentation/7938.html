Print Blocks

Description     

Indicates exactly what is to be printed. You may print: Text, Attributes, Variables, Bitmaps, Lines, etc.

Each Print Block has an assigned name.

Print Blocks are like small Form Editors that may also use controls like Transaction, Prompts and Web Panels. It is also possible to perform multiple operations with controls defined within the same Procedure but corresponding to different Print Blocks. For example, align attribute fields in one print block with their text titles in another print block.

Inserting a Print Block

To insert a Print Block below another, you must click right button and select Insert Printblock contextual menu option.

The Delete key deletes the selected Print Block, but it can only be deleted when block is empty.
