JWT Domains

Note: This is part of the GeneXus JWT Module and determine the available algorithms..

JWTAlgorithm

It defines the asymmetric or symmetric algorithm to be used.

Available values:

HS256, HS512, RS256, RS512, ES256, ES384, ES512
  • The values ES256, ES384, and ES512 are supported since GeneXus 17 upgrade 1.

RegisteredClaim

It defines valid registered claim keys.

Available values:

iss, exp, sub, aud, nbf, iat, jti
