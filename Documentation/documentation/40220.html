Generate Update Flow property

If it is true, GeneXus will generate the Update Flow.

Scope

Objects: Conversational Flows Pattern

Description

The Generate Update Flow property is available for the Transaction node under the Conversational Configuration (Pattern Settings).

Availability

This property is available since GeneXus 15 upgrade 12.
