IsPassword property

Displays asterisks (*) or black circles, depending the browser or platform, instead of what the user digitized. This property is not available when the attribute is a formula and applies only at execution time.

Syntax

control. IsPassword

Type returned: Boolean

Values

False
True

Scope

Objects: Panel for Smart Devices, Transaction, Web Panel, Menu for Smart Devices, Work With for Smart Devices
Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
