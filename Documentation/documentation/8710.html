BackStyle property

Determines whether a label is transparent or opaque. It is equivalent to the Fill design property.

Syntax

control. BackStyle = value

Where:
control
Is the name of a control inserted in the form

Values

0 Transparent - background color and graphics are visible behind the text.
1 Opaque - the textâ€™s BackColor fills the control and obscures any color or graphics behind it.

Run-time/Design-time

This property applies only at run-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
Controls: Attribute/Variable
