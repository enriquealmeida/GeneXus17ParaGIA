Collection property (IsCollection checkbox)

Indicates whether the element has multiple instances or not.

Values

False
True

Scope

Generators: Android, Angular, Apple, Java, .NET, .NET Core
Level: Domain, Variable, SDT level node, SDT member

Description

This property is available for:

1) SDT level nodes and SDT members:

2) Variables:

In both cases, you can achieve the same effect by selecting/clearing the IsCollection checkbox as well as by setting the Collection property.

3) Domains


Default value: False.


Note

If the variable corresponds to a Chatbot User Input, see Variable collection property behavior for User inputs.

Run-time/Design-time

This property applies only at design-time.
