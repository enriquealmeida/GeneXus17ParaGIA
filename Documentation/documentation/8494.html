Systime function

Returns the system time in 'hh:mm:ss' format. In other words, returns the same as what the Now Function returns, but in string format.

Syntax

Systime()

Type Returned:
Character(8)

Scope

Objects:  Procedure, Transaction, Web Panel

Note: In the iSeries environment, this function returns the System Time and not the Start Time of the Job. See the Time function if you need to use the Job's Start Time. In a Client/Server environment, this function returns the time of the application server.
