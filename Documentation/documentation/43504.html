AdMob Application ID property

AdMob Application Identifier. Can be found at https://apps.admob.com by registering your application.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)

Description

This property appears to be configured if you set the Ads Provider property = GoogleMobileAds. 

It is required as of GeneXus 16 upgrade 5.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
