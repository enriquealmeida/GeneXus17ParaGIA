SetAll method

Sets a new style to the cell borders

Syntax

&ExcelCellBorder.SetAll(borderStyle)

Where:
borderStyle
     Is an ExcelBorderStyle data type. It defines the new color and type of the borders.

Type Returned:
Boolean

Scope

Extended Data Types: ExcelDocument data type
Generators:  Java

Description

It sets borderStyle as the new style of all the borders of the cell.
