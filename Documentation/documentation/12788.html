TableGridContainer Property

It allows setting a theme class to a table that it contains the grid in the WW<Object>.

You will find this property in the Theme icon of the WW Configuration node in the Work With icon of the Preferences window.

Values

Any of the default table classes defined in the property.

Default = Table

Scope

Objects: Work With Pattern
Languages: .Net, Java, Ruby (up to GeneXus X Evolution 3)
Interfaces: Web
