Custom Properties property

Specifies additional properties for a class in the GeneXus Theme.

Syntax

control. Custom Properties

String values with CSS properties separated by ";"

Scope

Objects: Theme
Generators: Angular

Description

Properties specified here will be included in the Class definition, in the CSS file generated for the Theme.

Additional CSS properties are added to the Class or HTML node where this property is specified.

Samples

background-color: white; background-image: url('imagens/fundo'); z-index: -1
