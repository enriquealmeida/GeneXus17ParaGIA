Control Properties - Font

It allows choosing a font for the control in a Form.

Property Description
Name Allows selecting the desired Font.
Size Allows choosing the desired Font Size.
Unit Unit of measure for the Font.
Bold Specifies whether the Font is bold or not.
Underline Check it to underline the text.
GdiCharSet Obtains the number of bytes that specify the GDI character set used by this font.
GdiVerticalFont Boolean value that indicates if this font is derived from a GDI vertical font.
Italic Allows choosing the desired style.
Strikeout Allows inserting a horizontal line over selected text.
Underline Allow inserting a line under the selected text.
