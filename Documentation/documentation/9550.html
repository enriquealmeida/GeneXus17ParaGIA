Control Type property

Indicates which kind of control you want to show. Possible values depend on whether the control is an Attribute/Variable, a Grid, an Action Group, or another, as well as the platform, etc.

Values

Combo Box
Radio Button
Edit
Image
Check Box
Dynamic Combo Box
List Box
Dynamic List Box

Description

For Attribute/Variable controls, the possible types are:

  • Check Box
  • Combo Box
  • Dynamic Combo Box
  • Dynamic List Box Setup
  • Edit
  • List Box
  • Radio Button
  • FCK Html Editor
  • GXSpreadsheet
  • SD Scanner (*)
  • Rating Control (only for variables or domains)
  • SD Linear Gague (*)
  • SD Multi Wheel (*)
  • SD Physical Measures (*)
  • SD Advanced Image (*)
  • SD Wheel (*)
  • SD Facebook Button (*)
  • SearchBox (*)
  • Chronometer
  • Switch
  • Touch Controls - Carousel (*)
  • Web Linear Gauge

Note: This property is not available when the attribute is a formula.

For Grid controls, the possible types are:

  • SD Charts (*)
  • Horizontal Grid ControlÂ 
  • SD Image Gallery (*)
  • SD Image Map (*)
  • SD Leaves (*)
  • SD Magazine Viewer (*)
  • SD Paged Grid (*)
  • SD Maps (*)
  • SD Spark Line (*)
  • SD Geolocation (*)
  • SD Matrix Grid (*)
  • SD Flex Grid (*)
  • SD Smart Grid (*)

(*) Only in Native Mobile Generator

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Transaction, Web Panel
Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
Controls: Grid, Attribute/Variable
