Scroll Indicators Visibility property

Specifies the behavior for scroll indicators visibility

Values

Both The indicator is visible while tracking is underway in any direction (horizontal or vertical) and fades out after tracking
Platform Default Each platform makes what's best for its context (default)
Horizontal Only the horizontal scroll is visible while tracking is underway
None Scroll indicators are always hidden
Vertical Only the vertical scroll is visible while tracking is underway

How to apply changes

To apply changes made by this property, Build a main object.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Theme for Smart Devices
Platforms: Smart Devices(IOS)
