App Id property

Application ID provided by WeChat to use with WeChat Pay and WeChat Login

Description

Associated to the AppID property provided by WeChat to use with:

  • WeChat Pay API
  • GAM WeChat Authentication type

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Menu for Smart Devices
Platforms: Smart Devices(Android, IOS)
