GeneXus IDE/Shortcuts

Here a list of shortcuts you can use, if you don't want to use the mouse: 

General
 
Ctrl+C
Copy
Ctrl+V
Paste
Ctrl+X
Cut
Ctrl+Y
Redo
Ctrl+Z
Undo
Delete
DEL
F4
Opens Properties dialog box
F5
Run startup program
Ctrl+Tab Shows preview of selected object
File Menu
 
Ctrl+Shift+N
New Knowledge Base
Ctrl+N
New Object
Ctrl+Shift+O
Open Knowledge Base
Ctrl+O
Open Object
Ctrl+F4
Closes an object
Ctrl+S
Save
Ctrl+Shift+S
Save all changed objects 
Ctrl+P Print selected objects
Edit Menu
 
Ctrl+Z
Undo
Ctrl+Y Redo
Ctrl+X Cut
Ctrl+C Copy
Ctrl+V Paste
Delete DEL
Ctrl+A
Select All
Ctrl+F
Find
Ctrl+H
Replace
Ctrl+G
Go To Line
Ctrl+Shift+F4
Reopens the last closed object (undo of Ctrl+F4)
Ctrl+Q
Comment selection
Ctrl+Shift+Q
Uncomment selection
Ctrl+B
Toggle Bookmark
Ctrl+Shift+B
Enable Bookmark
Shift+F2
Previous Bookmark
F2
Next Bookmark
Ctrl+K
Toggle Key
Ctrl+D
Toggle Description Attribute
Ctrl+L
Insert Level
Ctrl+Shift+L
Insert Sibling Level
Ctrl+Shift+U
Make Uppercase
Ctrl+Shift+L
Make Lowercase
View
 
Ctrl+Shift+I
Knowledge Base Information
Ctrl+F12 References
Ctrl+Shift+H History
F4 Select Object Properties
Ctrl+J Work With Objects
Ctrl+Shift+F11 On/Off Full Screen View
Insert  
Ctrl+Shift+J Insert objects
Ctrl+Shift+A Insert attributes
Ctrl+Shift+D Insert domains
Ctrl+Shift+V Insert variables
Ctrl+Shift+F Insert system functions
Ctrl+Shift+E Insert control events
Ctrl+Shift+R Insert rules
Ctrl+Shift+K Insert commands
Build
 
  
F8
Build Developer Menu
Ctrl+Shift+F8 Rebuild Developer Menu 
F5 Run Developer Menu
Other
 
Ctrl+F12
Opens the references dialog
Ctrl+MouseWheel
Zoom in/out (in edition)
Ctrl+Shift+A
Opens the attributes dialog
Ctrl+Shift+D
Opens the domains dialog
Ctrl+Shift+V
Opens the variables dialog
Ctrl+ Mouseover Shows Tooltip when Mouse is over an object in text editor
Ctrl+C / Ctrl+V in model tree
Save As of the object
Ctrl+PgUp / Ctrl+PgDn Sets as active the next/previous Object Part
Ctrl+F5 Runs a Business Process Diagram
Ctrl+Shift+C Copy bitmap to clipboard - Business Process Diagram
Ctrl+Shift+G Save As Image - Business Process Diagram
Ctrl+Space Opens the valid objects window for insert in the context
Layout  
Ctrl+L  Align Left 
Ctrl+E Align Center
Ctrl+R Align Right
Ctrl+T Align Top
Ctrl+M Align Middle
Ctrl+B Align Bottom
Debug  
F9 Toggle Breakpoint
F10 Step Over
F11 Step Into
Shift+F11 Step Out
Source Code Editor  
Shift+Left (or Right) Arrow Select horizontal text
Shift + Left (or Right) Arrow + Alt + Down (or Up) Arrow Select vertically (like selecting columns)
Shift+Left (orRigh) Arrow + Down (or Up) Arrow Select text horizontally and vertically
Object Comparer  
Ctrl+Left (or Right) Arrow Next or Previous difference
Ctrl+ Shift + Left (or Right) Arrow Last or First difference
