Botmaker Token property

Botmaker Authorization Token.

Scope

Objects: Conversational Flows Instance

Description

To integrate with WhatsApp using the Chatbots Channels API, set the Enable WhatsApp property to True. After activating this property, the WhatsApp Partner property becomes available. There, select the partner of your choice (Botmaker) and, in the Botmaker Token property, configure the Auth Token given by Botmaker.

Read the document Chatbots using WhatsApp: How to set up Botmaker for information on how to get that Auth Token.

Availability

This property is available since GeneXus 16 Upgrade 11.
