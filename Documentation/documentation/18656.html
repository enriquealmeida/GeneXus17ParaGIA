Generate iOS property

Enables the generation of code for Apple's iOS platform.

Values

False
True

Scope

Generators: Apple
Level: Generator

Description

Default value: False

In addition, the True value will add the iOS Specific group to the list of properties and you will be able to set the necessary properties for prototyping in this platform.
