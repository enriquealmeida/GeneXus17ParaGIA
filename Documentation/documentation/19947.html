GAM options in GeneXus toolbar

The purpose of these options is to allow to manage the actions which execute on the GAM database, actions that configure the database using the information of the KB.

  • GAM - Create tables option
  • GAM - Initialize Metadata option
  • GAM - Applications Registration option
  • GAM - Reprocess Permissions Option

The following item is to manage the GAM Examples in the KB.

  • Installation Settings. See GAM Activation Process for details.

The following items are available for deployment purposes.

  • GAM - Update Connection File
  • Create Deploy File
