XSLTApply function

Applies XSLT to XML in order to obtain a formatted file. 

Syntax

Xsltapply(&XML,&XsltFileName)

XSLTApply(string-filepath | variable, xsltfile-name)

Where:
xsltfile-name
   Is the xslt file that will applied against the xml file to transform it.

string-filepath | variable
   Contains the path of file-name.

Scope

Generators: Java, .NET, Ruby (up to GeneXus X Evolution 3)

Description

The XSLTApply function receives the XSLT file path, and applies it to a string variable or XML file variable. 

Samples

&XmlFile = "C:\Temp\cdcatalog.xml"
&StringResult = XSLTApply(&XmlFile,"C:\Temp\cdcatalog.xsl")
Note: In Windows Java, it only works with the Sun VM.
