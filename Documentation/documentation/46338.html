Fill Color property

Fill color to use for drawing polygons on a Grid control of Maps type.

Scope

Controls: Grid (Control Type: Maps)

Description

You have two options to apply the changes after setting the property:

1) Build a Main Object that involves the property.
2) Execute a Build All.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build a main object.
