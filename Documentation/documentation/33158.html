Include Max and Min Property

Include Max and Min property it's a QueryViewer control and a Query object property. It's only available when the Type property in QueryViewer control has the Card value assigned and allows displaying (under the Card indicator) the highest and lowest value taken by the indicator.

Values

False Maximum and minimum values are not shown. This is the default value.
True Maximum and minimum values are displayed under the Card's indicator.

Description

If the Query object includes a Date or DateTime attribute, the highest and lowest values that the indicator took can be calculated and shown by setting the Include Max and Min property in True:
