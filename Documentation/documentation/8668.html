AddItem method

Adds a new item at the end of an object.

Syntax

control.additem(code, description [ , index ] )

Where:
index
    It is optional, and represents the position where the item is inserted. Its value must be greater than 0. If no Index is specified, then the item is added at the end. It is not supported by Form.Meta or Form.MetaEquiv.

Scope

Controls: Combo Box, Dynamic Combo Box, Dynamic List Box, List Box
Generators:  .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Description

This method is used to add a new item at the end of a Transaction or a Web Panel.

Samples

In this example, a Combo Box with all countries in the American continent is loaded. The first item will be 'All'. The AddItem method is used to load each item to the combo.

&Var.clear()
&Var.additem(0, 'All')
For each
    where ContinentCode = 'AME'
         &Var.additem(CountryCode, CountryName)
EndFor

In this example, a MetaTag to the Web Panel is added.

Form.Meta.additem(‘Content’, ‘GeneXus’)

Notes:

  • Form.Meta and Form.MetaEquiv only apply to Web Interface.
  • Form.Meta and Form.MetaEquiv are Objects' properties.
