GAM - Initialize Metadata option

This is a menu option : Tools -> GAM -> Initialize Metadata.

Executes the initialization process of GeneXus Access Manager metadata: defines a GAM repository, GAM version, the administrator and connection user.

Note: the GAM database metadata is initialized automatically in build process (F5); this menu option could be neccessary in cases when a previous initializations fails.
