GXtest versions

On this page, you will find all information about the different GXtest versions. 

GXtest is published continuously over Beta and Preview GeneXus channels.

Beta Channel

GXtest Beta version is daily integrated into the GeneXus Beta channel, and the main purpose is to release new features to gather feedback within the community. This means that if you are using the latest GeneXus Beta version you are already up to date. If not, you can always download the latest GXtest Beta version from the links below:

GXtest for GeneXus Beta

GXtest Extension for GXServer Beta

Preview Channel

GXtest Preview channel is also daily integrated into GeneXus Preview Channel and will become the official Upgrade on each GeneXus release. This means that if you already have the latest upgrade of GeneXus, your GXtest version is also up to date. In any case, you can always install manually the latest version of GXtest by downloading the GXtest Preview version from the links below:

GXtest for GeneXus Preview

GXtest Extension for GXServer Preview

Older GXtest Versions

Besides, GXtest is released for the different versions of GeneXus (since GeneXus 15 U7) and it is necessary to install a compatible version. There are GXtest versions for older GeneXus versions, you can download and install them manually.
