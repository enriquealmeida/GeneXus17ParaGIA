entity property

Entity to match the input of the user when Match with Entity property is set.

Description

It's a User Input property.

When match With Entity property is set. you have to specify the name of the entity in the provider that should match with the user input.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows
