Chatbots Collection property

Indicates whether the user's input has multiple entity values.

Scope

Objects: Conversational Flows Instance

Description

The Collection property is available for Chatbot User Inputs when match With Entity property is set to TRUE and an entity is specified in the entity property.

In that case, if Collection is set to TRUE, it means that more than one entity value can be recognized from the user's input.

Temporary Limitation

For Web UI components (Web Component property or Conversational Object which are Web components), it's only supported in NET applications.

Compatibility

As since GeneXus 16 upgrade 10, this property does not exist at the user input and is part of the variable definition.

Availability

This property is available since GeneXus 16 upgrade 3.
