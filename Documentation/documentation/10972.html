Automatic Assignment Conditions Properties Group

This is a group of properties which allows you to define conditions through which tasks can be automatically assigned. 

They can be found in a task's properties (in a Business process diagram), under the node Advanced properties>Automatic Assignment Conditions:

Reassign when Loop
Assign to process owner
Load balancing
Assign to previous user
Assign from a user in a relevant data
Assign to the same user who executes a previous task
Assign to default user
