User Name property

Specify the Watson Assistant user name.

Scope

Objects: Conversational Flows

Description

In the Conversational Flows object the NLP Provider property can be set to "Watson" value. In that case, you have to indicate the User Name of the Watson account.

For more information about the configuration to be done in order to use Watson provider, see Configuring GeneXus for using the Chatbot Generator.

Availability

This property is available since GeneXus 15 upgrade 12.
