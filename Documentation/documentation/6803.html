Automatic refresh property

Refreshes the Grid automatically without the user having to click a "Refresh" button.

Values

No The web panel does not refresh automatically.
Yes A refresh is performed automatically every time any variable in the web form which is used in the Refresh, Load event or conditions, changes. This is the default value.

Scope

Objects: Web Panel
Platforms: Web(.Net, Java)
