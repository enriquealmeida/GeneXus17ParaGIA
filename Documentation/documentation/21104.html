Server URL property

This property is defined when the Send Knowledge Base to GeneXus Server or the Knowledge Base from GXserver (X Evolution 2) actions are performed. It defines the URL to which the Knowledge Base is connected.

Values

A valid URL to aÂ GeneXus Server instance.

When using a local GXserver installation on your network confirm the installation virtual directory, a default value is constructed as follows:

http://MachineName/VirtualFolder/

where:

MachineName: details the machine name or IP address of the server.
VirtualFolder: application folder selected during the GXserver installation process, some examples are GeneXusServer, GeneXusServerXev2, GeneXusServerXev3.

Examples

http://sandbox.genexusserver.com/xev2
http://samplemachine/gxserverxev3u1
http://localhost/GeneXusServerXev3
