ContentInfo Data Type

To store information for the document to be indexed.

Syntax

&DataType.Property = value

Where Property:
    Id, Title, Type or Viewer

Properties

Id  Document key used to retrieve the object from the original source
Title  Content title
Type  Document type identifier
Viewer  URL used to display a retrieval result

Methods

None

Scope

Objects: Procedures(Web Only), Transactions, Web Panels
Languages: .NET, Java, Ruby
