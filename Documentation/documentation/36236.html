SelectionChanged event

Event triggered when the end user selects a new item in a Grid control, both programmatically (by using Select/Deselect methods) or interactively (by tapping on a row). In this last case, the event will be executed before the default action.

Syntax

Event Grid.SelectionChanged
     <event code>
EndEvent

where <event code> are coding lines executed when the event is triggered.

Notes

  • SelectionChanged event will not be triggered when Selection Type property has 'No selection' value.

Limitations

  • Cannot access attributes values. Use default action event for achieving that aim.

Scope

Controls Grid control
Platforms Smart Devices (iOS,Android)

Availability

This Grid control event is available as of  GeneXus 15 Upgrade 6 for iOS and GeneXus 15 Upgrade 7 for Android.
