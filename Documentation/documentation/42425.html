Google Api Key property (for GoogleMapControl)

Sets a Google API Key to display the Google Map control at runtime.

Scope

Generators: Java, .NET, .NET Core
Controls: GoogleMapControl
Level: Environment

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build a main object.
