GxCopyFile Function

Invokes a dialogue to make a copy of a file.

Syntax

Call(‘GxCopyFile’, Source, Target[, Ret-value]) | [Ret-value =] GxCopyFile(Source, Target)

Type Returned:
Numeric

Where:
Source 
   Character type. Is the input file name.

Target
   Character type. Is the ouput file name.

Ret-value
   Numeric type. Is the error code returned by the function.

Description

In order to use the second syntax, the object property “Standard Functions” must be set with the value “Allow non-standard functions”.

Examples

Event 'CopyFile'
     Call('GxCopyFile', &source, &target)
EndEvent 
Event 'CopyFile'
     &ret = GxCopyFile(&source, &target)
EndEvent

Scope

Objects: Procedures, Transactions, Work Panels
Languages: .NET, Java, Visual FoxPro
Interfaces: Win
