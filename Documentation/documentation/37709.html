Description Class property

Indicates which theme class will be used to customize a Progress Indicator description.

Description

When aÂ Progress Indicator User Control control is used, the description (secondary text) format can be customized setting a Text Block-type theme class name in this property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 9.

Scope

Objects: Theme for Smart Devices
Platforms: Smart Devices(Android, IOS)
