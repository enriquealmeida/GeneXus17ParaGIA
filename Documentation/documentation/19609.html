Export Properties group

This group of Properties is made up by the export formats.

The export formats supported are XML, HTML, XLSX, XLS and PDF. The developer can select those that will be available for use by the end user and those that will not.

At runtime, right-clicking anywhere on the Pivot Table or Table area displays a context menu with all the export options which are set to True.

ExportHeader and ExportHeaderHeight properties

The ExportHeader property allows indicating a text that will be included in the export and will be displayed as report header.

Since images of various sizes can be included in the header, the ExportHeaderHeight property allows setting its height manually. This is not the height value for the text included in ExportHeader, but the height of the space reserved to include texts and images.

Example

Event Start
    QueryClients.ExportHeader = "<img src='http://files.sac.com/logos/SouthAmericanLogo.JPG' width='1000' height='150'/>"
    QueryClients.ExportHeaderHeight = 150
EndEvent

The same Html string may be introduced in design time.

Also, if you type "South American Investment Co." on the ExportHeader property, the text will be shown as the report header.
