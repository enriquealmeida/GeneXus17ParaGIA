Empty Grid Background Class property

Indicates the Theme Class to be used when showing the Empty Grid Background in the Grid of a Front End layout.

Scope

Generators: Android, Angular, Apple
Controls: Grid

Samples

In the following example you can see the "Image" class selected in the Empty Grid Background Class property.

It will show the "ImageToShow" image as a background of an empty Grid.

If you want to change some properties of the "Image" class, go to the class (Customization/Themes in the KB Explorer) and change them.
