Input View Appearance property

Sets the appearance style of the input view (i.e. virtual keyboard).

Values

Dark Force dark input view.
Default Default value. Use Light or Dark depending on platform default.
Light Force light input view.

Description

This property is available in Form theme-class for Smart Devices object and allows the developer to change virtual keyboard displayed when the end-user inputs text on an editable field. It's desirable that a Panel which allows input data on an editable filed (such as text, numbers, etc) displays the keyboard preserving the look and feel.

The following example, based on My first application, shows both appearances of the keyboard when we try to insert a new customer.

Light Dark
   

 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 9.

Scope

Objects: Theme for Smart Devices
Platforms: Smart Devices(IOS)
