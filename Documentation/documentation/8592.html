Eject Command

Forces a Form Feed or a Clear Screen action.

Syntax

Eject

Description

This command forces the generated program to skip to the next page when the selected output is set to the printer or to clear the screen when the output is set to the screen.

Scope

Objects: Procedures
Languages: .NET, Ruby, Java, RPG, Visual Basic, Visual Fox Pro, Cobol
