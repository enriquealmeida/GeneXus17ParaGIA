OneSignal - REST API Key property

API Key used to use the OneSignal REST API internally by the OneSignal provider.

Description

This property is available at Java and .NET generator level when Notifications Provider property has OneSignal value.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 4.
