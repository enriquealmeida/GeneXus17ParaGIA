IncludeCookies Property

Indicates if cookies generated by previous requests will be shared in the HTTP request so that multiple requests made to stateful services in the same domain share the same web session. 

Syntax

&DataType.IncludeCookies

Type Returned:
Boolean

Default Value: True

Description

In every HTTPClient request, you can indicate if the cookies obtained from the previous request made to the same domain will be sent. The previous request does not need to have include cookies.

Scope

Extended Data Types: HttpClient
Languages: Web(.NET, Java, NET Core)

Availability

This property is available from GeneXus 17

For Android and Apple it is available as of GeneXus 17 upgrade 5.
