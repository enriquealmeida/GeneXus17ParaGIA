MailRecipientCollection Data Type

This Data Type is used internally and indicates a collection of MailRecipient type objects. 

Unifies the interaction functions with the sending and reception of messages for the different generated languages. 

An important advantage of this implementation over previous ones is that this allows you to use more than one connection event to the mail server.
