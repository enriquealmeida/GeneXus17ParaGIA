Data Type property

To indicate the attribute, variable, relevant data or domain data type.

Audio

Boolean
Character
Date
DateTime
GUID
Image
Long Varchar
Numeric*
VarChar
Video

* Default value

Notes

  • The Blob data type is not valid for variables.
  • Variables and domains are classified into Basic Data Types, Extended Types, External Objects and Structured Data Types.
  • Attributes only support basic data types.
  • Relevant data only support Character, Date, DateTime and Numeric.

Scope

Objects: Structured Data Type
Controls: Attribute/Variable
