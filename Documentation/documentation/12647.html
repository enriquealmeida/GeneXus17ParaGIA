Month method

Returns the numeric month identifier of a given date. A null value (0) is returned if the received parameter is null.

Syntax

Date-Expression | Datetime-Expression.Month()

Type returned:
Numeric N(2)

Scope

Objects:  Procedure, Transaction, Web Panel

Samples

If &Today.Month() = 12
   &TxtMonth = "December"
EndIf
