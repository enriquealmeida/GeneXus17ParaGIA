IDE Configuration Options

Certain parts of the IDE can be configured to provide developers with a more comfortable experience using it. This can be achieved by specifying the users' preferences. For instance, developers can determine the location of the Knowledge Bases.

The IDE can be configured from the Options window dialog (by selecting in the main GeneXus Menu: Tools > Options). Configurable options include Editor configuration and Search engine management, among others.

The following options are available for configuration:

  • Wiki
  • Themes
  • Build
  • Custom Build
  • Patterns
  • Comparer
  • Search
  • Knowledge Base
  • Editors
  • Options - KB Explorer

    These functionalities are frequently enhanced, so be sure to check back regularly for updates. 
    The list varies by GeneXus version, so not all of the above listed may be available to your installation.
