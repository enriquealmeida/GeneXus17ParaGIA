HowTo: Changing Your Inbox Theme

Steps to follow to change your Inbox Theme to grey:

  1. Download and unzip the file from here.
  2. If you are working with .NET environment must copy to the web directory of the model. (C:\Models\<KB>\<directorio del modelo>\Web).
  3. If you are working with Java environment must copy to the static directory of the web application. (C:\Program Files\Apache Software Foundation\Tomcat 5.5\webapps\<web application>\static).

Note: You must backup the folder with the blue theme.
