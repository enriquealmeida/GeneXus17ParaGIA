Objects for Native Mobile applications development

The methodology used by GeneXus to create mobile applications involves the following objects:

  • Menu object
  • Theme object
  • Work With for Smart Devices object
  • Panel object
  • UI Test object
  • Offline Database object
  • Data Provider object
  • Procedure object

Notes

  • The above objects are not exclusively for Native Mobile applications.
  • There are also other concepts used when developing Native Mobile applications like Business Components and more.
