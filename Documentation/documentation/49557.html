Deploy to Cloud Troubleshooting

This article guides you through the steps you should follow if some trouble occurs using the Cloud prototyping feature.

  1. Check that you followed these steps: Deploy to cloud: Step by Step
  2. Apply the solutions provided in Deploy to GeneXus Prototyping Cloud: under the hood

If you're still in trouble:
