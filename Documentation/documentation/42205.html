Results Panel property

It must be set with a Panel (specially designed) by the developer in order to display the search result. In such panel is where the search box will be displayed on the application bar.

Description

Results Panel is a property of the SearchBox control (a facility for the developers to improve the search mechanism in their applications).

It's available for every string variable whose Control Type property is set to SearchBox value.

For more information see HowTo: In-app search in Smart Devices.

Scope

Platforms: Smart Devices(Android, IOS)
Controls: Attribute/Variable
