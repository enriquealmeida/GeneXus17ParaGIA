Run Target property (GeneXus 17 upgrade 7)

Sets the target server that will be used to serve an Angular application.

Values

AWS S3 Prepares the Angular Project to be manually uploaded to AWS S3 via command line.
GeneXus DeployToCloud Indicates that the Application will be uploaded to the Cloud of DeployToCloud. Applications deployed with DeployToCloud run under AWS S3 and are accessed through the Cloudfront CDN.
Default By default it runs on the Angular Http Server of the Angular Platform. It is a Server that runs local to the machine on a random Port.
Angular Dev Server It runs on the Angular Http Server of the Angular Platform. It is a Server that runs local to the machine on a random Port.
Local WebServer The application is transpiled and copied to the local Web Server of the GeneXus Environment (Java, Net). It is executed directly in the URL of the Web back end. It is housed in an "ng" subfolder.

Scope

Generators: Angular
Level: Generator

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 upgrade 7.
