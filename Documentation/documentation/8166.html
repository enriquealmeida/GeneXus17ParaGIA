Delete event

To delete the current record. It occurs upon pressing the Delete button or associated control.

This standard event is used in Transactions to delete the current record. It occurs upon pressing the Delete button or associated control.

Win interface

The associated controls may only be buttons.

Web interface

The associated controls may be buttons, images, text blocks and read-only edit boxes.

Note: The GeneXus developer cannot associate any code to this event.

Scope

Objects Transaction object
