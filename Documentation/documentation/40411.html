Filter control orientation property

Specifies whether to arrange the filter values horizontally or vertically.

Values

Horizontal Values are displayed side by side (this is the default value)
Vertical Values are displayed one above the other.

Description

When displaying the filter as a Radio button or a Check box, this property specifies whether to arrange the filter values horizontally or vertically.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Dashboard
