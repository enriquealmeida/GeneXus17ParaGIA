Diagrams

Allow you to view the relationships between the Transactions or Tables objects.

Description

One of the most outstanding characteristics of GeneXus is the ability to recognize the relations existing between the objects it manages. The reports provided by it are very useful to those who develop applications; however, this option will allow you to see the relations existing between the objects via diagrams. Undoubtedly, these diagrams will ease the work for all persons involved with the model (developers, users, managers, consultants, auditors).

The model can have as many diagrams as it needs. The way each diagram is set depends on the user who will only indicate which objects must be included. So, the information used by large applications will be a lot easier to manage.

You can choose between table diagrams or transaction diagrams.
