GxCalculate Function

To invoke the windows calculator.

Syntax

Call(‘GxCalculate’, Result)

Type Returned:
Numeric

Where:
Result
   Numeric type. Input/output numeric value. It is the result of the calculation performed.

Example

1. From the Calculate button located on the Win Form, or...

2. Right-click on the numeric field and select Calculator option.

Scope

Objects: Procedures, Transactions, Work Panels
Languages: .NET, Java, Visual FoxPro
Interfaces: Win
