Trim method

Returns the specified character expression with all leading and trailing blanks removed.

Syntax

character-expression.Trim()

Where:
char-expression
     Is the character expression from which you want to trim all leading and trailing blanks.

Type Returned:
Character

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Description

Returns a character expression with all leading and trailing blanks removed.

Samples

&Var = &Text.Trim() 
// &Text = '   My character expression   '
// Result of &Var: “My character expression”
