Detail property

Indicates the JDBC log’s detail level.

Values

High It’s the maximum exposure level.
Medium It’s a shortened version of the High option (it doesn't show the result of the cursor execution, among other data).
Low It’s a shortened version of the Medium option (it doesn't show the SQL sentences executed, among other data).

Scope

Platforms: Web(Java)
