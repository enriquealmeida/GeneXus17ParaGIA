Required property

Indicates if User Input is required. Always -> The parameter is required if its value is empty. Never -> The parameter is not required. Condition -> The parameter is required if the 'Required condition' is true.

Description

This property is in the Chatbot User Input and allows establishing if the user input will be required to be entered or not.

Its possible values are:

Always     -> Means that the user input is required if its value is empty.
If the Clean Context Value property is FALSE, and the value is in the context, the user won't be asked to enter another value.

Never       -> The user input is not required.

Condition -> The user input is required if the Required Condition property is true. Only for Watson.

 

Availability

This property is available since GeneXus 16 upgrade 5.

Scope

Objects: Conversational Flows Instance
