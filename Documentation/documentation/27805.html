Live Editing

What does it take to make a nice application? How do we incorporate a design in our application? In the development team we know that these tasks are time consuming, and the tool we use can play a key role in making them easier.

What is Live Editing?

Live Editing is a revolutionary way to design applications: the moment we make a change in the design, we can easily observe its effect, giving our application-making process a much more natural flow..
This allows us to make changes to the Languages, Events and Themes -including transformations- and immediately see them in the running application even without saving! 

See more about Live Editing on the GX26 conference's video.

How to use it

  1. In the GeneXus toolbar select the Live Editing build mode
  2. The Live Editing tab will appear to the right of the IDE.

     
  3. When the application starts, it will attempt to connect to the PC where GeneXus is running. When this happens, the device or web browser connected will appear on Live Editing tab showing information:
  • Smart Devices
  • Web
  • Troubleshooting
