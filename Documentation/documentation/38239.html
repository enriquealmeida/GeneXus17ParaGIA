Header property (Menu object)

Specifies an image header.

Scope

Objects: Menu
Generators: Android

Description

The property applies to the root item of the Menu object and allows you to display a header image on the application (through an Image object stored in the Knowledge Base).

Notes

  • In Android, this property is ignored when the Menu object has the Control property set to "Tabs".

 

Run-time/Design-time

This property applies only at run-time.
