Show data as property

Specifies whether to show the actual values, the values as a percentage of the target values, or both.

Values

Percentages Shows actual values as percentages.
Values Shows actual values (Default value).
Values and percentages Shows actual values and percentages.

Scope

Objects: Dashboard

Description

Run-time/Design-time

This property applies only at design-time.
