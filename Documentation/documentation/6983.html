AutoFit property

Sets whether the column width will be automatically adjusted.

Syntax

control. AutoFit

Values

0 (False) Indicates that column width will not be automatically adjusted. This is the default value. 
1 (True) Indicates that the width of the columns will be automatically adjusted whenever any of the properties of a cell or group of cells is modified.
The automatic adjustment will be made in such a way that the contents of all the modified cells will be visible. 

Values

False
True

Scope
