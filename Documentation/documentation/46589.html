Local Network Usage Description property

Specifies the reason for your app to access to the local network.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple

Description

You need to add a usage description string to be presented to the user the first time an API tries to access the local network. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
