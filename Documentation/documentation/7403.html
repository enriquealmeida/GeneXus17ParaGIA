Standard Functions property at Knowledge Base level

Enables using specific functions that are not explicitly supported by GeneXus.

Values

Allows non-standard functions when specifying Functions not supported by GeneXus can be used.
Error on non-standard functions when specifying Only the standard functions supported by GeneXus can be used. This is the default value.

Scope

Generators: .NET, .NET Framework, Java
Level: Knowledge Base

Description

This property is useful when you want to use specific functions that are not explicitly supported by GeneXus or in any generator. The check will be made when the object is saved.

All functions supported by GeneXus become standard functions, even if they are only available for some generators.

Note that this property is available at the following levels:

  • Knowledge Base 
  • Environment 
  • Generator 
  • Object 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

No action is required to apply the corresponding changes when the property value is configured.
