Initial Zoom Radius Field Specifier property

Field of the SDT associated with the Grid containing the minimum distance (in meters) that should be visible around the designated center point of the map.

Description

The field indicated in this property contains the value for the minimum distance (in meters) that should be visible around the designated center point of the map. The map's zoom level will be adjusted so that at least this distance is visible in all directions.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(Android, IOS)
Controls: Grid
