Show printer dialog on reports property

Allows you to set whether you want to jump the Printer Dialog before printing your reports.

Values

No No <printer dialog=""> will be displayed.</printer>
Yes The printer dialog will be shown before your reports are sent to the printer. This is the default value.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)
