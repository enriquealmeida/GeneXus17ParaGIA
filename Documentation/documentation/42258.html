Minimum Seconds property

Sets a number of seconds, such that when the counter becomes equal or lower than it, the text set in the Minimum Text property is shown.

Run-time/Design-time

This property applies both at run-time and at design-time.

Samples

Suppose that the property Minimum Seconds = 30. When the counter becomes lower than or equal to 30 seconds, the value entered in the Minimum Text property will start to be displayed. 

 

How to apply changes

To apply changes made by this property, Build a main object.

Scope

Objects: Theme for Smart Devices
Platforms: Smart Devices(IOS)
