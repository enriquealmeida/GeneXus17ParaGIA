GXtest 4 Releases - Public Channels

GXtest is released for several versions of GeneXus from GeneXus 15 U7 and it is necessary to install a compatible version.

Install Gxtest manually (3 steps)

  1. Download GXtest version you want to install (search it below)

  2. Extract all the files over your GeneXus IDE installation (eg: C:\Program Files (x86)\GeneXus\GeneXus16)

  3. Run genexus.exe using /install parameter from your command line, to let GeneXus install the GXtest extension. 

Or using Ansible (alternative)

If you want to automate the installation of GXtest and have a little experience over Ansible, you can download and use this playbook:

https://github.com/abstracta/ansible-playbooks/blob/master/GXtest/install_gxtest_gx16.yaml

The command line should look like this:

> ansible-playbook -i hostmachinefile install_gxtest_gx16.yaml
