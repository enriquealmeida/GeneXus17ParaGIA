Diagram Guid property

GUID identifier of a workflow element.

Scope

Objects: Business Process Diagram

Description

This is the Workflow metadata GUID for the Business Process Diagram's element selected.

It is the value stored in the Workflow database.

Run-time/Design-time

This property applies only at run-time.

Availability

This property is available since GeneXus 17 upgrade 1.
