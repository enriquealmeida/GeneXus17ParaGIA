Filter dynamic property

True if the control's values are calculated dynamically from the database.

Description

Define whether the Dashboard filter's values are calculated dynamically from the database or not.

When False, set the values in the Control values property (this is the default).

When True, define how the values will be retrieved from the database by configuring the Item values and Item descriptions properties.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Dashboard
