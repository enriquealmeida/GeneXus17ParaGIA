Is Factory property

If enabled, the method will create a new instance of the object.

Values

False
True

Scope

Objects: External Object

Description

Some Static functions work as Factory methods, returning an instance of an object or instantiating an object.
This allows calling functions that create objects such as Employees, Documents, Parts, Materials, and so on.
