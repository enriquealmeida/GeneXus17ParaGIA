Max Zoom property

Sets the maximum allowed zoom level(1-20) that the end-user can apply.

Description

You have to define the maximum zoom value (1-20) that the end-user can apply to the Geolocation or Map control. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, Build a main object.

Availability

This property is available since GeneXus 15.

Scope

Platforms: Smart Devices(IOS), Web(.Net, Java)
Controls: Attribute/Variable
