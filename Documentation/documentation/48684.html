Newest Version

GeneXus 17 Upgrade 6

This article is an overview of GeneXus 17 Upgrade 6 features (compared to GeneXus 17 Upgrade 5) and what needs to be taken into account to adopt it.

It has been released at 29th. October 2021.

Download

https://www.genexus.com/developers/downloadcenter?data=5934

Overview

This is a very important upgrade that improves the way you model your design. Improved modeling always results in better applications, and so it happens when you use the Design System Object. With this domain-specific language for modeling designs, you get more accuracy and performance at runtime.
Furthermore, generated applications get improved by new UI Controls and capabilities for Native Mobile applications, technology improvements for the back end, new integration capabilities, and last but not least, security updates and features.

Design Systems and DesignOps

  • Design System Object
  • Several improvements to accuracy when importing designs from Sketch

Native Mobile

  • Toggle Button Group control 
  • ScrollIntoView method
  • Android-specific
    • Line Dash Pattern property for MapPolygon and MapRoute classes.
    • Dynamic Services URL can be scanned.
  • iOS-specific
    • Apps are now compiled with the iOS 15 SDK.
    • Support for Scanner Control new properties.

Cloud

  • Back-end services of Native Mobile and Angular apps can be deployed to Azure Functions.

.NET Core Generator (.NET 5)

  • SAP Hana Database, Informix, DB2 Universal Database
  • Google Cloud Storage Provider

Integration

  • cURL Inspector is a good shortcut to import simple services from a cURL sample provided by the service provider.
  • OpenAPI import tool supports importing services described by the OpenAPI 3.0 specification.
  • JSON serialization improvements, and new External Name property in API object

Security

  • GAM supports these flavors:
    • .NET Core + SAP HANA Database
    • .NET Core + Informix
    • .NET Core + DB2 UDB
  • OpenId Connect support for authentication with external Identity Providers.
  • Support for authentication with external Identity Providers in an Identity Provider built with GAM.
  • GeneXus policies regarding security issues in applications have been updated.

BPM

  • GXflow supports these flavors:
    • .NET Core + SAP HANA Database
    • .NET Core + Informix
    • .NET Core + DB2 UDB

Testing

  • Added Log Level property for web tests execution
  • Improved AddCapability command. Also, now supports adding extensions in runtime, selecting browser binary, among other settings
  • Added Compare Image command for web UI tests
  • Running web UI tests on NetCore + Kestrel is now supported

Compatibility

  • XCode 13 is required for building iOS applications.
  • Some bug fixes related to navigation, data handling, and serialization may imply compatibility issues. Please check the details.
  • Extensibility: Deprecations and Removals in Android and Apple flexible clients
