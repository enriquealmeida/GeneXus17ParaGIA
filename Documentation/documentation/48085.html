Default Template Layout property

Specifies a Template used to generate default Transactions and WorkWith web layouts.

Scope

Objects: Design System
Generators: Android, Angular, Apple, Java, .NET, .NET Core

Description

This property is used to indicate the set of Templates of the Work With Pattern that are used with the Design System Object. 

The designs, in addition to their set of classes that give them style, have a particular structure in the form, that is why the screens are generated differently depending on the value you select for this property.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 upgrade 6.
