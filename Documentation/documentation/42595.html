Test Target property

Defines the main object to use as a test target.

Scope

Objects: UI Test
Generators: Android, Apple

Description

Every UI Test object object requires a Main object that indicates the application it was written for, that is, the application it is meant to test.

That Main object is indicated in the Test Target property, and it is required. If not specified, the following error is shown: 

error: Test '<test_name>' has no main target set
error: No main targets found to test

 

 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.

Availability

This property is available since GeneXus 16 upgrade 5.
