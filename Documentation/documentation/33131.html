Include Sparkline Property

Include Sparkline Property it's a QueryViewer control and a Query object property. It's only available when the Type property in QueryViewer control has the Card value assigned and allows displaying (under the Card indicator) a Sparkline chart showing the indicator's evolution.

Values

False The Sparkline chart it's not displayed. This is the default value.
True A Sparkline chart it's displayed under the Card's indicator.

Description

If the Query object includes a Date or DateTime attribute, a Sparkline chart can be calculated and shown by setting the Include Sparkline Property in True:

The period displayed in the chart it's determined by the query's structure (this means that all the returned data define it).

Availability

This property's available as of GeneXus 15 Upgrade 2
