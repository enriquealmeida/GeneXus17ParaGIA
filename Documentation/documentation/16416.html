ToUniversalTime method

Converts its value to Coordinated Universal Time (UTC), assuming that the DateTime to convert to is the server's time zone.

Syntax

&DateTime.ToUniversalTime()

Type returned:
DateTime

Scope

Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Samples

&MeetingInUTC = &MeetingDateTime.ToUniversalTime()
