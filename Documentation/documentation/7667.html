Language localization property

The purpose is to associate Language localization to the language.

Description

The language localization is a string (with no embedded spaces or special symbols) that, in conjunction with values of "ISO Language Code" and "ISO Country Code", identifies the Language. For example: different regions in a country may use different terms for one same thing. Both regions are in the same country and speak the same language. This property lets you create one Language object for each region. One should be, for example "Spanish-Uruguay" and the other "Spanish-Uruguay-East".

Scope

Objects: Language
Platforms: Web(.Net, Java)
