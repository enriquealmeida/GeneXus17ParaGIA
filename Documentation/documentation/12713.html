Substring Method

Returns a substring from a given string.

Syntax

s1.Substring( n1 [, n2])

Where:
s1
   Must be a character expression.

n1 and n2 
   Must be numeric expressions.

  • If n2 = 0 an empty string is returned.
  • Omitting n2 is equivalent to setting it to s1.length() - n1 + 1.  

Type Returned
Character

Description

Returns a substring of s1 beginning at position n1, with a length of n2 characters.

Example

&Txt = '1234567890'
&Substr = &Txt.Substring(5,2)  // Result: &Substr = '56'

Constraints (not apply to .NET, Java and Ruby):

1 <= s1 length <= 255
 0 < n1 <= s1 length
n2 >= 0

Availability

Omitting n2 is possible as of GeneXus 15 Upgrade 7

Scope

Objects: Procedures, Transactions, Web Panels, Work Panels
Languages: .NET, Java, RPG, Cobol, Smart Devices (Android, iOS)
