Map type property in Query

Specifies the map type to display.

Values

Bubble
Choropleth

Scope

Objects: Query
Generators: Java, .NET, .NET Core

Description

Use this property to select one of the available map types.

A choropleth map is a type of map in which a set of pre-defined areas is colored in proportion to the value of a statistical variable that represents an aggregate summary of a geographic characteristic within each area, such as population density or per-capita income.

In a bubble map, each geographic area is represented by a circle whose size is proportional to the value of the variable. This type of map is perfect to compare proportions over geographic areas without the problems caused by the areas' size, which are very common in choropleth maps.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.

Availability

This property is available since GeneXus 17 upgrade 4.
