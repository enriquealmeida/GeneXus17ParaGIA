Animation Duration property (for Animation Theme class)

Duration of the animation (in milliseconds). Empty uses the platform default.

Description

This property is available in Theme object classes only when Animated property is enabled. Its aim is to improve the UI/UX of a smart device application when it includes transformations.

Values

It accepts positive integer values indicating the time in milliseconds (ms) that the animation will last.

Notes

  • As of GeneXus 15 Upgrade 4, altought the developer can enable the Animation property in order to include a smooth transition when a cell is collapsed, the Animation Duration proprety is not taken into account for this scenario. It only applies for transformations.
