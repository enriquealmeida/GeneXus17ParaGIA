Cache storage size property

To configure the amount of memory of the server running the application within the process in execution, dedicated to storing the data cache.

Description

Values

0 = Infinite quantity of  KBytes
Other values

Default = 0

The value of the property is expressed in KBytes, where zero represents infinite.

Note: This property is enabled if the value of the Database access caching property is set to True.

Scope

Platforms: Web(.Net, Java)
