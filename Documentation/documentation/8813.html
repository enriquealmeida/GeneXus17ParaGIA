Reload method

Reloads all the possible values of a Dynamic Combo BoxÂ from its corresponding table.

Syntax

Control.Reload( )

Scope

Controls: Dynamic Combo Box, Dynamic List Box
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to Genexus X Evolution 3)

Description

Reads the corresponding table and loads all the values into the combo box.
This property applies only at execution time.

Note: This method is not supported in Web environments: SAC#40258
