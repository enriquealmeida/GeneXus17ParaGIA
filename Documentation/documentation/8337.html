BrowserVersion function

Returns a browser’s description.

Syntax

BrowserVersion()

Type returned:
Character

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Description

The BrowserVersion function returns the description corresponding to the browser’s version (or agent). If the browser used is not recognized (the function BrowserId returned 0), it returns an empty string.

Note: It can be used in any GeneXus object. The result makes sense only if the object was called directly or indirectly by a Web Panel or if executed in a Web environment.
