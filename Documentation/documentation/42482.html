SizeAttribute property

The attribute or variable indicating the size of the square item in an SD Image Map control

Description

Indicates the size of the square item in an SD Image Map control (blue)

As of GeneXus 16 Upgrade 3, the property use is replaced for Height Attribute property and Width Attribute property, allowing to specify squares and rectangles as items.

 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(Android, IOS)
Controls: Grid
