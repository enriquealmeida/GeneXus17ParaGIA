PrintDocument function

Prints a document (it will not open it if not necessary).

Syntax

&ret = PrintDocument(‘document’)

Type Returned:
Numeric

Scope

Objects: Procedure, Transaction, Work Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)

Availability

Since Genexus 16 Upgrade 2, you must include the GXClasses.Win.dll file manually in your deploy when you use this function. See SAC#44468

This is available for Windows applications and batch procedures.

Security Tips

If this function is used, sanitizing external user's inputs is a must.
