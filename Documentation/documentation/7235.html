Column title property

Title associated to the attribute/variable in the columns of a Grid.

Description

The defaults for Transaction forms, and Selection/List in a Work With for Web/Smart Devices objects provide a context, and therefre apply contextual title rules

In other words, Attributes "belonging to the context" will use the Contextual Title, and attributes that do not (e.g. foreign keys, inferred attributes) will use the Column Title.

Scope

Objects: Transaction
Platforms: Web(.Net, Java)
