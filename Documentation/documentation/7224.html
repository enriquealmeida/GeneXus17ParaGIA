Autonumber step property

This property is offered for a key numeric attribute, if its Autonumber property is set with True value. It allows to specify the field increment (between two records).

Description

Values

The values to be given to the property must correspond to number (without decimals).

Scope

Objects: Procedure
Platforms: Web(.Net, Java)
