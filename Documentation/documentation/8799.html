InputType property

The purpose of this feature is to subtitute what the user needs to type in an attribute with a more user friendly value. This property is not available when the attribute is a formula.

Values

Values The user will enter the attribute value. This is the default value.
Descriptions Will allow the user to enterÂ the description attribute. The description attribute must be in the transaction structure.

Description

In order to make this work, each description must match only one Value. The description must be a Candidate Key, so as to create a unique index for each description.

Availability

InputType = Descriptions on Android is available as of GeneXus X Evolution 3 Upgrade 1.

Scope

Objects: Panel for Smart Devices, Transaction, Web Panel, Menu for Smart Devices, Work With for Smart Devices
Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
