Generate Web UI property

Enables web user interface generation for this instance.

Scope

Objects: Conversational Flows Instance

Description

If set to FALSE, the Web objects related to the Conversational Flows object are not generated.

For example, the main object <InstanceName>WebUI (main Web Panel object which calls the "PanelChatWeb" resource object) is not generated. The objects specified in the Generated Web Component property of the Flows are not generated neither.
