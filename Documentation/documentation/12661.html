Delete row image property

Sets the image name to be displayed for the first column in transaction grids, which offers to delete a row.

Description

Each row of a grid in a Transaction object has a column containing a control that offers to delete the row.

This property allows you to set the image name to be displayed for this column. 

The default value is that of the DeleteRowImage image.

How to apply changes

To apply the corresponding changes when the property value is configured, execute Build any object with the purpose of generating the *.config files.
