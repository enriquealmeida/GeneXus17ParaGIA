Is Right To Left property

Represents those languages written from right to left.

Values

False
True

Scope

Objects: Language

Description

Specifies if theÂ Language object is:

  • RTL (right-to-left) - The text starts on the right side of the page and continues to the left.
  • LTF (left-to-right) - The text starts on the left side of the page and continues to the right.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16 upgrade 3.
