SAP Cloud Foundry User property

Sets the user to connect to your account of SAP Cloud Platform Cloud Foundry Environment.

Scope

Level: Deploy Target Options

Description

You must specify the User for your account in SAP Cloud Platform Cloud Foundry environment. It is the same user you use to access the Cockpit (control panel) of SAP Cloud Platform Cloud Foundry environment.

Availability

This property is available since GeneXus 16 upgrade 10.
