ClientStorage.Clear method

Clears all pairs of key-value stored in the device via ClientStorage.Set method.

Syntax

ClientStorage.Clear()

Type returned:
None

Scope

Objects: Objects for Native Mobile applications development, Offline Procedures
Generators: Android, Apple

Availability

This feature is available since GeneXus X Evolution 3
