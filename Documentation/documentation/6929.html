ErrDisplay Property

It indicates whether error messages will be displayed or not.

Syntax

&DataType.ErrDisplay

Type Returned:
Numeric
 
Where:
DataType
Is the variable name of the ExcelDocument, MAPISession, OutlookSession, POP3Session, SMTPSession or WordDocument type.

Values

1: Specifies that the message displayed to warn the user of an error occurrence will include the error’s text.
0: Otherwise.

Default Value = 0

Scope

Extended Data Types: ExcelDocument, MAPISession, OutlookSession, POP3Session, SMTPSession, WordDocument
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
