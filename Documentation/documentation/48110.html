Maximum pool size property

Sets the maximum number of HttpClient connections handled by the pool.

Scope

Generators: Java
Level: Generator

Description

If you add httpclient-4.5.13.jar and httpcore-4-4-13.jar to the classpath, connections allow for reusability (persistent connections). By setting a value to the Maximum pool size property, you can reuse the same connection up to the value specified. The property supports concurrent tasks.

Default Value: 1000

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute Build any object with the purpose of generating the *.config files.

Availability

This property is available since GeneXus 17 upgrade 4.
