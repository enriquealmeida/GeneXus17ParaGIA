Regular Expression property (GeneXus 17 Upgrade 3)

Specifies a regular expression for string-type attributes or variables (Character, VarChar, etc.), with which to validate the data entered.

Scope

Objects: Attribute, Domain, Variable

Description

Verification is done at runtime when the value is entered in a Web object by an end-user, or with a Business Component by a program.

This property is not available when the attribute is a formula.

Note: For domains, attributes, or variables based on domains of attributes, the property is inherited and can't be modified.
