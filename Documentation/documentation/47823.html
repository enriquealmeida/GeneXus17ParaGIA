Disambiguation property

Enables disambiguations for this instance.

Scope

Objects: Conversational Flows Instance

Description

If its value is True, it activates all disambiguation functions. Otherwise, when False, the disambiguation function is not considered.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 upgrade 3.
