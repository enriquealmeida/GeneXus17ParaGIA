KeyPhrases procedure

Gets the key phrases (including keywords) from a written text.

Parameters

  • in:&text:: Text, GeneXusAI
    A plain text to get its key phrases.

  • in:&language :: Language, GeneXusAI
    The language of the input text for improving the output accuracy.

  • in:&provider :: Provider, GeneXusAI.Configuration
    Provider settings.

  • inout:&Messages :: Messages, GeneXus.Common
    A collection of warning and error messages returned by the task. You should check in your code if an error was returned. Refer to error codes and descriptions for more information.

  • out:&Labels:: Label, GeneXusAI (collection)
    A set of labels indicating the key phrases detected.

Notes

  • The term 'key-phrase' also includes 'key-words'.
  • The labels are sorted from higher to lower confidence.
  • Baidu AI provider only returns a result when you input Chinese-text.
    e.g.
    Input : "提出的第一个问题是：什么是GeneXus？ GeneXus是一个自动生成软件程序的工具，例如Web应用程序和智能设备，始终处于技术发展的最前沿。"
    Output : ["软件"]

Availability

This procedure is available as of GeneXus 16.

  • As of GeneXus 16 upgrade 1:
    - Google Cloud AI is available.
  • As of GeneXus 16 upgrade 2:
    - Amazon WS and Tencent AI are available.
  • As of GeneXus 16 upgrade 3:
    - Baidu AI is available.
  • As of GeneXus 16 upgrade 4:
    - Alibaba AI is available.
  • As of GeneXus 16 upgrade 8:
    - SAP Leonardo has been deprecated
