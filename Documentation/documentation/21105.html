Server Linked Knowledge Base property

This property is defined when the Send Knowledge Base to GeneXus Server or the Knowledge Base from GXserver (X Evolution 2) actions are performed. It defines the name of the Knowledge Base to which the local Knowledge Base is connected to in the GeneXus Server.

Values

A valid KB alias from the GeneXus Server instance.

Example

TravelAgency
