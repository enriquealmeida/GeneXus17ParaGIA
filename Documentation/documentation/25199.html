Object property (BPD Task)

Indicates the Panel object that will be invoked when executing the task from a mobile device.

Scope

Objects: Business Process Diagram
Controls: Task

Description

This property indicates the GeneXus Object for a Native Mobile application to be called when using the GXflow Client and ignored when executing the GXflow Client.

Availability

This property is available since GeneXus 17 upgrade 7.
