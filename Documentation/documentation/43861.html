Type Property

Returns the qualified name of the Object

Syntax

object.Type


Where:
object 
Is the name of an object of the Knowledge Base.

Type
Character

Description

This property returns the Qualified Name of the object

Run-time/Design-time

This property applies only at run-time.

Scope

Objects All
Platforms Web (.NET,Java), Smart Devices (iOS, Android)
