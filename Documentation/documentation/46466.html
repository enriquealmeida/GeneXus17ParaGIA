Enable Zoom property

Allows zooming in/out on an image.

Values

False
True

Scope

Generators: Android, Apple
Controls: Attribute/Variable (Control Type: Advanced Image)

Description

Allows zooming in/out on the images. This property is directly related to the properties Max Zoom, Max Zoom Relative To, Zoom Outside Control.

Run-time/Design-time

This property applies only at design-time.

Samples

How to apply changes

To apply the corresponding changes when the property value is configured, Build a main object.
