Web Transactions

Transaction objects with web forms are usually known as “Web Transactions”. They are generated to be available on Internet applications, with all the strength of this type of object in data entry because they allow solving the data entry without defining Web Panels and Procedures to solve it, performing all the necessary referential integrity controls automatically.

Web Transactions are executed in browsers.
