Dynamic List Box

It’s similar to a Dynamic Combo Box where the items are loaded from a database table. The collection of items is loaded from a database table in runtime; it’s also possible to add items manually. The options are detailed in the Dynamic Combo Box section.

Item Descriptions

Specifies the attribute from which the descriptions will be loaded. Mark the check box if you want to display the list box alphabetically ordered by the descriptions. The selected attribute must be string type attribute. Selected attribute for descriptions.

Item Values

Specifies de selected attribute from which the values will be loaded. Selected attribute from which the values will be loaded.

Sort Descriptions

Sorts the attribute by description.
