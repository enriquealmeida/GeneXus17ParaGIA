QueryString Property

This property returns the URL portion that appears after the “?” sign; i.e.: the parameters. 

Syntax

&DataType.QueryString

Type Returned:
String

Scope

Extended Data Types: HttpRequest
Languages: .NET, Java, Ruby(up to and including GeneXus X Evolution 3)
