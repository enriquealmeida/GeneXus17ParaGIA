Disabled warnings property

To indicate a list of specifier warning codes to be ignored at specification time for each object. This property applies to both the Environment and the object itself. When the object and the Environment contain matching values, the object values are applied first. In the absence of object values, all values included in the Environment property are applied.

Description

This property allows the specifier to ignore certain warning codes, and not display them in the navigation report. The list must be separated by spaces or commas, i.e.:

            Spc0005 Spc0038

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Platforms: Web(.Net, Java)
