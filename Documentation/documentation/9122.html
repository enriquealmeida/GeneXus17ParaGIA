Servlet directory property

This property is read-only and it is generated based on the Tomcat path property. Allows specifying the directory where the servlets generated will be transferred. At compile time of each Web Object, the generator will copy it to the specified directory.

Scope

Platforms: Web(Java)

Description

Notes:

  • The path specified must be relative to the client, that is the path seen from the PC in which it is working with GeneXus.
  • The generator will copy, in addition to the .class files, the .cfg files which contain the information of the entered properties; therefore, if some value of the model properties is modified, it is necessary to generate again at least one object and to compile some Web Object. In addition, so that this change is taken into account at runtime, it is necessary to restart the Servlets server.
