Drag event in Web

It allows you to code the activity of the element to be dragged.

Syntax

Event control.Drag( [out:] &parameter )
Event_code
EndEvent

Where:

control
The name of the form control over the drag operation is performed.

&parameter
The parameter is only one, and it is always an out variable.

Event_code
Code associated with the event. The parameter will be loaded here with the data you want to drop into another control as a consequence of the drag operation. To drag several elements of the information you have to load them into an SDT, being the &parameter variable.

Scope

Controls Image control, Text Block control, Table control, Free Style Grid control, Grid control, Web Component control, Button control
