Interactive Activity Properties

They help you define the transaction's behavior.

Name: Interactive activity name

Web Application: Genexus object associated to task. Pressing the button, the following dialog will be displayed:

By clicking the same button again, you will be able to select the object.

Visible in history: Indicates whether the task must be shown in the history.

Roles: Associated roles to execute the task.

Subject Rule: It allows defining the task's subject at runtime. It admits characters, attributes and relevant data (eg."Customer registration number"+CustomerId+&relevantData).
