Height Attribute property

The attribute or variable indicating the item's height in a SD Image Map control

Description

The value will be used to set the height of the element (blue)

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Availability

This property is available since GeneXus 16 upgrade 3.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(Android, IOS)
Controls: Grid
