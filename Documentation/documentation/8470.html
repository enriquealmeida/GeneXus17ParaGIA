Time function

Returns the time in the 'hh:mm:ss' format.

Syntax

Time()

Type Returned:
Character(8)

Scope

Objects:  Procedure, Transaction, Web Panel

Notes:

  • In the iSeries Environment, this function returns the start time of the Job. See the Systime function if you want to use the actual System Time. 
  • In a PC Environment, it returns the System Time, so Time and Systime functions are equivalent.  
  • In a Client/Server Environment, this function returns the time of the PC.
