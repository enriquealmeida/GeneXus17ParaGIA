GXSpreadsheet Control

Purpose

Allows you to add a spreadsheet to your web applications.

Scope

This ControlType applies to Varchar or Long Varchar Domains, Attributes and Variables.

Usage

Set the ControlType propertyÂ  to "GXSpreadsheet"

Properties

  • Width
  • Height
  • Autoresize
  • Rows (quantity of rows by default)
  • Columns (quantity of columns by default )
  • CellWidth
  • CellHeight
  • Title
  • Additional Info

GXspreadsheet is based on JQuery Sheet.

This control is available as of GeneXus X Evolution 1 Upgrade #1.

Considerations

It is only supported one control per Web page otherwise an error will be triggered.
