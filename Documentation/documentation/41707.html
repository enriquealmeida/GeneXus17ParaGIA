Image Height property

Image height in pixels

Scope

Objects: Image
Platforms: Smart Devices(IOS)

Description

Read-only property. Returns the image height in pixels.

Run-time/Design-time

This property applies only at run-time.

Samples

&height = &image.ImageHeight

Availability

This property is available since GeneXus 16 upgrade 2.
