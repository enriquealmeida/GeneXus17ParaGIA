Display Value property

Indicates whether the control should display a bubble with the current value upon interaction.

Values

False
True

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Generators: Android, Apple
Controls: Attribute/Variable (Control Type: SD Slider)

Description

Indicates whether the control should display the value. This value is shown as a floating bubble above the slider.Â 

If the control is on the upper screen, it could lack the space required to display the value, and in that case, it will not be shown. To avoid this undesired behavior, it is advisable to leave a separating space between the control and the application bar. It is False by default.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.
