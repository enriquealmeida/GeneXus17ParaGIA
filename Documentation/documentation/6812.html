Get method

Returns a string that corresponds to the session's key entrance.

Syntax

&WebSession.Get(Key)

Where:
Key
   Is the Session key, and it has to be a String

Type Returned:
String

Scope

Extended Data Types: WEBSession
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)
