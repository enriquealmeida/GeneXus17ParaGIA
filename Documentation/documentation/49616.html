GeneXus 17 Upgrade 8

This article is an overview of GeneXus 17 Upgrade 8 features (compared to GeneXus 17 upgrade 7) and what needs to be taken into account to adopt it.

Upgrade 8 has been released on February 21st, 2022.

Download

https://www.genexus.com/developers/downloadcenter?data=5944

Overview

This upgrade provides important advancements on GeneXus modeling capabilities, support for the latest technologies, and security enhancements for the generated applications.

Image and File objects can now be part of Modules, and these now support sharing design systems. This allows for more powerful and efficient reuse of design, either among different components of a single application or even among separate applications or knowledge bases. 

The Microsoft technology stack was updated and extended so that the .NET Generator now supports .NET 6 - the latest LTS version of the platform, which includes numerous performance improvements and their resulting lower compute costs.

Web applications may now use time-based one-time passwords (TOTP). These passwords are not vulnerable to replay attacks, eliminate the need for users to remember or store fixed passwords and avoid the risks implied by the common practice of choosing the same password for multiple services.

IDE

  • Superfast Create KB when using SQL Server 2019 or LocalDB 15 and UTF-8 collation. (SAC 50653).
  • Modularization
    • Image object and File object can be modularized
    • A Design system can be packaged with a module (That is, the Design System Object, User Control object, Stencil object, Image object, and File object). More information...
    • IDE Icon overlay for objects that are private or internal

Security

  • Support for Time-based One Time Password (TOTP) in web applications

.NET

  • Naming according to Microsoft's strategy
    • .NET Generator renamed to GeneXus .NET Framework generator
    • .NET Core Generator renamed to GeneXus .NET Generator
  • .NET Generator now generates .NET 6 (LTS)
    • .NET Generator Requirements

Native Mobile

  • Panel to configure Services URL can be customized

Reporting

  • Support for click event in maps visualization in Queryviewer control. See ItemClick Event
  • Support for Geopoint attributes in Choropleth map visualization with Query viewer control
  • New properties in Queryviewer control to hide totals for Row and Columns. See Total For Rows property in QueryViewer and Total For Columns property in QueryViewer
  • Guide to include countries maps that are not distributed by default. See How to use maps that are not provided by QueryViewer

Testing

  • A New tests runner is built-in to reduce build times
  • New commands to take screenshots of controls and elements in web UI tests
  • Test data can be read from CSV files

Compatibility & Setup highlights

  • Once opened with this version, Knowledge bases will display images and files in the root module when opened with older versions. SAC 50730
  • Changes in the default value of SOAP Protocol property. SAC 50679
  • .NET: Changes in libraries (and licensing) related to PDF reports and indexing PDF files. SAC 50700
  • .NET 6 required in .NET Generator. .NET Generator Requirements
  • XCode 13.2.x required for generating for Apple technologies. Apple Requirements
  • GAM database version requires to be 4.0.6. SAC 50807
