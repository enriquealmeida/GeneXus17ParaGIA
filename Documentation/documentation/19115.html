Android Maps API Key property

Indicates the API key to use Maps services.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)

Description

This property must be set when using Maps Control or SD Geolocation Picker Control.

Value

An alphanumeric key that can be obtained from Baidu Maps or AutoNavi's Gaode Maps depending on the value of Android Maps API property.

Notes

  • When modifying the value of this property, make sure to update the related Application Signing section (Key Store File, Alias, and passwords).
  • As of GeneXus 15 Upgrade 5, this property applies only when Android Maps API is Baidu Maps API or AutoNavi Maps API.
  • Until GeneXus 15 Upgrade 4, this property also applies to Google Maps API v2 and you must refer to HowTo: Get an API Key from Google in order to fill this property. Higher upgrades must use Android Google Services API Key property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
