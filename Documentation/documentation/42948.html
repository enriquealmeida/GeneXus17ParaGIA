Flips for Right to Left property

Boolean value that indicates whether the image should be flipped in a right-to-left layout.

Values

False
True

Scope

Objects: Image
Platforms: Smart Devices(IOS)

Description

In RTL languages, some images need to be flipped horizontally, and others don't. For example, Foreign Key pickers and selectors should be automatically flipped depending on the selected language.

With this property, the developer can flip elements as needed.
