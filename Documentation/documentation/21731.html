InputHistory property

Assists the application user to enter information in a field, by remembering the last values entered in it so the user can select one of those and save them to input information.

Values

False
True

Scope

Generators: Android
Controls: Attribute/Variable

Description

This property indicates whether to assist the application user or not, to enter information in a field. This is done is by remembering what has been entered in a field on a screen. By remembering this, the next time a user enters information in that field the app will show the last values entered, giving the user the opportunity to select one of the previous values and save them to input information.

Default value: True

Considerations

  • This property doesn't work for Date based fields.

  • When the IsPassword property is set to yes the Input History property is disabled.

  • A value is "saved" for the input history when an action is executed on the screen the field is on.
