Min Height property

This attribute impose minimum height constraint in dips. The control won't be shrank less than the value of these attributes (varies based on the Flex Direction attribute as to which attribute imposes the size constraint along the main axis) regardless of the Flex Shrink attribute.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.||}}}
