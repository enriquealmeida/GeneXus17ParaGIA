iOS Application Icon property

Apple application icon. It requires several image sizes for all the different devices.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple

Description

This property allows you to define the application icon. 

The image to be loaded depends on the size and resolution of the device.
If you provide only one image, GeneXus generates automatically images with a smaller size.
If more than one image is provided, GeneXus identifies which image is suitable for the device.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.
