Generate Android property

Enables the Native Mobile generator to generate the code corresponding to the Android platform.

Values

False
True

Scope

Generators: Android
Level: Front end

Description

Default value: False

When you create in your Knowledge Base the first object for mobile development, the property is automatically set to True.

In addition, the Android Specific group will be added allowing you to set the necessary properties to generate applications in this platform.
