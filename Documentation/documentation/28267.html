Label Width property in Attribute class of a Theme object

Establishes the width for an Attribute class of a Theme object.

Scope

Objects: Theme
Generators: Android, Apple

Description

Setting this property allows you to align all labels of the same length. 
When a numeric value is set, it is automatically switched to DIP format. 

Considerations

When the label width has a very small value and overlaps the control, you may need to adjust the RowSpan property from the control.

Run-time/Design-time

This property applies only at design-time.

Samples

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
