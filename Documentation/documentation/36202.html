Selection Flag property

Used when a Grid control allows  multiple selections for indicating which is the attribute or variable that keeps the selection state (its value is bounded with the selector on the screen).

Run-time/Design-time

This property applies only at design-time.

Notes

  • If the value of this property is a variable based on an SDT or BC, the developer must also set the Selection Flag Field Specifier property.
  • As of GeneXus 15 Upgrade 6 this property is display only if Enable Multiple Selection property is True. 

Scope

Controls Grid control
Platforms   Smart Devices (iOS, Android)

Availability

This property is available as of GeneXus X Evolution 3.
