DisplayMode property

Allows indicating how messages will be displayed in the Error Viewer control.

Syntax

control. DisplayMode

Where:
Control
    Is the name of a control inserted in the form.

Values

Bullet List It displays the messages one below the other, with bullets. Default value.
List It displays the messages one below the other

Run-time/Design-time

This property applies both at run-time and at design-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
Controls: ErrorViewer
