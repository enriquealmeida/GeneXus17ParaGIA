SearchResult Data Type

To unify the interaction functions with searching keywords in an index.

Syntax

&SearchResult.Property

Where Property:
             MaxItems
             ElapsedTime

Properties

ElapsedTime Time (in milliseconds) taken to do the search.
MaxItems Total hits resulting from the search.


Methods

SearchResultCollection

Items()

Scope

Objects: Procedures, Reports, Transactions, Web Panels
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
