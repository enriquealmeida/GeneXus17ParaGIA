Confidence domain

Represents a confidence value between 0.000 and 1.000 in the context of GeneXus Artificial Intelligence, where higher values indicate a more reliable prediction.

Scope

AI Task Classify procedure, DetectFaces procedure, DetectObjects procedure, DetectScene procedure, OCR procedure, DetectLanguage procedure
Platforms  Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This domain is available as of GeneXus 16.
