Interrupts activity property

Indicates whether the activity will be interrupted when the deadline arrives or not.

Values

False
True

Scope

Objects: Business Process Diagram

Description


Values

True The activity will be interrupted when the deadline arrives. This is the default value.
False The activity can still be executed, but it will have an "expired" status.
