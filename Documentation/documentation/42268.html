Max Value Image property

Allows the set-up of an image representing the control's maximum value.

Run-time/Design-time

This property applies both at run-time and at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(IOS)
Controls: Attribute/Variable (Control Type: SD Slider)
