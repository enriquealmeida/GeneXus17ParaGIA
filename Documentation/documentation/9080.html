Database name property

Used when the environments need database names. It indicates the database name which will contain the application's tables and indexes.

Description

Values

The application's database name.

Technologies to Access Data

JDBC, ADO.NET

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Platforms: Web(.Net, Java)
