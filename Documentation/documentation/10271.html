WorkflowAttribute Data Type

Description

This object represents a generic attribute. It is used in other objects (currently, in the Activity and Process Definition objects) to describe extended attributes.

Properties

Property Type Access Description
Name WorkflowName Read/Write Attribute name
Value WorkflowValue Read/Write Attribute value
