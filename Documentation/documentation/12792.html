Subtitle Property

It allows assigning a text block subclass, like the text block classes in the Theme object of the Folder View Customization node.

Values

Any of the values included in the combo box.

Default = Subtitle

Scope

Objects: Web Panel object (in WW Configuration Preferences).
Interfaces: Web
