GAMSDLogin object

GAMSDLogin object is an entry panel for smart devices applications, which is incorporated automatically to the KB when Enable Integrated Security Property is set to yes.

This object is the login default panel, and is called at application startup.

It´s basically a form where  the user can enter username and password to login. In case that the user has no username to login to the application, there´s a "Register" button by which a new user can be added to the GAM repository.
