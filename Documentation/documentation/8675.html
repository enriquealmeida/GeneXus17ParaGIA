Collapsed property

Indicates if the grid will be collapsed or expanded when opening the object.

Syntax

control. Collapsed

Where:
Control
    Is the name of a grid control inserted in the form

Default value: False

Values

False
True

Description

Collapsed is only enabled if AllowCollapsing property is set to True.

Run-time/Design-time

This property applies both at run-time and at design-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
