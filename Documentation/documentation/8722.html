Fill property

Sets: the controls transparency (if the background color is specified or not). In controls with Read Only set to true, the background color is generally white. In ExcelFont Data Type the object that controls the background of a cell.

Values

False
True

Scope

Objects: Procedure, Transaction, Web Panel
Platforms: Web(.Net, Java)

Run-time/Design-time

This property applies only at design-time.
