Google API Key property (at Environment level)

Sets a Google API Key to be able to call Google Location services.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16 upgrade 5.

Scope

Platforms: Smart Devices(Android, IOS), Web(.Net, .Net Core, Java)
Level: Environment
