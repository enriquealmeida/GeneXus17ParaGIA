Animation Key Attribute property

Attribute or variable which is the identifier (key) of the entity to animate (i.e. VehicleId).

Description

It applies only to Grids whose:

  • Control Type property = SD Maps
  • Animations Layer property = True

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Availability

This property is available since GeneXus 16 upgrade 5.

Scope

Platforms: Smart Devices(IOS)
Controls: Grid (Control Type: SD Maps)
