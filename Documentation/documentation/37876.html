Enable Logging property

Enables Logging properties group for debugging the application by generating log messages.

Description

This property can be found in Native Mobile Main object properties and allows the developer to enable the Logging properties group, which has the following properties:

  • Default Log Level property
  • Offline Synchronization Log Level property
  • Http Connections Log Level property
  • Offline Data Base Access Log Level property

Once the developer sets the desired log level, they can inspect the log messages on each plataform

Run-time/Design-time

This property applies only at run-time.

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Objects: Main Smart Devices
Platforms: Smart Devices(Android, IOS)
