ExitBeaconRegion event

This event is Invoked when the user exits a monitored beacon region added previously with the AddBeaconProximityAlert or AddBeaconProximityAlerts Methods.

Syntax

Event LocationAPI.ExitBeaconRegion(&<BeaconRegion>)
      Event_code
EndEvent

Where:

BeaconRegion
    An object containing information about the beacon region that was exited. This parameter is based on the BeaconRegion Data Type.

Event_code
      Code associated with the event. 

 

Availability

As from GeneXus X Evolution 3 Upgrade 2.
