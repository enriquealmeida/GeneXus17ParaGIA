Response Name property (Conversational Flows)

Defines a Response Name to identify it easily.

Scope

Objects: Conversational Flows Instance

Description

This property defines a name for the Chatbot Response of the Flow of a Conversational instance.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17.
