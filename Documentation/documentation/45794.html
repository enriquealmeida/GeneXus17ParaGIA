SAP Cloud Foundry Space name property

Sets the space name for SAP Cloud Platform Cloud Foundry Environment.

Scope

Level: Deploy Target Options

Description

You must specify the Space Name for your account in SAP Cloud Platform Cloud Foundry environment. You can get this information in the Cockpit (control panel) of SAP Cloud Platform in the Spaces section of your Cloud Foundry environment.

Availability

This property is available since GeneXus 16 upgrade 10.
