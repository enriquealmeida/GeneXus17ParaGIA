Set Method

This allows you to add information to the current active session when using the WebSession data type. In the case of the expression data type, it sets a value to a variable. The method can also be applied to set a variable to year-month-date (time-minutes-seconds), as do functions YMDtoD and YMDHMStoT.

Syntax

&WebSession.Set(Key, Value)
&expressionDataTypeVariable.Set(Key, Value)
&date.Set(numeric-expression1,numeric-expression2,numeric-expression3)
&datetime.Set(numeric-expression1 ,numeric-expression2 ,numeric-expression3 , [numeric-expression4 ,numeric-expression5 ,numeric-expression6] )

Where:
Key
   Is the Session or Expression key, and it has to be a String.

Value
   The value has to be a string.

numeric-expression1
   Represents the year. This value does not adjust according to the property ‘First year of 20th century’.

numeric-expression2
   Represents the month.

numeric-expression3
   Represents the day.

[ numeric-expression4 ]
   Represents the hour. The time value must be specified in 24-hour format.

[ numeric-expression5 ]
   Represents the minutes.

[ numeric-expression6 ]
   Represents the seconds.

Scope

Extended Data Types: WEBSession, Expression
Standard Data Types: Date, DateTime
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
