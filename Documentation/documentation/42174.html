Y-Data To Attribute property

Attribute or variable — with the value of the identifier defined on the Y-Axis Value Field Specifier — which indicates the final position.

Scope

Generators: Android, Apple
Controls: Grid (Control Type: Matrix Grid)

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.
