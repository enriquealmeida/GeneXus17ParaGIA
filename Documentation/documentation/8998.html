DatePicker property

It consists of a DHTML calendar which allows you to select a date or date/time from a PopUp window (the window that contains the calendar is displayed once you click on an image) or a flat one (it always remains visible in the HTML page).

Values

False
True

Scope

Generators: Java, .NET, .NET Core
Controls: Attribute/Variable (Data Type: Date, DateTime)

Description

Properties

Enable Datepicker This property allows you to enable / disable the DatePicker for all date / datetime values of the KB. Its default value is "Yes".
Show week numbers This property allows you to deploy the corresponding number of each week. Its default value is "No".
First day of week This property is the first day of the week for the calendar. The default is "Sunday".

 

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
