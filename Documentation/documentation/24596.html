RemoveDiacritics method

Returns the text without diacritic characters.

Syntax

&DataType.RemoveDiacritics()

Type returned
Character

Scope

Objects: Procedure, Transaction, Web Panel, SDPanel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3), Android, Apple

Samples

&Text = "áéíóú"   
&TextNoDiacritics = &Text.RemoveDiacritics()   // Result: "aeiou"
if &TextNoDiacritics = 'aeiou'
     //Success evaluation
endif

Availability

This feature is available since GeneXus X Evolution 3 RC (codename: Tilo)

Note: This method is useful in conditions evaluated on client side. On the server side this could be solved by the DBMS, for example SQL Server.
