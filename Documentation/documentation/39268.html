Apple Maps API Key property

API Key for Apple Maps. Depending on the selected API, it may have to be generated from the application's Bundle Identifier.

Scope

Objects: Menu, Panel, Work With (Only Main Objects)
Generators: Apple

Description

This property will be displayed when Apple Maps API property has a different value from its default (Apple Maps) in order to set the access key for displaying maps.

  • If Apple Maps API property has 'Google Maps' value, refer to HowTo: Get a Google Maps API Key for iOS to get its respective key.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 15 Upgrade 11.
