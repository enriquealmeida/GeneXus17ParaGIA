Order property

Establishes the desired order in which the associated query will be returned. It is equivalent to writing the Order clause in the For Each command.

Description

Controls the query order of both grid control types (standard and Free Style Grid control) as well as the Data selector ('Orders' property).

Although the editors to specify the orders are slightly different, the concept behind is the same.Â 

To specify that a descending order is desired for any attribute in the list of attributes, the name of the attribute must be written between brackets. You may include attributes with both ascending and descending orders.

This property does not work for transaction's Grid.

Scope

Objects: Data Selector, Web Panel
Controls: FreeStyle Grid, Grid
