External File Name property

Name of the file when it is extracted from the Knowledge Base to the file system.

Description

It contains the name of the physical file that the file object contains (as it has been uploaded). It cannot be renamed, but if you wish to change the name, you can upload a new file with a different name.Â 

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16 upgrade 5.
