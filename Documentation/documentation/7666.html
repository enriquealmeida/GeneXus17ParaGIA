Codepage property

To associate CodePage to the language.

Description

This property is forÂ  identifying the codepage in which the messages of this language are stored. This is necessary to display and convert them correctly in machines that do not use that codepage by default. If the value is 0, the machine's default codepage will be used. The value of this property should never be changed, except when creating a new Language object.

Scope

Objects: Language
Platforms: Web(.Net, Java)
