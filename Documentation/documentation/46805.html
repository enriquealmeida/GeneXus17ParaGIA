Preferences window renewed in GeneXus 17

The Preferences window has been renewed in GeneXus 17 to better reflect your intentions when defining an Environment. 

Note that:

1) The Preferences tree has been restructured:


2) The "New Environment" dialog has been updated consistently:


As you can see, since GeneXus 17 an Environment has differentiated sections to set:

  • Back end
    • It refers to everything related to the application server, database accesses, and services.  
  • Front end
    • It refers to everything related to the UI (what is shown and used to interact) and connects with the Back end. So, the resources to generate (and set) these are .NET, Java, Angular, Android, Apple. 
       
  • Deployment
    • Refers to the Deployment Unit objects defined the KB Version.
