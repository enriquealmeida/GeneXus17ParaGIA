Base Classpath property

Read-only property including the list of resources/JARS to be used when running the application.

Description

The Base Classpath property includes all the required JAR files to execute the GeneXus java generated application.

Besides, all the JAR files and resources in <Model Target Directory>\services andÂ <Model Target Directory>\modules directories, are included automatically in the Base Classpath property.

At runtime the Base Classpath is preceded by the Classpath property contents.

Notes

The version of the libraries added to the Base Classpath depends on the version of the JDK installed in the developer's machine (the last JDK version registered). So, only the libraries supported by the JDK installed are added to the Base Classpath.

For example, for JDK 1.6 the log4j-api-2.3.jar file is added to the Base Classpath, but for JDK 1.7, the log4j-api-2.11.0.jar library is added.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Platforms: Web(Java)
