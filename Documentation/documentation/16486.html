Business Process Diagram object

Represents the flow of a business process that occurs in real life.

A Business Process Diagram object has several sections that may be defined, for example:

  • Diagram: It allows you to draw the business process. To do it, you can drag and drop the objects (such as web panels) or you can use the Toolbox.
     
  • Relevant Data: Define the data that flows over the process.

As most of the objects, it also has a section for Documentation as well as a Properties Editor, in order to set up general aspects of the object.
