Asc function

Converts the first character of a string into its corresponding ASCII character.

Syntax

ASC(string)

Where:
string
    is the string of regular characters.

Type returned:
Numeric

Scope

Objects: Procedure, Transaction, Web Panel, Work Panel
Generators: .NET, Ruby (up to GeneXus X Evolution 3), Java, Visual FoxPro (up to GeneXus X Evolution 3)

Samples

&AscValue = Asc("Hello") // &AscValue = 72
