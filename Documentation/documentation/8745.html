ReturnOnClick property

Allows indicating the control that will cause a Return within a prompt and will assign the selected value to the foreign key.

Values

False
True

Description

This property applies only at design time.

Variables need to be setting as Read Only in order to get enabled that property.

Values

False: It does not behave as mentioned above. This is the default value.

True: The control triggers the return and the values assignment to the parameters returned by the prompt.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
Controls: Grid, Image, Text Block
