External utilities used by GeneXus generated Native Mobile applications

In Smart Device generation, GeneXus uses external libraries and projects in order to include useful development tools, to improve the application’s visuals and user experience, among others.

Some of these external dependencies are always necessary for the well functionality and compilation of the application, whereas some others are only necessary if the application uses them.

The following documents shows, for each platform, the external dependencies that GeneXus uses for generating Smart Devices applications and whether they are always necessary or not.

  • External utilities used by GeneXus generated Android applications

  • External utilities used by Genexus generated iOS applications

Considerations

Depending on the arquitecture of the built application, GeneXus may generate or not web programs. If web programs are used, then it is recomended to check out the External utilities used by GeneXus generated web applications.
