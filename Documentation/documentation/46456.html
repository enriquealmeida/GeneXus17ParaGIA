Features of Native Mobile App development that Angular generator still misses

This is a list of features that are supported when creating Native mobile apps but are not supported when generating with Angular.

Functionalities of the GeneXus Core module that are not supported

 

  • GeneXus.Common.Configuration.ConfigurationManager
  • GeneXus.Common.Configuration.ExternalStorage
  • GeneXus.Common.UI.Appearance
  • Navigation Split
  • GeneXus.Common.DirectionServices
  • GeneXus.Common.Analytics
  • GeneXus.Common.AnalyticsPurchase
  • GeneXus.Common.Clipboard
  • GeneXus.Common.Maps
  • GeneXus.Common.Messages
  • GeneXus.SD.Media.Audio
  • GeneXus.SD.Media.AudioPlayerCustomAction
  • GeneXus.SD.Media.AudioPlayerSettings
  • GeneXus.SD.Media.AudioRecorder
  • GeneXus.SD.Media.Files
  • GeneXus.SD.Media.ImagesCollection
  • GeneXus.SD.Media.MediaItem
  • GeneXus.SD.Media.MediaItemFinishedInfo
  • GeneXus.SD.Media.MediaQueue
  • GeneXus.SD.Media.MediaQueueState
  • GeneXus.SD.Notifications*
  • GeneXus.SD.Offline
  • GeneXus.SD.Store
  • GeneXus.SD.Synchronization
  • GeneXus.SD.AppLifeCycle
  • GeneXus.SD.ARPreview
  • GeneXus.SD.Beacon*
  • GeneXus.SD.CardInformation
  • GeneXus.SD.CardScanner
  • GeneXus.SD.Contact*
  • GeneXus.SD.DeepLink
  • GeneXus.SD.DeviceAuthentication
  • GeneXus.SD.Printer        
  • GeneXus.SD.ScannedBarcodes        
  • GeneXus.SD.Scanner        
  • GeneXus.SD.Search        
  • GeneXus.SD.SearchScope        
  • GeneXus.SD.WebBrowser

Note that some of them make no sense on Angular (eg.: GeneXus.SD.Offline).

UI Controls that have not been implemented yet

  • Ads View
  • Scanner
  • Facebook Button
  • Advanced Image
  • File
  • Relative Timer
  • Matrix Grid
  • Charts
  • Sparkline
  • Image Annotations
  • Slider

Note: Controls that already are deprecated in Apple and Android won't be supported in Angular either. More information at GeneXus 17 Compatibility Section
