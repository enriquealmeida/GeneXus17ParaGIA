Left scrolling arrow symbol property

Symbol used for the left arrow of Horizontal grids. UTF-8 symbols are supported (For example, using \2190 for a left-pointing arrow).

Scope

Platforms: Web(.Net, .Net Core, Java)
Controls: Free Style Grid

Description

The Left scrolling arrow symbol is a property of the grid Page Controller class.

This class is used for configuring the appearance of a Horizontal Grid (a grid whose Custom Render property is set to Horizontal grid).
