NotVisitedLinkColor Property

To allow indication of the color for the links that have not been visited yet. Any color may be used, but we recommend that the criteria bes kept throughout the whole application in order to avoid confusion for users.

This property only applies at design time.

Scope

Objects: Transaction object, Web Panel object
Controls: Forms
