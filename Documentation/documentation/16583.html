Enable Alpha Indexer Property

The Enable Alpha Indexer property is applied at the Orders level in the List.

It provides a better view of the first character of the word we’re moving through, in order to simplify searches in long lists.

  

Notes:

  • On Android devices, the alpha indexer appears when the Grid have more than 4 pages.
