Length property

Indicates the attribute, variable, domain or relevant data length.

Scope

Level: Attribute, Domain, Variable

Description

If the data type is Numeric you must consider that the whole length includes the decimal places, the decimal point, and the sign.

For example when defining a Numeric(4.1)- variable: valid values are -1.5 but not -10.5.
If the minus sign is not needed you can insert 10.5 and up to 99.9.

Values

See Maximum numeric length property

Default Value: 4
