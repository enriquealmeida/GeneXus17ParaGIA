Date format property (for Date/DateTime attributes/variables)

Sets whether to accept and display two or four digits for the year.

Values

Year with four digits (99/99/9999) Accepts and displays four digits for the year.
None Accepts and displays only the time part (no date).
Year with two digits (99/99/99) Accepts and displays two digits for the year.

Scope

Level: Attribute, Domain, Variable

Description

This property is offered for attributes, variables, or domains based on Date / DateTime data type. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build All.
