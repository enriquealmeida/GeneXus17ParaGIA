ToFormattedString method

Returns a string that results from applying an attribute, variable, SDT picture to its value.

Syntax

 &Var.ToFormattedString()

Type Returned:
Character

Scope

Objects: Attribute/Variable, Structured Data Type
Generators: .NET, Java, Android, Apple

Description

This method returns a string that results from applying the Picture property to its value.

The method may be applied to Numeric, Date, DateTime, Character, and VarChar data types. In the case of Date and DateTime data types, the date and time format returned will also depend on the Date format and Time format properties (they both depend on the language set in the KB).

Samples

If you have a variable &Total(10, 2) with Picture "Z,ZZZ,ZZ9.99" and the value is 4395.35, the &Total.ToFormattedString() function will return the "4,395.35" string.
