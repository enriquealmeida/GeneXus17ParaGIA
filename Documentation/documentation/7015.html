Sender Property

This property sets the sender data that is to be included in the messages sent. 

Syntax

&SMTPSession.Sender

Type Returned:
MailRecipient

Description

When a SMTPSession type variable is created, the Sender property has a MailRecipient object with empty Name and Address properties. Thus, the sender data can be directly assigned in the Name and Address properties of the already created object. Another option is to assign another MailRecipient type object from which these properties can be copied. 
 
It is necessary to have valid data in the Sender property so that the Login method can work.     

Scope

Extended Data Types: SMTPSession
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
