On Click Event property

Sets the event name that will be executed at the moment of clicking on the control.

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Transaction, Web Panel, Panel for Smart Devices
Platforms: Web(.Net, Java), Smart Devices(Android, IOS)
Controls: Button, Image, Text Block
