Notes

  • This domain is more general than Locale domain.
  • Not every language is available for every Provider. In such case, GeneXusAI will raise an GXAI5000 code with the information needed.
  • If some language is not available on the domain definition for your upgrade installation, you can add it at runtime by using FromString Method.
    e.g.: &language.FromString(!"xx"), being xx a valid language code.

Scope

AI Task OCR procedure, ExtractEntities procedure, KeyPhrases procedure, SentimentAnalysis procedure, Translate procedure
Platforms Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This domain is available as of GeneXus 16.
