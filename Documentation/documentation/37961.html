Stripe Charge Object

The Charge object allows charging a customer’s credit or debit card.

Currently, this SDK only supports creating a charge. 

To create a charge use:

  • Create Options: StripeChargeCreateOptions
  • Response: StripeChargeCreateResponse
  • Errors: StripeError
