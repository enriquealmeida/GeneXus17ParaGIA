BeaconProximityAlert Data Type

This SDT is defined in order to specify a Beacon Proximity Alert.

Fields

Name Type Description
BeaconRegion BeaconRegion Identifies the Beacon which will execute the alert.
NotifyOnEntry Boolean Alert will be executed when the user enters the region.
NotifyOnExit Boolean Alert will be executed when the user exits the region.

 

NOTE: In order for the Beacon Proximity Alerts to work you need to have configured the property Location Always Usage Description
