DW transaction property

Specifies whether the Transaction is used for Data Warehouse.

Values

False
True

Scope

Objects: Transaction
Generators: Cobol, Java, .NET, RPG

Description

The default value is False.
If this property is set to True, the DW transaction type property is displayed.

Compatibility

Ruby and Visual FoxPro generators offer this property up to GeneXus X Evolution 3.
