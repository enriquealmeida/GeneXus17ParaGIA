RemoveItem method

Removes one element of the SDT collection.

Syntax

Control.RemoveItem(code)

Where:
Control
    Is the name of a control inserted in the form.

Scope

Objects: Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Description

Deletes the item whose code is indicated by ‘code’ of the control.
Forms.Meta and Forms.MetaEquiv only applies to Web Interface.
