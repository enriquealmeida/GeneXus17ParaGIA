Windows Environment Considerations

GXflow does not support Windows environments.

Note: If you have a Win environment in the same Knowledge Base as the Web environment and it is not the default environment you will have to copy the wapi.dll, located in the bin directory of the Web environment, into the folder bin that is in the model's root directory.
