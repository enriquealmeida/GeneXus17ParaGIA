Picture Properties Group

The Picture group of properties offers a set of properties that allow setting the edition format (mask) for an attribute or variable.
It includes different properties available to set, depending on the data type of the attribute/variable which you are setting.

 Numeric

  • Left fill property
  • Thousand separator property
  • Prefix property
  • Picture property


Character and VarChar

  • Case property
  • Picture property


Date

  • Date format property
  • Picture property


DateTime

  • Date format property
  • Hour format property
  • Picture property

 

Availability

The picture 9999 for character type fields in smart devices, is available as of GeneXus X Evolution 2 Upgrade 3.

 

Note

When using picture 9999 for character type fields in smart devices, when focusing on the field, the numeric keyboard will appear.
