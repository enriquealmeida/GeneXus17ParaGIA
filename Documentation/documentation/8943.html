Compiler flags property

The information of this property will be included in the .rsp used to compile the main. It is useful, for instance, to generate debug information (including the string “/debug”) or to include a dll within the namespace (/r:xxx.dll ).

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Platforms: Web(.Net)
