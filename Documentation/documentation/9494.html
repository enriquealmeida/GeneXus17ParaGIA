Data View Composition

Purpose

Allows modifying the composition of the data view, mapping internal and external attribute names. This is displayed in the list box.

Description

When working with GeneXus, you will only see the internal name and it will never reference the external one. If an internal table is associated with a Data View, the composition may be empty, that is, no attributes are defined. In this case, it is assumed that the composition of the Data View is equivalent to the associated table and that the indices are also equal. This is called dynamic association.

Internal Name

Name of the GeneXus attribute you are mapping. You can write it in the "Internal Name" column.

External Name

Name of the corresponding field of the external file. The use of external names is optional. When the external and internal names match, it is not necessary to specify the external name.
