Profile Property

This property is used to indicate a profile to start the next session.

Syntax

&MAPISession.Profile

Type Returned:
Character

Description

The value of the property will be effective as of the following call to the Login method. If this property is not specified, the call to the Login method will show a window to select profiles. 

Scope

Extended Data Types: MAPISession
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
