Smart Grids

The Smart Grid control is an evolution of the classic Grid control that is used in mobile application development with GeneXus. As from GeneXus 16 Upgrade 11, when creating a new Grid control, the default value of the Control Type property is Smart Grid. 

In addition to the classic functionality of displaying a group of related data, it now provides a set of properties that allow managing how that information will be displayed in the Grid. For instance, it is possible to indicate the scroll direction or the number of items per row that will be displayed.

  • Auto Grow property
  • Scroll Direction property
  • Snap To Grid property
  • Items in Rows and Columns
    • Items Layout Mode property
    • Items Per Column property
    • Items Per Row property
  • Size
    • Maximum Width property
    • Minimum Width property
    • Maximum Height property
    • Minimum Height property
