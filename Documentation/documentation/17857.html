Process Manager

The Process Manager consists of applications aimed at process manager users.

These applications allow managing different processes, tasks, and users participating in the processes, among other things.  

The following figure shows this application interface:

The process manager consists of the following applications:

  • Processes
  • Tasks
  • Business Events
  • Process Definition
  • Calendars
  • Participants
  • Notification templates
