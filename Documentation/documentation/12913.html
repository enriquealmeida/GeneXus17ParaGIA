Options - KB Explorer

In this node you can configure how icon texts will be viewed in the KB Explorer.

Values are:

  • Name. Only the Name will be displayed to right of the icon.
  • Description. Only theÂ Description will be displayed to right of the icon.
  • Name & Description. Both values will be displayed to right of the icon.

The default value is Name.
