Devices property

To specify the target device of the iOS application.

Values

Universal Specify both iPhone & iPad as target device. This is the default value.
iPad Only iPad.
iPhone Only iPhone.

Description

This property indicates wich device your application was made for.

For example, if you have done the design for iPad, but not for iPhone, you will have to set the property on iPad value, then, your application will be available just for iPad.
