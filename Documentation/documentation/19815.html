HowTo: Installing the GXflow Runtime Clients

The GeneXus setup installs the GXflow client only for these environments: Java, .NET, and .NET Core; for the DBMS SQL 2012 or higher.
They can be found at: <GeneXus>\Packages\Gxpm\Platforms.

If you are using a different environment, you will see the following warning message in the output:

warning: There are pending workflow actions but your environment configuration is not supported by your current workflow installation

To use workflow in models configured with other environments you will have to download the corresponding setup:

17
GXflow Clients for GeneXus 17

 

You need internet connection to download associated components for the selected environment.
