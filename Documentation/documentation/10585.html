Size property

Sets the GeneXus connection pool size.

Scope

Available at the Data Store.
Data Store: Iseries, DB2 UDB, INFORMIX, MYSQL, ORACLE, POSTGRESQL, SQLSERVER

Description

The size of the pool is the maximum number of available connections to the database. This property becomes available only when the Unlimited size property is set to "False;" otherwise, it doesn't apply.

The default value is 10.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build any object.
