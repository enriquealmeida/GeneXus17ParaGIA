Use client login as userid('server') property

If you configure this property, the UserId("Server") function will return the same value as the UserId() function.

Values

No You can't use the UserId("Server") function. This is the default value.
Yes You can use the UserId("Server") function.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)
