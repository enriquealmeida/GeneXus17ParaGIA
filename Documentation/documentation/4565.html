Web Maximum Edit Length Property

Sets the maximum length in characters for controls of Edit type. If it is exceeded, the control will show a horizontal scroll.

Scope

Level: Knowledge Base

Description

Defines the maximum length, in characters, that an attribute can have to be defined as a "simple" edit control.

If the length of an attribute exceeds this value but is lower than the Web maximum single line edit length propertyÂ value, the associated control will be defined as having a horizontal scroll bar.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
