Google Cloud JSON Credentials property

Specifies your Service Account credentials (JSON format).

Scope

Objects: Conversational Flows Instance

Description

Allows inserting the generated JSON in the Google Cloud Console. It will then be encrypted and included in the chatbot config. Also, it will be read at runtime to get the access token required to consume the DialogFlow APIs.

Availability

This property is available since GeneXus 17 upgrade 3.
