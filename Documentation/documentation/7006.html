DatastoreName Property

This property indicates the GeneXus Data Stores associated to the variable.

Syntax

&DataType.DatastoreName

Type Returned:
Character

Description

When you have a DBConnection variable, the property DatastoreName returns the name of the GeneXus data store to which the variable is associated. This property is read only, so it cannot be assigned. To associate a data store to a DBConnection variable you have to use the GetDataStore function.

Scope

Extended Data Types: DBConnection
Objects: Procedures
Languages: Java, .NET
Interfaces: Web
