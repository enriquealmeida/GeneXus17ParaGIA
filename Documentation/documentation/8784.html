TitleForeColor property

Determines the foreground color of a Grid column’s title.

Syntax

control. TitleForeColor = value

Where:
value 
    Is a numeric representing an RGB color

Samples

Gridcol1.TitleForeColor = RGB(0 ,255 ,0)

This will set the column’s title forecolor to green.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
Controls: Grid
