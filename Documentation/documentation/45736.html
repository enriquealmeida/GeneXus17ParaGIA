Storage Endpoint property

Entry point to the service. The requests will be directed there.

Scope

Generators: Java, .NET, .NET Core
Level: Generator

Description

The Storage Endpoint property will be available depending on the Storage Provider for Multimedia files used (specified in the Storage Provider property).

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build any object.

Availability

This property is available since GeneXus 16 upgrade 10.
