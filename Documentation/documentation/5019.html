Port Property

Indicates the server's port in the host.

Syntax

&DataType.Port

Type Returned:
Numeric

Description

SMTPSession: The default value is 25, which is the port normally used by SMTP servers.   

POP3Session: The default value is 110, which is the port normally used by POP3 servers.

Location: The default value is 80.

Scope

Extended Data Types: LDAPClient, POP3Session, SMTPSession, Location, HttpClient
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
