Features that are exclusive to Angular development

The following is a list of features that is exclusive to Angular applications development, that is, may not be able when developing Native mobile apps or Web using the traditional approach.

User Controls

A User Control object is supported as a UI control in Panel object and Work With for Smart Devices object but, in GeneXus 17, is only supported in Angular (Not in Apple or Android generators).
You should create a layout that is specific to Angular if you want to use this feature.

More information at Multiple Layouts in Panels and Platforms node in the KB Explorer.

Master Panels

As the Web Master Panel object to the Web Panel object, so the Master Panel object to the Panel object.
In GeneXus 17, this object is only supported in Angular (Not in Apple or Android generators).

Web Platforms

Web Platforms have been added to the Platforms node in the KB Explorer, so you can design layouts that are specific for Angular apps running in browsers, or specific to Desktop screens 

Call Options (Target: Top)

Angular supports the target Top as a CallOptions Target.
