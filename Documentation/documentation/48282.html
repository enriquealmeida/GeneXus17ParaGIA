Title font style property for QueryViewer maps

Specifies the map's title font style.

Values

italic
normal
oblique

Scope

Objects: Design System, Theme
Generators: Java, .NET, .NET Core

Description

The font style property specifies the font style for a text.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Run the main object.

Availability

This property is available since GeneXus 17 upgrade 5.
