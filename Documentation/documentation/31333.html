Cache Password Property

The Cache Password property enables to configure the user password used to connect to the Cache system in a distributed caching environment.

Values

If required, specify the user password (along with the user name) to connect to the Cache system (the cache system should be specified in the Cache Provider property).

Description

It's required if the server supports SASL (e.g., Memcached SASL Authentication protocol).
The Cache Password property is under the Cache Configuration section of the generator's properties.

How to Apply Changes

Build only one object.
