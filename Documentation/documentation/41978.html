SetTheme function

Changes the application Theme at runtime for any other Theme included in the property Additional Styles property. 

Syntax

&NumericVble = SetTheme("ThemeObjectName")

Where:

"ThemeObjectName"
    Name of the Theme to change.

Type returned:
Numeric(1)

Description

This function changes the application Theme at runtime for any other Theme included in the property Additional Styles property. 
It returns 1 when it can change the Theme, or 0 otherwise. 
If the Theme can't be changed, the base Theme is kept. 
The change persists as long as the application session is active.

Samples

Event 'SetTheme'
   &num = SetTheme(!'SimpleAndroid')
Endevent

Scope

Generators: Android, Apple, Angular
