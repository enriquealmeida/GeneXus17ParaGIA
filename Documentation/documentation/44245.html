Predict procedure

Makes a prediction using a custom model based on the input data.

Parameters

  • in:&dataInput:: DataInput, GeneXusAI.Custom
    The input data item.

  • in:&model :: Model, GeneXusAI.Custom
    The model information used to predict.

  • in:&provider :: Provider, GeneXusAI.Configuration
    Provider settings.

  • inout:&Messages :: Messages, GeneXus.Common
    A collection of warning and error messages returned by the task. You should check in your code if an error was returned. Refer to error codes and descriptions for more information.

  • out:&DataOutput:: DataOutput, GeneXusAI.Custom
    Output data item.

Notes

  • If you want to get a set of possible candidates, use a predefined GeneXusAI task instead of the Predict procedure (e.g. use Classify procedure if your model was trained for predicting Labels, or DetectObjects for predicting Regions).
