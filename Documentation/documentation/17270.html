BPD Intermediate Events

An Intermediate Event indicates where something happens or occurs after a process has started and before it has ended.

Description

These events are represented by a grey circle with a double line boundary and its corresponding icon inside.

The events available are as follows:

Intermediate Events
None
 
Timer
 
Signal
 
Conditional
Cancel
 
Error
 
Compensate
 
Link
