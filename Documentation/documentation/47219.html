Label Width property

Establishes the width, in percentages, of the control label for each screen size in a Responsive Web Application. The remaining percentage will be given to the control itself.

Scope

Generators: Java, .NET, .NET Core

Description

The Label Width property is available in the Responsive Sizes dialog box, for all the controls whose Label Position property = Left.

Its default value is 25%.
