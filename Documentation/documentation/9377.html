OS for iSeries version property

Sets the Operating System version of the DB2 Server for which the code is generated.

Values

V5R1
V5R2
V5R3
V5R4 or higher V5R4 or higher (lower than V6R1)
V6R1 to V7R1
V7R1(SF99701-38) or higher

Scope

Available at the Data Store.
Data Store: Iseries

Description

It is highly recommended to use version 6.1 or higher. Further information: SAC 40453
