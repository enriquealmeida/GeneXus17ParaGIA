BrowserId function

Returns a browser’s identifier.

Syntax

BrowserId()

Type Returned:
Numeric

Scope

Objects:   Procedure, Transaction, Web Panel
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Description

The BrowserId function returns a browser identifier:

Id Description
0 Unknown Agent
1 Internet Explorer
2 Netscape
3 Opera
6 Mozilla Firefox
7 Chrome
8 Safari
9 Edge

 

Note: The result makes sense only if the object was called directly or indirectly by a Web Panel, or if executing in a Web environment.
