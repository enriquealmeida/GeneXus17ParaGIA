ToDate method

Returns a Date data type corresponding to a given DateTime data type.

Syntax

&datetime.ToDate()

Where:
&datetime
   Is the DateTime data type to be converted.

Type Returned:
Date

Scope

Objects: Procedure, Transaction, Web Panel
Generators: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Cobol

Samples

&Date = &DateTime.ToDate()
