ValueText Property

The value of the cell in ExcelCellRange data type of type text.

Syntax

&ExcelCellRange.ValueText

Type returned

Numeric (ExcelCellindex domain)

Description

Sets a value of type character in the cell and gets the value of the cell if it is a character

Scope
