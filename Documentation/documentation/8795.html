WidthUnit property

WidthUnit is a run-time property specifying the unit of measure that the value of Width Property is in.

Description

Values

"px"

Pixels.

This is the default value.
"%" Percentage

Run-time/Design-time

This property applies only at run-time.

Samples

BLData.Width = 100
BLData.WidthUnit = "%"

Scope

Controls: Embedded Page
