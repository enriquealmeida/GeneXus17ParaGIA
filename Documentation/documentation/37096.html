Configuring GeneXus for using the Chatbot Generator

To build a chatbot using the GeneXus Chatbot generator, create a Conversational Flows object and configure at least, the following properties:

NLP Provider property
Watson
Authentication Type property 
API Key property
Region property
User Name property
User Password property
Workspace Id property
DialogFlow
Dialogflow Agent property
Google Cloud Project property
Client Id property
Client Secret property
