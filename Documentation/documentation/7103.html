Basic and Digest Properties

Constants that determine an authentication type.

Syntax

&DataType.Basic
&DataType.Digest
 
Type Returned:
Numeric

Description

These constants determine an authentication type. They are used in the AddAuthentication method.
Basic=0 : To authenticate: the user and password are sent with no encryption. 
Digest=1: To authenticate: the user and password are sent encrypted.

Scope

Extended Data Types: HttpClient
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
