Apple Launch Screen property

Apple custom launch screen interface file (.storyboard). It is automatically generated if empty.

Description

Warning: As of April 2020, all applications and application updates submitted to the Apple Store will be required to have a Launch Screen. More information at Apple Requirements.

It allows attaching a .storyboard, which will be shown when the application is started and then will be replaced by the application once it has loaded all its startup data. 

To learn more, read: Apple - Launch Screen

If the property is empty (default value), a recommended.storyboard will be automatically created. 

If you want to create your own .storyboard is recommended follow this documentation: 
Apple - Understanding Auto Layout

Note: It replaces the previous Launch image property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Objects: Menu for Smart Devices, Panel for Smart Devices
Platforms: Smart Devices(IOS)
