Synchronize with External Wiki property

Controls whether or not the documentation will be synchronized with an External Wiki.

Values

False
True

Scope

Level: Environment

Description

This property is available in all the documents and documentation that are part of every object, to control if the documentation will be synchronized with an External Wiki.

The default value for all documents can be set from the corresponding property in the Preferences section at Environment level.

If most documents in the KB are to be synchronized with an External Wiki, set this property value to Yes. Next, at the documentation level, set any exceptions you want.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

No action is required to apply the corresponding changes when the property value is configured.
