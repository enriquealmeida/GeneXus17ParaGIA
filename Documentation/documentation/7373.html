Date data type

Stores Date values.

The following table illustrates the conversion done by GeneXus according to the DBMS:
 

DBMS Conversion
Oracle DATE
DB2 Universal Database DATE
Informix DATE
SQL Server DATETIME
Access DBDate
DB2 for iSeries CHAR(8) or DATE1
DBF DATE(8,0)
PostgreSQL DATE

1 Depends on the Date data type definition environment Property.

   msg("The date has expired")
EndIf
