Maps - GeographySaved Event

Launched when a geography finishes editing, which can happen in two ways:

-  the SaveEdition() method is called, or
-  EditableGeographies property value changes, and therefore the next edition corresponds to another type of figure.

Parameters 

&Geography Geography It is the Geography object resulting from the edition.
&GeographyId Varchar It is an autogenerated GUID that identifies the drawn geography.


Syntax

MapGrid.GeographySaved(Geography , VarcChar)


Example 

Event MapGrid.GeographySaved(&Geography , &GeographyId)
    composite
        &GeographySDT = new()
        &GeographySDT.Id = &GeographyId
        &GeographySDT.Feature = &Geography
        &Geographies.Add(&GeographySDT)
    endcomposite
Endevent

Availability

This method is available since GeneXus 17.
