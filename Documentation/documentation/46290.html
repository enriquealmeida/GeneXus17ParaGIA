CurrentWorksheet property

Returns the worksheet that ExcelSpreadSheet data type is currently working on.

Syntax

&ExcelSpreadSheet.CurrentWorksheet

Type returned

ExcelWorksheet data type

Description

This property is read only, it cannot be assigned.

Scope

ExcelSpreadSheet data type

Availability

This property is available since GeneXus 16 Upgrade 11
