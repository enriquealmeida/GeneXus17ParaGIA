Host Property

This property is used to indicate the server address. 

Syntax

&DataType.Host

Type Returned:
Character

Description

Both the name of the server and its IP address can be specified.
Location: The default value is “localhost”.

Scope

Extended Data Types: HttpClient, Location, LDAPClient, POP3Session, SMTPSession
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
