Master Page property

Assigns one of the Master Pages defined in the Knowledge Base to a Transaction or Web Panel.

Description

Every Transaction object and every Web Panel object offers this property, so that you can assign a Master Page to it.

If you do not assign an explicit Master Page to this property, it inherits the Master Page set in the Default Master Page property (offered at version level).

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Transaction, Web Panel
Platforms: Web(.Net, Java)
