Web Theme object

Defines graphic and functional design of controls for Web applications (equivalent to Theme object but only for Web).

Description

The Web Theme object helps you to define the look and feel of your Web application.

Every Knowledge Base contains a Web ThemeÂ object named 'Carmine' automatically created and it is used by default when generating Web applications.

The 'Carmine' Theme is referenced in the Default Theme property at version level.

Besides, every Web Panel object and Transaction object contains the Theme property (which by default inherits the Theme configured in the Default Theme property).

Of course, you are able to change the values of all these properties by other Web Themes defined in your KB.
