Failed Files property

A FileUploadData SDT collection containing the failed uploads.

Scope

Generators: Java, .NET, .NET Core
Controls: FileUpload

Description

A FileUploadData SDT collection containing the failed uploads.

The files that could possibly fail are those that exceed the maximum size allowed by the server or that have an invalid extension for the server.

Run-time/Design-time

This property applies both at run-time and at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Build with this Only of the object.

Availability

This property is available since GeneXus 17 upgrade 1.
