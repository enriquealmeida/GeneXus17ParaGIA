SetObjectTheme Property

It allows assigning the theme defined in the "Theme" property.

Values

False The value of the property "Theme" will not be taken into account. This is the default value.
True The value of the property "Theme" will be taken into account.

 

Scope

Objects Web Panel object (in WW Configuration Preferences)
Interfaces Web
