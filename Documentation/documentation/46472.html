MaxZoom property

Indicates how much an image can be enlarged.

Scope

Generators: Android, Apple
Controls: Attribute/Variable (Control Type: Advanced Image)

Description

It allows indicating with a percentage the maximum zoom that can be applied to an image. 

If the value indicated is lower than 100, the image size will be reduced. 

Run-time/Design-time

This property applies only at design-time.

Samples

Max Zoom 125 Max Zoom 250

 

How to apply changes

To apply the corresponding changes when the property value is configured, Build a main object.
