Generate Huawei property

Enables the Android generator to generate the code corresponding to the Huawei platform.

Values

False
True

Scope

Generators: Android
Level: Front end

Run-time/Design-time

This property applies only at design-time.

Samples

When enabled, a new APK file will be generated specifically for the Huawei Platform. All the files that need to be generated will be saved under the TargetEnvironment\mobile\huawei folder.

How to apply changes

To apply the corresponding changes when the property value is configured, Build the Main Object.

Availability

This property is available since GeneXus 17 upgrade 2.
