Custom Chatbot Definition property

Selects a custom Data Provider for the Chatbot Definition.

Scope

Objects: Conversational Flows Instance

Description

The Custom Chatbot Definition property is available for Scripted Chatbots; that is, when the NLP Provider property is set to None.

It allows selecting a Data Provider (its Output property must be ChatbotStruct). The purpose of this property is to allow changing at runtime the Conversational Flows object to be used (mostly for multi-company applications).

For more information, see Dynamic scripted chatbots.

Availability

This property is available since GeneXus 16 upgrade 10.
