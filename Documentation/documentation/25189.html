GetCurrentProximityAlert method

Gets the information related to the ProximityAlert which triggered the action.

Type Returned: 
GeolocationProximityAlerts
    Collection of GeolocationProximityAlert SDT.

Description

If a Proximity Alert has an action defined, an event is executed when the alert is triggered. In that event, some of the information used to set that alert could be needed. 

This method only works when called from an action triggered by a ProximityAlert, returning an SDT with the information of the alert which triggered the event.
