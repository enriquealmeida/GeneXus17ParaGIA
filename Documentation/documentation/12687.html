Age Method

Calculates the difference in years between two date expressions.

Syntax

date-expression1 | datetime-expression1 .Age([ ,date-expression2 | datetime-expression2 ])

Type Returned:
Numeric N(4)

Description

Returns the difference in years between the two parameters (second parameter - first parameter). If the second parameter, which is optional, is omitted the default value depends on the first parameter’s data type:

1.  if it is Date type then the default value is the value returned by the function Today().
2.  if it is Datetime type then the default value is the value returned by the function Now().

If the first parameter is smaller than the second parameter then Age function returns a positive value, on the contrary it returns a negative value.

Example

&Nbr = &Birthday.Age(&Today)
&TxtAge = "Years old: " + Str(&Nbr)

Scope

Objects: Procedures, Transactions, Web Panels
Languages: .NET, Ruby (up to GeneXus X Evolution 3), Java, RPG, Visual FoxPro (up to GeneXus X Evolution 3), Cobol
Interfaces: Web
