QueryViewer SelectionColor property

Sets a color for the selection in the QueryViewer control.

Description

This property allows the user to pick a different color for the selection in the QueryViewer control. It's available in the related Theme object in the QueryViewer class, under the Chart specific, Pivot table specific and Table specific categories.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 16 upgrade 4.

Scope

Objects: Theme
Platforms: Web(.Net, .Net Core, Java)
