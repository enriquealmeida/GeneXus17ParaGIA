Scroll Direction property

This attribute determines which direction will use the scroll of a Smart Grid control.

Values

Horizontal The scroll will be performed horizontally
Vertical The scroll will be performed vertically

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Platforms: Smart Devices(Android, IOS)
Controls: Grid
