Page Name property

Shows the corresponding document name in an external wiki.

Description

The Knowledge Base documentation can be synchronized with an external wiki (that documentation is defined in the Document object or any object Documentation tab). GeneXus generates a wiki page for every object with documentation.

Because you can have several objects with the same name, the name of the wiki page is generated using an expression for the object type and its name. This name is not customizable but is shown with this property.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17.
