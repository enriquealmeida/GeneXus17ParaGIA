Selectable for ad-hoc Property

It allows setting up whether the task appears in the successor tasks selection dialog shown on completing an ad-hoc task.

Values

Default True
True The task appears in the successor tasks selection
False The task not appears in the successor tasks selection

 

Scope

Applies to Business Process Diagram,BPD Tasks,BPD Subprocesses
