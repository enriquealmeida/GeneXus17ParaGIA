Identifying the Active Development Version

GeneXus provides several ways to easily identify what the active development version is at any given time.

  • Its name is shown in the Version node of the Preferences windows
  • It is displayed in bold in the Knowledge Base Versions tool window, with a bubble next to it that reads: "You are working here"
