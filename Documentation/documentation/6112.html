Section Control

This control is aimed at establishing areas (HTML div tags) in Web Forms, which may contain other GeneXus controls.

The <div> tag defines a division or a section in an HTML document. The purpose of the <div> tag is to group block-elements to format them with CSS.

To add a Section control to the selected location, drag into the Web Form the corresponding icon from the toolbox.

Note:

For the  Web Abstract Editor the control is available as from GeneXus X Evolution 3 Upgrade 6.

For previous versions, you should use a Responsive Table in its place.
