Password property

Indicates the password that must be used in order to be authenticated in the server.

Syntax

control. Password

&DataType.Password

Scope

Generators: .NET, Java
Level: Variable

Description

This property is offered for variables based on the LDAPClient, POP3Session Data Type, SMTPSession Extended Data Types.

For SMTPSession this property will be ignored if the Authentication Property is set to 0.
