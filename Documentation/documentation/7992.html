Footer on last page property

To indicate whether user defined footers are printed or not, on the last page of a report.

Values

No The footer is not printed on the last page.
Yes The defined footer will be printed on every page, including the last one. This is the default value.

Scope

Objects: Procedure
Platforms: Web(.Net, Java)
