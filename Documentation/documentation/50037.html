How to deploy an app generated by Angular Generator (GeneXus 17 upgrade 8 or prior)

An application generated for Angular has the following main parts

  • Front-end, to be deployed to an HTTP Server
  • REST Services, to be deployed to an Application Server
  • Database

Deploying the Front-end

After having built and tested your application,  open a command-line prompt, change directory to <KB Model Directory>\mobile\Angular\<main> (e.g.:LightCRM\CSharpModel\mobile\Angular\LightCRM) and run:

ng build --prod

Then copy the result to any HTTP server, for example, an IIS

If you require to change the default favicon and appicon, you must provide your own png images by replacing the default provided images located in the folder src\assets\icons.

Deploying the Services

Create a Deployment Unit object with your main object and deploy the services like described in Application Deployment tool.
