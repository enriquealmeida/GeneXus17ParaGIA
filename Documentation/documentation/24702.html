New Environment dialog

Creates a new Environment in the Knowledge Base.

Description

Through this dialog a new Environment can be created in your Knowledge Base by specifying the basic information needed for that purpose.

  • Runtime Environment
    • Local
    • On the Cloud
  • Back end
    • Language
    • Data Store
  • Front end
    • Generator

Through the Advanced tab, additional information can be configured.
