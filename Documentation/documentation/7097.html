GetResponse method

Returns the HTML code that is returned by the Web object specified in the object property.

Syntax

&WebWrapper.GetResponse()

Scope

Extended Data Types: WebWrapper    
Generators: .NET, Java, Ruby (up to GeneXus X Evolution 3)

Description

The GetResponse Method must always be assigned to Mail property HTMLText.
