Win maximum multi line edit lines property

Sets the maximum number of lines for multi-line controls of Edit type.

Scope

Level: Knowledge Base

Description

This property defines the maximum height, in lines, the default control should have.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, Build any object.
