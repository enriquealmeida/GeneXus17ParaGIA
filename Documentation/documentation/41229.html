Collection Name property

Names the collection returned by the Data Provider (this property is offered if you have set the Collection property of the Data Provider to True; therefore, a name for the collection to be returned is required).

Scope

Objects: Data Provider
Generators: .NET, Android, Apple, Java

Description

If you define a Data Provider object and you set its Collection property to True, the Collection Name property is automatically added in order to give a name to the collection to be returned by the Data Provider.

A default name is set for the property and you can edit it.

Run-time/Design-time

This property applies only at design-time.
