Chatbot Messages Condition property

This message will be sent if the condition is true.

Description

Condition to be evaluated in order to display a message. It's available for any Chatbot Message of the Responses node of a Conversational Flows object.

The condition can depend on any context parameter.

You can use constant values also.

The condition as you specify it, is used in GeneXus code and is translated to the code of the Provider to be evaluated by the AI Provider itself.

Availability

This property is available since GeneXus 15 upgrade 12.

Scope

Objects: Conversational Flows
