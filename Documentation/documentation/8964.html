Help files base URL property

Indicates the directory where the Help (HTM) files are located.

Description

In this directory, which can be indicated as absolute or relative, we must include the help files, generated in HTML format, corresponding to each object (the help is not taken into account at variables/attributes level).

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)
