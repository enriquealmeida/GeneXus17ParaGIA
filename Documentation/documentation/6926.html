MailRecipient Data Type

It unifies interaction functions with message sending and reception for the different languages generated. 

An important advantage of this implementation compared to previous ones is that this one allows the use of more than one connection event to the mail server.   

Description

It allows you to configure the name and address of the recipient of a message.
