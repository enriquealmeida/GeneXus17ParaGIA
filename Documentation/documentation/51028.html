Err variable

Used to assign an error code returned by a function or a method.

Data Type:
Character(3)

Description

You can use this standard variable to assign to it the result returned by a function or a method.

Samples

The Login method returns an error code, so it is possible to call it as follows:

&Err = &DataType.Login()

The Delete method returns an error code, so it is possible to call it as follows:

&Err = &DataType.Delete()
