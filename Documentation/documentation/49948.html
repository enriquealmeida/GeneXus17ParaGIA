Which objects can be defined in a module? (GeneXus 17 Upgrade 7)

Most, but not all objects can be defined in a Module object. Below is a table that shows the objects or concepts that do not belong to modules.

  • Language object
  • Image object
  • Theme object
  • Web Theme object
  • Color Palette object
  • File object
  • Attribute
  • Deployment Unit object

Note

  • The module of a Table is defined by the Transaction object that it defines
