Dimensions property

To allow defining scalar, one or two dimension type domains or variables. Depending on the selection the Rows and Columns properties options will be enabled.

Values

Scalar Specifies a scalar domain or variable. This is the default value.
Vector Defines a domain or variable of one dimension.
Matrix Defines a domain or variable of two dimensions.

Description

Note

All elements of an array must have the same type definition.

Data types

Character data type, Date data type, DateTime data type, LongVarChar data type, Numeric data type, VarChar data type
