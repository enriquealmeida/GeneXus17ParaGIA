How To: Maps - Mapbox

Mapbox is a leading open-source mapping platform worldwide; it offers APIs to access its UI, services, and functionalities.

This provider has been added to those already available for mobile application generators in order to expand their offering and make it easier for customers who already work on this platform to adopt it.

Scope

Generators: Android


Implementation

To use this provider, in your Main Panel object set:

    Android Maps API property = Mapbox API

Two properties will be enabled that need to be configured:
          Android Maps API Key property 
          Mapbox API Private Key property

Configure, respectively, a public and a secret token that can be obtained at Mapbox Access Token

The implemented functionalities are as follows:

  • Show my location    
  • Location Attribute property 
  • DrawGeography    
  • Editable geographies  
  • Directions Layer    
         
  • Geometry Layer   Full functionality as with the other providers
          
  • MapsOffline External Object


Availability

Since GeneXus 17 upgrade 4.
