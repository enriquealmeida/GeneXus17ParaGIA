Enabling TimeZone Support

TimeZone Support can be enabled or disabled for an application generated with GeneXus.

  • It is enabled when the property DateTime storage timezone property doesn’t have the "Undefined" value.
  • It is disabled when the property DateTime storage timezone property is set to "Undefined".
