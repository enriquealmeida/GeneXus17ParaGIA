Custom Center Attribute property

Attribute used to indicate the location (coordinates) used to center the map.

Description

Allows indicating an Attribute/Variable to store the value of the location used to center the map. This Attribute/Variable has to be of Geolocation domain or Geography data type. 

The property is used only when Center property is set to Custom.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(Android, IOS)
Controls: Grid (Control Type: SD Maps)
