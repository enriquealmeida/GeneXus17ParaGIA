Remember Layout property

Sometimes end users make significant changes to the tables (especially the Pivot table), and they may find it useful to keep those changes for the following session, so that they don’t have to make them again. This property allows keeping the latest table status.

This property keeps the status for the same user on the same browser.

Values

True The latest table status will be saved. This is the default value.
False The latest table status will not be saved.

 

Scope

Output type    Table, PivotTable
