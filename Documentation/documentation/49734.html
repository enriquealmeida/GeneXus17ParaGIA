Deploy Command property

Sets the command used to deploy the Angular application.

Scope

Generators: Angular
Level: Generator

Description

The default value for this property is "npm run deploy."Â It is a command that executes the generated script to run the deployment process. That script depends on the "Run Target property" value.

Run-time/Design-time

This property applies only at design-time.

Availability

This property is available since GeneXus 17 Upgrade 5.
