HowTo: Taking up the full height of the page

Suppose that we have the following design for a web page in a Responsive Web Application, where the left component should take the full height of the page:

In our example, it's a Work With object that displays the filter panel on the left-hand side of the screen, which we want to take the full height of the page as the picture shows:

This Responsive Table that contains the filter (&CountryName) is associated with the AdvancedContainer class.

The solution is to use Viewport units. So, assign the value 100vh to the height property of the AdvancedContainer class, which implies taking the full height of the page.
