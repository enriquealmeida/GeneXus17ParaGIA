Calendar external object

The Calendar external object allows the end user to select a date to input into the application by using a calendar.

Properties

It does not have any.

Methods

Schedule method

Allows you to schedule some task on the end user's calendar. Every parameter is optional except Title and StartDate.

Return value  None
Parameters Title:Character(30), StartDate:Date, EndDate:Date, StartTime:Time, EndTime::Time, Place:Character(30)

Events

It does not have any.

Scope

Platforms  Smart Devices (iOS, Android)
