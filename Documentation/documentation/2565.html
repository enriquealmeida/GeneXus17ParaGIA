Extended Data Types Read-Only property

Opens the next document as read-only.

Syntax

&DataType.ReadOnly

Type Returned:
Numeric

Where:
DataType
ExcelDocument type or WordDocument type variable name

Values

1 The next calls to the Open method will cause the documents to be opened as read-only.
0 There will be an attempt to open the documents as reading/writing, but if they are already opened they will be available solely as read-only. This is the default value.

Scope

Extended Data Types ExcelDocument data type, WordDocument Data Type
Languages .NET, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
