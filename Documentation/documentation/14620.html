HeaderRawHTML Property

Allows writing HTML code in the header of the generated HTML document for the object form.

Syntax

Form.HeaderRawHTML = ...

Description

The value asigned to this property will be placed exactly like it was written inside the <head> tag of the generated HTML document for the object.

Scope

Objects: Transactions, Web Panels
Controls: Forms
Interfaces: Web
