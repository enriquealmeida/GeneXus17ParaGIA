Selection Layer property (GeneXus 17 Upgrade 5)

Enables the possibility to navigate the map and select a location point using the map center in addition to executing an event.

Values

False
True

Description

When Selection Layer is enabled, the map allowsÂ the user to navigate and select a location point using the map center. In addition, an event is executed when the point is selected.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Availability

This property is available since GeneXus 16 upgrade 3.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices
Platforms: Smart Devices(IOS)
Controls: Grid (Control Type: SD Maps)
