DayOfWeek method

Returns the number (1...7) of the day of the week. This number is associated to a day of the week. (Sunday = 1). The value 0 is returned if the received parameter is null.

Syntax

date-expression | datetime-expression.DayOfWeek()

Type returned:
Numeric N(1)

Scope

Objects: Procedure, Transaction, Web Panel, Work Panel
