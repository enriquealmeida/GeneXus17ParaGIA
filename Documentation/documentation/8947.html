.Net Application namespace property

Determines the application namespace where the program is generated and compiled.

Description

The programs generated by GeneXus and compiled with .NET are available under the namespace indicated by this property. It is useful for advanced users who want to carry out some kind of deployment in the GAC (Global Assembly Cache).

The default value for this property is GeneXus.Programs

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net)
