Check For New Data After Minutes Elapsed property

Indicates the waiting time to check with the server if the cached data has changed (in minutes).

Description

Suppose you choose 5 minutes (the default value). If the data for this object is cached, the system uses the cached data and does not check with the server for new data until 5 minutes from the last query have elapsed. After this, the regular procedure is used. That is, the cached data is shown but if this data has changed the query is performed.

Scope

Objects: Panel for Smart Devices, Work With for Smart Devices, Menu for Smart Devices
Platforms: Smart Devices(Android, IOS)
