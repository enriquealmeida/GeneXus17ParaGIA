External utilities used by GeneXus generated applications

Everytime GeneXus builds an application, it uses external utilities in order to include development tools, to improve the application’s visuals and user experience, among others.

To learn about the external utilities used by Web applications check out the External utilities used by GeneXus generated web applications.
