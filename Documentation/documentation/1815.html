J2EE

Acronym of Java 2 Enterprise Edition.

 J2EE is a platform-independent, Java-centric environment from Sun for developing, building and deploying Web-based enterprise applications online. The J2EE platform consists of a set of services, APIs, and protocolos that provide the functionality for developing multitiered, Web-based applications
