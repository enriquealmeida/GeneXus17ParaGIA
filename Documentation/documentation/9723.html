Find option

Allows you to search for a string in source code.

Values

Find what String to be searched for.
Match case Indicates if the search must be case-sensitive or not.
Match whole word Indicates that the search must be exactly.
Search hidden text Forces the search to check within hidden text, such as collapsed code.
Search In selection The search will only take into account the selected code.
Mark with bookmarks For each matching string, adds a line bookmark.
Use Ckeck this option to enable the related combobox:
  • Use Regular expressions: enables a Regular Expression search.
  • Wildcards: The search includes wildcards. For example, the pattern E* will find the strings Event and EndEvent, the pattern E?? will find at least the End string.
Direction Indicates if the search must be performed from the current position to the beginning (Up), or to the end (Down).
Find Next Performs next search.
Mark All Marks all items matching the pattern.
Close Closes the dialog.
