History Management property

True to enable automatic history management. When enabled, every time a user clicks on a tab, a new entry is added to the browser History. When the user presses the browser's back button, the previous tab is activated, and the TabChange event is triggered.

Values

False
True

Scope

Generators: Java, .NET, .NET Core
Controls: Tab
