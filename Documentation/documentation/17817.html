Native Mobile Main object properties

Menu object, Panel object or Work With for Smart Devices object instances have several properties when they are set as main objects. These properties will apply to the whole application represented by the main object.

Description

When you set to True the Main program property of a Native Mobile object, the Main object properties section is displayed with the following properties.

  • Main Object Properties
    • Application title property
    • Default Layout Orientation property
    • Ads Provider property(GeneXus 16 upgrade 10 and lower)
    • Enable Ads property (6)
    • Deep Link Base URL property (3)
    • Google Cast Receiver Application Id property
    • Obfuscate Application property
    • Enable Test Mode property
    • Enable Notifications property
    • Enable Analytics property
    • Enable Logging property
    • Offline Database property
    • Android
      • Android Version Code property
      • Android Version Name property
      • Update URL property (1)
      • Android Package Name property
      • Android Application Icon property
      • Android Notification Icon property
      • Android Portrait Launch Image property
      • Android Landscape Launch Image property
      • Base Color Scheme property
      • Android Maps API property (2)
      • Android Google Services API Key property (5)
      • Android Maps API Key property (2) -- Visible only if Android Maps API Key is not Google Maps.
      • In App Billing Public Key property (2)
      • Use Proximity Alerts property (2)
      • Include Network Id in Client Information property (2)
      • Compilation Mode property (2)
      • Multidex Build property (2)
      • Application Signing( 2)
        • Key Store File property
        • Key Alias property
        • Key password property
        • Store password property
    • iOS
      • iOS Version Code property
      • iOS Version Name property
      • App Store URL property
      • Apple Bundle Identifier property
      • Development Team ID property
      • Devices property
      • iOS Application Icon property
      • iOS Launch Image property
      • Empty Target Background Image property
      • Permissions
        • Application Queries Schemes property
        • Background Modes property
        • Purpose Strings
          • Apple Music Usage Description property
          • Bluetooth Peripherical Usage Description property
          • Calendars Usage Description property
          • Camera Usage Description property
          • Contacts Usage Description property
          • Health Share Usage Description property
          • Health Update Usage Description property
          • Home Kit Usage Description property
          • Location Always Usage Description property
          • Location When In Use Usage Description property
          • Microphone Usage Description property
          • Motion Usage Description property
          • Photo Library Usage Description property
          • Reminders Usage Description property
          • Siri Usage Description property
          • Speech Recognition Usage Description property
        • App Transport Security
          • Allows Arbitrary Loads property
          • Allows Arbitrary Loads In Media property
          • Allows Arbitrary Loads In Web Content property
          • Allows Local Networking property
          • Exception Domains property
  • Logging
    • Default Log Level property
    • Offline Synchronization Log Level property
    • Http Connections Log Level property
    • Offline Data Base Access Log Level property
  • Security
    • Allow Webview execute javascript property (5)
    • Allow Webview execute local files property (5)
  • Facebook (4)
    • Facebook App Id property
    • Facebook Display Name property
    • Facebook App URL Suffix property
  • Twitter (4)
    • Twitter Consumer Key property
    • Twitter Consumer Secret property
  • WeChatPay
    • Application Id property (5)

Scope

Objects: Panel object , Work With for Smart Devices object, Menu object
Platforms: Smart Devices(Android,iOS)

Availability

This set of properties is available as of GeneXus X Evolution 2.

(1) Name changed from "Google Play URL" to "Update URL" as of GeneXus 15 Upgrade 4.
(2) Available at Main object level since GeneXus 15 Upgrade 5.
(3) Available as of GeneXus 15 Upgrade 6
(4) Grouped as of GeneXus 15 Upgrade 7 whose properties were available in previous versions at Main Object properties level.
(5) Available as of GeneXus 15 Upgrade 8.
(6) Deprecated as of Genexus 15 Upgrade 10. Use Ads View control.
