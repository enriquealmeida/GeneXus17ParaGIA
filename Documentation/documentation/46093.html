SetProtected method

Enables password protection to the worksheet content

Syntax

&ExcelWorksheet.SetProtected(password)

Where:
password
     Specifies the password for protecting the content of the worksheet

Type Returned:
Boolean

Scope

Extended Data Types: ExcelWorksheet data type
Generators: Java

Description

It sets the worksheet as protected, so it can not be modified unless the user inputs the value defined in password.
