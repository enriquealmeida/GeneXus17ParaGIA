Extract for Java Generator property

Indicates if the file must be extracted for Java Generator.

Values

False
True

Description

When this property is enabled (True value), it will display Java Generator Extraction Directory property for indicating the path where the file will be extracted (by default, the value of Target Path property followed by "/web" subdirectory).

Notes

  • This property replaces Extract property when it is different from "No" value and Generator property has "Any" or "Java" value.
  • File is copied only if it's newer than the existing one.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Availability

This property is available since GeneXus 15 Upgrade 11.

Scope

Objects: File
Platforms: Web(Java)
