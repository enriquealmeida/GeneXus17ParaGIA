Modules - Object names

One important feature of Module objects is that they act as "qualifiers" of the value of the Name property of the objects in them. That is, the value of the Name property of an object must be unique among objects within the Module, but it can be reapeted in another module as shown in the image below. Therefore the Qualified Name property must be unique for every object on the Knowledge Base.
