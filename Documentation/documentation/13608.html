Workflow Generator property

To allow indication of the platform to be used by the Workflow client. The default value of this property is that of the environment (set by default).

This property is useful for using the application and GXflow with different generators.  For example, when you want to use RPG as a reorganization generator and .Net or Java for the Workflow client.

How to apply changes

Rebuild all objects.
