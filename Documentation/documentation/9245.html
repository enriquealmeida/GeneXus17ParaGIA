Graphic report output property

Controls the manner in which the graphic reports of the model will be generated.

Values

Report Viewer the printing is made using the Report Viewer utility (the rbuilder.dll and gxdib32.dll are used). The resulting report is a .gxr file. The JVM for 32 bits is required on the machine where the report runs (further information: sac 20155). Default value.
Text the printing is text type, obtaining an output file with .prn extension. The only valid value for Unix platforms.

Scope

Objects: Procedure
Platforms: Web(Java)
