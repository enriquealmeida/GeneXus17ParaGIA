Maps - MarkerDragStarted Event

It is launched when a geoPoint is start to drawn.

Parameters

GeographyId Character Identifier (Guid) of the marker which is drag


Syntax

MapGrid.MarkerDragStarted(Character)


Example 

Event MapGrid.MarkerDragStarted(&character)
       msg("GeographyId:" + &character)
Endevent


Availability

This method is available since GeneXus 17.
