Main Platform property

When more than one platform is selected (i.e. Android, Huawei or iOS), it is necessary to set one of them to be considered upon pressing F5 for executing the application.

Values

Android This value is available when the Generate Android property is set to True.
Huawei This value is available when the Huawei property is set to True.
iOS This value is available when the Generate iOS property is set to True.

Scope

Generators: Android, Apple
Level: Front end

Description

When more than one platform is selected, it is necessary to set one of them to be considered upon pressing F5 for executing the application. Nevertheless, pressing F5 will generate everything necessary for both platforms.

When you have only one platform, it is assumed by the property by default.

Run-time/Design-time

This property applies only at design-time.
