Searchable property

The purpose is to allow the end user to find specific information from different sources.

Values

False Business components are not made searchable. This is the default value.
True All Business components of the Knowledge Base are made searchable.
Use Environment property value

Description

If the Searchable property is set to True, all Business Component of the Knowledge Base are made searchable. You can change it locally for certain Business Components if you want.

How to apply changes

To apply changes made by this property, do a Build with this Only of the object.

Scope

Platforms: Web(.Net, Java)
