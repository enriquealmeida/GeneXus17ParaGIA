Deploy Server URL property

To easily deploy the application to the specified web server.

Description

It is available if Deploy to cloud property is set to Yes.

GeneXus will deploy the application to the specified server in build time on each run.

Default value

Depends on generator and GeneXus version. More information:Servers available for Cloud prototyping 

The webserver specified in this URL has to follow some rules, see Cloud prototyping.

Note: to use HTTPS instead of HTTP just change the protocol HTTP to HTTPS (e.g. https://apps3.genexusx.com) and the Protocol specification property to HTTPS too.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Platforms: Web(.Net, Java)
