Server property

Enables generators to know which INFORMIX server they must connect to.

Values

On Line Indicates whether the Informix server is Informix on Line.
Standard Engine Indicates whether this Informix server is an Informix Standard Engine.

Scope

Available at the Data Store.
Data Store: INFORMIX
Generators: Java, .NET, .NET Core
Level: Data Store

Description

The Server property is available when the Access technology to set property is set to JDBC.

The default value of this property is = On Line.

How to apply changes

To apply the corresponding changes when the property value is configured, Build any object.

Compatibility

The Ruby generator offers this property up to GeneXus X Evolution 3.
