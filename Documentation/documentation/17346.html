BPD Events


An event is something that happens during the execution of a process, and they can catch or throw processes. For example, a process could be modeled to start when a certain condition is met or when a timer notifies that a certain time period has elapsed. An event is represented by a circle and the border style shows the type of event.

There are three types of events:

  • Start Events: Allows to start a process flow.
  • Intermediate Events: Happens during the course of a process flow.
  • End Events: Ends a process flow.
