User Properties

Some Knowledge Base properties are considered to be User properties, and the value of this properties will be saved for each User who opens the Knowledge Base. Therefore it is possible that different users opening the same Knowledge Base have different values for these kind of properties. In addition these properties will not be involved in the Commit to GeneXus Server and Update From GeneXus Server operations.
