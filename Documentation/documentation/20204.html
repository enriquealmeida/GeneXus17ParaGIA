Auto Grow property

It allows you to automatically adjust the height of the control when the content (i.e. text) is larger than the size of the container.

Scope

Generators: Android, Apple

Description

This property is used in Smart Device applications to develop their user interface; more specifically, it allows you to automatically adjust the height of the control when the content (i.e. text) is larger than the size of the container.

Values

True The height of the control grows automatically, as much as necessary to show all the content.
False The height of the control does not grow automatically. It is up to the control to adjust the content when it is larger than the container; e.g. containers may add scroll bars if possible, text controls may clip text or add scroll bars, etc.

Controls - Default Value

This property applies to different types of Smart Device controls: containers and text. The default value depends on the type of control. The following table lists the controls that support this property and their default values.

Container Controls Default Value
Grid False
Table True
Group True
Text Controls
Attribute False
Variable False
LongVarChar (Attribute or variable) True
Texblock False
