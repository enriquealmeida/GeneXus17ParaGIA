OutputText data type

Represents a text along with its confidence and additional data in the context of GeneXus Artificial Intelligence. 

Members

  • Text: Text, GeneXusAI
  • Confidence: Confidence, GeneXusAI
  • Info: Collection -- Additional data on key-value pairs
    • Property: VarChar(40)
    • Value: VarChar(40)

Notes

  • The property-value pairs of Info field represent the word sequence of Text field with the associated start/end time (in seconds) inferred from the speech.

Scope

AI Task SpeechToText procedure
Platforms  Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This data type is available as of GeneXus 16 upgrade 9.
