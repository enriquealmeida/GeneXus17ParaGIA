Web Panel form

The purpose is to define the form for specifying the way in which the user will access data in Web applications. The form will be HTML.

The controls that can be used in web objects are classified basically into three groups:

1. Controls that may be used in all GeneXus objects:

  • Attributes
  • Variables
  • Buttons
  • Error viewers

2. Web object controls, or those with different behavior than in the rest of objects:

  • Images
  • Text Blocks
  • Hyperlinks
  • Tables
  • Horizontal Rules
  • Grids
  • Free Style Grids
  • Error Viewers
  • Web Components
  • Embedded Pages
  • Groups
  • Sections

3. Extended controls born in the GeneXus community and third parties, such as:

  • FBHoverPanel
  • Google Analytics
  • Google Annotated Time Line
  • And more...
