Selecting the Platform

Purpose

Allows selecting a platform from the list of platforms available from de combo box Platforms.

Description

Data View properties can be defined for each platform or Environment. These definitions will allows you to declare, for example, different locations of the same Data View within the same platform but for different Environments. This is very important if you wish to use a certain location for theÂ Environment and another for the application in runtime, even though both of them use the same DBMS.

This allows you to define Data View properties for a platform or for a specific model. The platform property definition is used, by default, by all the models that use this platform.
