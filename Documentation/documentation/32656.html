Deferred Deep Linking by Facebook

Deferred Deep Linking term is used to describe a particular scenario of mobile deep linking when the end user does not have the handler application installed on the device but it it is required for accessing to in-app content.

When do you Deferred Deep Linking?

You must add deferred deep linking to an ad if the target audience who did not install the app yet. If you are only targeting people who already installed your app, you do not need to add deferred deep linking.
Deferred Deep Linking is required when creating App Promotion Campaigns on Facebook. GeneXus will automatically handle the Deferred Deep Link in order to notify Facebook that the app has been installed.

Note

  • Deferred Deep Linking concept is not exactly the same as Deep Linking.

Scope

Platforms Smart Devices (iOS, Android)

Availability

This property is available since GeneXus 15.
