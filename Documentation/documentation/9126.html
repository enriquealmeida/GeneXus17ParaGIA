Translation type property

Defines whether to translate the current Environment.

Values

No translation Sets no translation for the current environment. This is the default value.
Run-time Allows translation at runtime.
Static Sets translation for the current environment.

Scope

Level: Environment

Run-time/Design-time

This property applies only at design-time.
