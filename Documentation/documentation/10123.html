AddLines method

Adds lines to a grid.

Syntax

control.AddLines(N)

Where:
control 
    Is the grid/free style grid ControlName

N 
    Is the number of lines to be added.

Scope

Controls: Grid, Free Style Grid

Note: This method doesn't validate; it only adds blank lines.
