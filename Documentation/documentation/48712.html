Design System Objects Overview

This table of content is a beginner's guide to learn about Design System objects. 

How do you specify in GeneXus the design that you want for your app? 

Going through a brief review of its bases, you will become familiar with the Design System concept, and you will find out how it is expressed in GeneXus by means of the Design System Object (hereinafter referred to, in short, as “DSO”).

So, based on the design of a few screens for a travel agency, a Design System will appear on scene, with the possibility to start the implementation of the app by getting acquainted with the methodology that is currently recommended for all developments, that is: DesignOps.

This will enable a consideration of the concepts associated with the DSO and how they are articulated with the implementation of screens for attaining the design sought.
