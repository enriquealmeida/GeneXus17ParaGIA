Publish Allowed property

Indicates if the login will ask for publishing permissions

Values

False
True

Description

A boolean value to indicate if the app requires publishing permissions.
By default it's set to False

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Build All.

Scope

Platforms: Smart Devices(Android, IOS)
Controls: SD Facebook Button
