Animated property

It's used to include an animation between state transitions. This transitions will be triggered by the end user when interacts with some controls in the application.

Values

False
True

Description

When this property is set to True, it also enables the Animation Duration property (for Animation Theme class).

This property is available in Theme object classes.

When some control has enabled this property, the animation could be associated with:

  • A transformation of the control. 
  • A smooth transition when a cell is collapsed or expanend (available as of GeneXus 15 Upgrade 4).
