WorkflowActionPerformedEvent Data Type

Description

This object represents an event corresponding to the execution of a specific action. It inherits all the properties of the Event object.

Properties

Property Type Access Description
ActionPerformed WorkflowActionPerformed  Read Action that was performed
Context WorkflowContext  Read Context (workitem, process instance) associated to the action
Id WorkflowEventId  Read Id
TimeStamp DateTime Read TimeStamp
Type WorkflowEventType  Read Type
Source WorkflowEventSource  Read Source
TargetType WorkflowObjectType  Read Target Type
Target WorkflowObject  Read Target
User WorkflowUser  Read User
