Synchronization.SetSendCheckpoint method

Syntax

Synchronization.SetSendCheckpoint()

Description

This method allows setting a checkpoint in the pending events list, in order to perform a batch synchronization in several requests instead of using only one big-sized request.

Scope

Objects Objects for mobile applications development
Platforms Android, Apple iOS 

Availability

As from GeneXus 16 Upgrade 3
