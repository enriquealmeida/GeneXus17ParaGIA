GetBeaconRegionState method

Retrieve the cached state of the specified region.

Parameters

Name Type Description
regionId VarChar(256) The beacon region's identifier to retrieve state.

Return Value

  • RegionState
    • Type: Numeric
    • Returns the state of the given region -it could be unknown, inside or outside.
