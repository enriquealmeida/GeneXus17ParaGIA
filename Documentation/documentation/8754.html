ControlName property

Is a name assigned to the control, and used to assign its properties at run-time.

Description

The name can be any alphanumeric string that begins with a letter, and can include the underscore (_) character.

Run-time/Design-time

This property applies only at design-time.

Scope

Objects: Transaction, Web Panel
Controls: Button, Embedded Page, ErrorViewer, FreeStyle Grid, Grid, Image, Text Block, Web Component
