Use HTML As Root Form

How to create an HTML form from the beginning, when the model uses the Web Abstract Editor by default?

This is an option of the contextual menu of the Web Abstract Editor form.

Description

Although by default the KB may use the abstract web editor, the user can change the Root form of each web object to HTML.

The steps to be followed are:

1. Create a new web object.
2. Right click on the abstract editor form, and select Use HTML As Root Form from the contextual menu.

Since then, the Root Form for this object will be HTML.
