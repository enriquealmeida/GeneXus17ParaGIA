TopMargin Property

To add a space by default between the window and the page content. The space separating the window of the page's start can be modified with the Top Margin property. The space defined is completed with the background color and image of the form.

Description

This property applies only at design time.

Scope

Objects: Transactions, Web Panels
Controls: Forms
Languages: .NET, Java, Ruby (up to GeneXus X Evolution 3)
Interfaces: Web
