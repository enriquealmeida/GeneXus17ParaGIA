DW transaction type property

To specify the data warehouse transaction type, indicating if the transaction is either a measure or a dimension one.

Values

Dimension The transaction is a dimension transaction. This is the default value.
Measure The transaction is a measure transaction.
Does not apply

Description

This property is enabled when the DW Transaction Property is set to True.

Scope

Objects: Transaction
