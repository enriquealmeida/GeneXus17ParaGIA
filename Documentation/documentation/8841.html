Text Method

Used to obtain the description of a certain item from one of the supported controls

Syntax

ControlName.Text(n)

Where:
n
    Is the position of the item in the list (1 refers to the first item from the list)

Description

The returned value is the item in the position passed.
Forms.Meta and Forms.MetaEquiv only applies to Web Interface.

Example

&CountryName = CountryList.Text(4)

In this example variable &CountryName is loaded with the description of the attribute CountryList which was defined as a Dynamic List Box and contains a list of countries loaded from a table. In this case the fourth item from the list is used.

Scope

Objects: Transactions, Web Panels
Controls: Combo Boxes, Dynamic Combo Boxes, Dynamic List Boxes, Forms.Meta, Forms.MetaEquiv, List Boxes
Languages: .Net, Java, Ruby (up to GeneXus X Evolution 3), Visual FoxPro (up to GeneXus X Evolution 3)
Interfaces: Web
