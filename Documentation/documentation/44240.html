Model data type

Represents a trained model in the context of GeneXus Artificial Intelligence for custom models.

Members

  • Id: VarChar(256)
  • Version: VarChar(64)
  • Dataset: VarChar(512) -- Filepath to CSV file with dataset summarize.
  • Type: DataOutputType, GeneXusAI.Custom

Notes

  • This data type is used as input of every task in the Custom model except for the Train procedure where this data type is the output.
  • The fields' value of this data type can be used as Properties of the Provider data type in order to use a custom model in a predefined GeneXusAI's tasks (e.g. in Classify procedure instead of using Predict procedure). In this case, you should use ProeprtyKey.ModelId and ProeprtyKey.ModelVersion to set the properties' values.

Scope

AI Task Check procedure, Delete procedure, Deploy procedure, Evaluate procedure, Predict procedure, Train procedure
Platforms  Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This data type is available as of GeneXus 16 upgrade 6.
