Lock retry count property

Indicates the number of retries to be made when a statement fails because it is trying to lock something that has already been locked.

Syntax

control. Lock retry count

A numeric valueÂ starting from 0. The default value is 10.

Scope

Available at the Data Store.
Data Store: POSTGRESQL, SQLSERVER, ORACLE
Objects: Procedure
Platforms: Web (.Net, Java)

Description

PostgreSQL support is added in GeneXus X Evolution 2 Upgrade #4.
