Default Master Page property

Sets the Web Master Panel to be used by default for every Web Panel object and every Transaction object.

Scope

Generators: Java, .NET, .NET Core
Level: Version

Description

The Default Master Page property is available at version level and by default, it is set with the 'RwdMasterPage' value, which is the name of a Web Master Panel object automatically created in every Knowledge Base.

In addition, every Web Panel object and Transaction object offer the Master Page property, which by default inherits and uses the Web Master Panel object set in the Default Master Page property (unless you explicitly change the Master Page property of the object).

Values

None or any Web Master Panel object name defined in the Knowledge Base. 

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
