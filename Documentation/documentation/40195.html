ProviderType domain

Enumerated domain representing every provider type in the context of GeneXus Artificial Intelligence.

Values

Alibaba Alibaba AI cloud-based provider.
Amazon Amazon Web Services cloud-based provider.
Baidu Baidu AI cloud-based provider.
Google Google Cloud Platform cloud-based provider.
IBM IBM Watson cloud-based provider.
Microsoft  Microsoft Azure Cognitive Services cloud-based provider.
MLKit Firebase ML Kit on-device provider
SAP SAP Leonardo cloud-based provider.
Tencent Tencent AI cloud-based provider.

Scope

Platforms  Web(.NET,.NETCore,Java), SmartDevices(Android,iOS)

Availability

This domain is available as of GeneXus 16.

  • As of GeneXus 16 upgrade 1:
    - Support for Google value.
  • As of GeneXus 16 upgrade 2:
    - Support for Amazon and Tencent values.
  • As of GeneXus 16 upgrade 3:
    - Support for Baidu provider.
  • As of GeneXus 16 upgrade 4:
    - Alibaba AI is available.
  • As of GeneXus 16 Upgrade 11:
    - Firebase ML Kit is available.
