Classpath property

Adds custom resources/JARS to the classpath. Note that these elements are added to the beginning of the base classpath, so they take precedence over resources at Base Classpath property.

Scope

Generators: Java
Level: Generator

Description

TheÂ Classpath can be set using this property.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.

Compatibility

This property's behavior has changed since GeneXus 15 upgrade 10. It can be configured by the user with any of the resources needed to run the application, which will precede the Base Classpath. The resources and JARs used by GeneXus are automatically assigned to the Base Classpath property.
