Theme and Default Theme properties (GeneXus 17u4 or prior)

Assigns a specific Web Theme.

Scope

Objects: Transaction, Web Panel
Level: Version

Description

Every Web Panel object and every Transaction object offers the Theme property so that you can assign a Web Theme object to it.

If you do not assign an explicit Web Theme to this property, it inherits the Web Theme set in the Default Theme property (offered at version level). The Default Theme property by default is set with the 'Carmine' value, which is the name of a Web Theme object automatically created in every Knowledge Base.

You can assign to both properties any of the Web Themes defined in your Knowledge Base.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply the corresponding changes when the property value is configured, execute a Rebuild All.
