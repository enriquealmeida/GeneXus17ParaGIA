Include GAM Backend property

If true, GeneXus Access Manager backend programs will be deployed with your application.

Values

False
True

Scope

Level: Deploy Target Options

Description

This property is at the deployment unit Target options of the Application Deployment tool.

When GeneXus Access Manager is activated, then the Include GAM Backend property will be shown. Since the Backend objects are distributed as binaries (GAM Web Backoffice), this property works to include it or exclude it in the deployment unit. If true, GeneXus Access Manager backend programs will be deployed with your application.

Availability

This property is available since GeneXus 16 upgrade 8.
