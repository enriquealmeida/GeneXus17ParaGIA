Dynamic structure property

Indicates whether the SDT (Structured Data Type) structure is dynamic or not.

Values

False
True

Description

Default value: False.

This property is used with the Infer Structure property. When set as True in the SDT, all changes made to the Data Provider's structure (with this SDT set in its Output property) will be applied to it (inferred).

Note

If the SDT structure is changed by the user, this property will automatically be set to False, thus disabling the inference mechanism.

Scope

Objects: Structured Data Type
