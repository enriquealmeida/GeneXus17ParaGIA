Decimal separator property

Configures the decimal separator to be used.

Values

',' Comma The decimal separator is a comma.
'.' Point The decimal separator is a point.

Description

English Assumes a point
Spanish Assumes a comma
Portuguese Assumes a comma
Italian Assumes a comma

Note

It is necessary to forcibly generate all model objects if the value of this property changes to ensure the change is taken into effect.

Run-time/Design-time

This property applies only at design-time.

How to apply changes

To apply changes made by this property, do a Re-Build All.

Scope

Objects: Language
